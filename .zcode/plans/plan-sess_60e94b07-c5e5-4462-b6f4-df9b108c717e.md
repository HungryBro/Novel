# แผนการเขียน Arc 3 — "เงาใต้แผนที่" (ตอน 51-80)

## สถานะเรื่องก่อนเริ่ม Arc 3
- **ความคืบหน้าแกนกลาง:** 15% (ปิดรอยแยกระดับ 4 ที่นอร์ธพอร์ตแล้ว)
- **ทีมเรย์:** สแนปโซ (Bond 30), เกโควาชิ (Bond 52), อีวุย (Bond 42), โบกุเระ (Bond 33)
- **เป้าหมายถัดไป:** เมืองซิลเวอร์ฟิลด์ (ภูมิภาคที่ 3, ระยะ 220 กม.)
- **คำสั่งระบบใหม่:** "Arceus — ตามหาต้นกำเนิด, Arc 3"
- **เงื่อนไขที่ตั้งไว้:** หัวหน้าสมาคมแคลร่าคนใหม่ + อาจารย์ของโนอา = อดีตแชมป์ VGC ที่ถูกดูดมา

## การแก้ไขความขัดแย้ง (canon fix)
1. **เปอร์เซ็นต์ความทรงจำ:** ใช้ **20%** ทั้งเรื่อง (แก้ค่า 18% ใน continuity-log)
2. **คำสั่งระบบ "Arceus":** ยืนยันเปิดตัวที่ตอน 50 และดำเนินต่อในอาร์ค 3
3. **เพิ่มไฟล์ใหม่:** `05-world-building/regions.md` บันทึกภูมิภาคที่ 3 ซิลเวอร์ฟิลด์

## โครงสร้าง Arc 3 (30 ตอน / 3 ช่วง)

### ช่วงที่ 1 — "เส้นทางสู่ซิลเวอร์ฟิลด์" (ตอน 51–60)
**ประเด็นหลัก:** เดินทางผ่านป่าสน, เจอสมาคมแคลร่าสาขาใหม่, เปิดเผยว่าแคลร่าเป็นแค่เครื่องมือเล็ก

| ตอน | ชื่อ (ชั่วคราว) | เนื้อหาหลัก |
|------|--------|----------|
| 51 | ป่าสนที่พึมพำ | เริ่มเดินทาง 220 กม. สแนปโซนำทาง, ป่าเงียบผิดธรรมชาติ ระบบแจ้งเตือนคลื่นมิติอ่อนๆ |
| 52 | หมอกเหล็ก | เจอพื้นที่ที่กฎเกมเริ่มเพี้ยน (ป่าสนเป็นเหล็ก) เรย์ใช้ความรู้เกมไม่ได้ |
| 53 | เด็กหนีภัย | เจอเด็กนักฝึกหนูที่หนีจาก "คนสวมเครื่องหมายกรงเล็บดำ" — เปิดตัวแคลร่าสาขาใหม่ |
| 54 | รอยกรงเล็บในเปลือกไม้ | สืบเบาะแส พบค่ายลับของแคลร่าสาขาใหม่ที่จับโปเกมอน Anomaly |
| 55 | หัวหน้าคนใหม่ — เวเรีย | เปิดตัวหัวหน้าสาขาใหม่ (ผู้หญิงเย็นชา, ใช้ Anomaly Toxicroak) อันตรายกว่าคนเดิม |
| 56 | ศึกป่าเหล็ก | ต่อสู้กับเวเรีย เกโควาชิบาดเจ็บ เรย์แพ้ครั้งแรก — **ความพ่ายแพ้ครั้งแรกของเรื่อง** |
| 57 | รอยแยกในใจเรย์ | ตอนสำรวจตัว — เรย์สงสัยว่าตัวเองเก่งจริงหรือแค่พึ่งเกม, โนอาปลอบใจ |
| 58 | ร้านยาสมุนไพรป่าลึก | **ตอนพักใจ** — เจอร้านยา, ฝึกฝนทีม, เรียนรู้กฎใหม่ของโลก (เครื่องหมายกรงเล็บดำ = สมาคมแคลร่า) |
| 59 | ลางสังหรณ์ของสแนปโซ | สแนปโซตอบสนองต่อบางอย่างในป่า — เบาะแสว่ามันเกี่ยวข้องกับรอยแยกมิติ (ปูปมตัวตนสแนปโซ) |
| 60 | เงาที่ขยายใหญ่ขึ้น | ปิดช่วง — เวเรียหนีไป แต่ทิ้งเครื่องหมายลึกลับ: **"กลุ่มเสียงประสานแห่งความว่างเปล่า"** เป็นครั้งแรก |

### ช่วงที่ 2 — "ซิลเวอร์ฟิลด์กับเสียงประสานแห่งความว่างเปล่า" (ตอน 61–70)
**ประเด็นหลัก:** เข้าเมืองซิลเวอร์ฟิลด์, เจอลูคัสอีกครั้ง, เปิดเผยกลุ่มเสียงประสานอย่างเต็ม

| ตอน | ชื่อ (ชั่วคราว) | เนื้อหาหลัก |
|------|--------|----------|
| 61 | ประตูเมืองซิลเวอร์ฟิลด์ | ถึงเมืองป่าสนสีเงิน เมืองค้าไม้-แร่ เจอผู้เฝ้าประตูระดับ 5 |
| 62 | ลูคัสที่ลานประลอง | เจอลูคัสอีกครั้ง ท้าแข่ง — เรย์ชนะ (ครั้งที่ 2 ตามกฎตัวละคร) |
| 63 | ตราทดสอบแห่งพงศ์สน | ทดสอบตราระดับ 5 — เกี่ยวข้องกับต้นสนโบราณที่ "มีชีวิต" |
| 64 | สมาคมลับในเหมืองเก่า | เรย์สืบพบกลุ่มลึกลับใช้เหมืองร้างเป็นฐาน — เบาะแสแรกของ "การบูชาผิดๆ" |
| 65 | คำสอนผิดเรื่องอาร์เซอุส | เปิดเผยหลักคำสอนของกลุ่มเสียงประสาน — เชื่อว่าความว่างเปล่าคือความสมบูรณ์ |
| 66 | บาทหลวงเงา | เปิดตัวผู้นำกลุ่มเสียงประสานสาขา — บาทหลวงผู้พูดจาอ่อนโยนแต่อันตราย |
| 67 | รอยแยกที่มนุษย์สร้าง | ค้นพบว่ากลุ่มเสียงประสาน **เร่ง** รอยแยกมิติโดยเจตนา |
| 68 | เครื่องหมายประสาน | **ตอนพักใจ** — เทศกาลป่าสน, เรย์ฝึกทีม, โนอาตัดสินใจเป็นของตัวเอง (ตามกฎตัวละคร) |
| 69 | คำใบ้ของอาจารย์ | โนอาพบบันทึกเก่าของอาจารย์ — เชื่อมโยงกับเสียงประสานแห่งความว่างเปล่า |
| 70 | เสียงที่เธอได้ยิน | ปิดช่วง — บาทหลวงเงาปรากฏตัว เปิดเผยว่ากลุ่มรู้เรื่อง "ผู้ถูกเลือก" — จบด้วยคำขู่ |

### ช่วงที่ 3 — "ความจริงของโนอาและรอยแยกใหญ่" (ตอน 71–80)
**ประเด็นหลัก:** ปมอาจารย์ของโนอาเฉลย, ศึกใหญ่กับกลุ่มเสียงประสาน, ปิดอาร์ค 3

| ตอน | ชื่อ (ชั่วคราว) | เนื้อหาหลัก |
|------|--------|----------|
| 71 | เส้นทางสู่เหมืองลึก | เรย์และโนอาตามร่องรอยอาจารย์เข้าเหมืองลึกใต้ซิลเวอร์ฟิลด์ |
| 72 | ห้องทดลองของผู้บูชา | พบห้องทดลองที่ใช้สสารความว่างเปล่าทดลองกับโปเกมอน |
| 73 | ประสานเสียงอาจารย์ | **เฉลย:** อาจารย์ของโนอา = อดีตแชมป์ VGC ระดับ 1 ที่ถูกดูดมา ถูกกลุ่มเสียงประสานจับตัว |
| 74 | ร่างที่กลืนกิน | เจออาจารย์ — แต่ถูกสสารความว่างเปล่ากลืนกินไปบางส่วนแล้ว (โศกเศร้า) |
| 75 | น้ำตาของนักวิจัย | โนอาต้องตัดสินใจยาก — เรย์สอนกลับว่าบางครั้งต้องปล่อยไป |
| 76 | พิธีกรรมแห่งความว่างเปล่า | กลุ่มเสียงประสานเริ่มพิธีเปิดรอยแยกมิติขนาดใหญ่ใต้เมือง |
| 77 | ศึกใต้ดิน | ต่อสู้กับบาทหลวงเงา + โปเกมอน Anomaly ระดับสูง |
| 78 | เปลวเพลิงสีดำตื่นขึ้น | สแนปโซปลดปล่อยพลังเปลวเพลิงสีดำเต็มรูปแบบ — ทำลายพิธีกรรม (ปูปมตัวตนสแนปโซต่อ) |
| 79 | ปิดรอยแยกใต้ซิลเวอร์ฟิลด์ | ปิดรอยแยกเสร็จ ความคืบหน้าเพิ่มเป็น ~30% บาทหลวงเงาหนีไปพร้อมคำขู่ |
| 80 | ม่านอาร์คสาม | **อาร์ค 3 ไฟนัล** — โนอาจากไปชั่วคราวเพื่อไว้ทุกข์อาจารย์, เอ็ดเวิร์ดปรากฏอีกครั้งเตือนเรย์ |
