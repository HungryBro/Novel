# ระบบบันทึกความจำของเทรนเนอร์

---

## ภาพรวม

ระบบ **บันทึกความจำของเทรนเนอร์** ไม่ใช่พลังวิเศษ แต่เป็นส่วนต่อประสานความทรงจำและทักษะที่ **Arceus ฝังไว้ในร่างของเรย์** ตอนที่ดึงเขาจากเกมเข้าสู่โลกโปเกมอนจริงๆ

**วัตถุประสงค์ที่แท้จริง:** Arceus ต้องการให้เรย์มี "เครื่องมือ" ที่เพียงพอจะเอาตัวรอดและค้นหาความจริง โดยไม่ทำให้เขาเก่งเกินไปจนทำลายสมดุลของโลก

**รูปแบบ:** แสงจางๆ ตรงหน้า ไม่ลอยเต็มหน้าจอเหมือนหน้าต่างหน้าจอข้อมูลทั่วไป

**กติกาเหล็ก:** ระบบไม่เคยบอกเรย์ตรงๆ ว่ามันมาจากไหนหรือทำไมเขาถึงมี — เขาคิดว่ามันคือ "ปาฏิหาริย์ที่ติดมาพร้อมกับการข้ามโลก" หรืออาจเป็น "ความเสียหายจากการดูดเข้ามา"

---

## ฟังก์ชันหลัก

### 1. หน้าต่างวิเคราะห์ข้อมูลโปเกมอน

| หัวข้อ | รายละเอียด |
|--------|-----------|
| **ฟังก์ชัน** | มองเห็นข้อมูลพื้นฐานโปเกมอนที่เคยพบในเกม (ชนิด ท่าที่เป็นไปได้ จุดอ่อนคร่าวๆ) |
| **ข้อจำกัด** | ใช้ได้เฉพาะโปเกมอนที่ "เคยเจอในเกมที่เขาเล่น" เท่านั้น |
| **กรณีพิเศษ** | ถ้าเป็นตัวที่โลกนี้มีแต่เกมไม่มี → ระบบเงียบสนิท |
| **การแสดงผล** | เรย์ "เห็น" ข้อมูลในหัว ไม่ใช่หน้าจอลอยกลางอากาศ |

### 2. ระบบคำนวณกลยุทธ์

| หัวข้อ | รายละเอียด |
|--------|-----------|
| **ฟังก์ชัน** | ช่วยประเมินประเภทที่ได้เปรียบเสียเปรียบ / ความเสี่ยงของท่าไม้ตายคร่าวๆ |
| **ข้อจำกัด** | ใช้ได้เฉพาะตอนจิตใจสงบ สมาธิดี |
| **กรณีพิเศษ** | ถ้าตื่นตระหนก / บาดเจ็บหนัก → ระบบเพี้ยนหรือดับ |
| **การแสดงผล** | เรย์คิดออกเสียงตอนคำนวณ (เพื่อให้ผู้อ่านเข้าใจโดยไม่ต้องยัดเยียดข้อมูล) |

### 3. ระบบวัดสายสัมพันธ์

| หัวข้อ | รายละเอียด |
|--------|-----------|
| **ฟังก์ชัน** | วัดระดับความผูกพันกับโปเกมอนของตัวเอง |
| **ข้อจำกัด** | ในโลกนี้สะท้อน "อารมณ์จริง" ของโปเกมอน ไม่ใช่ตัวเลข |
| **ผลลัพธ์** | บังคับให้เรย์เรียนรู้ว่าโปเกมอนคือชีวิต ไม่ใช่ตัวเลขในเกม |
| **การแสดงผล** | เรย์รู้สึก "แปลก" ตอนที่โปเกมอนไม่ตอบสนองตามที่คำนวณ |

### 4. สัมผัสสิ่งผิดปกติ (ปลดล็อกช่วงอาร์คที่ 2)

| หัวข้อ | รายละเอียด |
|--------|-----------|
| **ฟังก์ชัน** | รู้สึกได้เมื่ออยู่ใกล้พื้นที่ที่ "ผิดเพี้ยนจากกฎเกม" |
| **ข้อจำกัด** | ยังไม่รู้ต้นตอที่แน่ชัด |
| **ตัวเขาเองตีความ** | เป็นสัญชาตญาณนักแข่งที่ไวขึ้น |
| **การแสดงผล** | ปวดหัว / หูอื้อ / รู้สึกไม่สบายตัวโดยไม่มีเหตุผล |

## ส่วนลึกลับ: คำสั่งระบบจาก Arceus

ระบบบันทึกความจำของเทรนเนอร์ไม่ใช่แค่ระบบที่ติดมา แต่มี **ส่วนลึกลับ** ที่ค่อยๆ ปรากฏ — เป็นส่วนหนึ่งของระบบประสานงานที่ Arceus ฝังไว้ในร่างของเรย์ตอนดึงเขาเข้ามา

### ข้อความคำสั่งระบบที่ปรากฏเป็นครั้งคราว

| ข้อความ | ตอนที่ประมาณ | ความหมาย |
|---------|-------------|----------|
| "คำสั่งระบบ: ตามหาความผิดปกติ" | อาร์คที่ 2 | ให้ค้นหาสิ่งผิดปกติ |
| "คำสั่งระบบ: อาร์เซอุส — ตามหาต้นกำเนิด" | อาร์คที่ 3 | ให้ค้นหาต้นตอของปัญหา |
| (ข้อความอื่นๆ จะเพิ่มตามเนื้อเรื่อง) | — | — |

### ลักษณะการปรากฏ
- หน้าจอระบบกระพริบข้อความสั้นๆ แล้วหายไป
- เรย์ไม่รู้ว่ามาจากไหน — คิดว่าเป็นข้อผิดพลาดของระบบหรือความเสียหายจากการดูดเข้ามา
- คนอ่านรู้ก่อนว่าคำสั่งระบบมาจาก Arceus โดยตรง (เพื่อสร้างความสนุกแบบผู้อ่านรู้ความจริงก่อนตัวละคร)

## ระบบปลดล็อกระดับขั้น

| ระดับขั้น | ตอนที่ประมาณ | ฟังก์ชันที่ปลด | เงื่อนไขการปลด |
|------|-------------|---------------|----------------|
| **ขั้นที่ 1 (เริ่มต้น)** | 1–30 | หน้าต่างวิเคราะห์ข้อมูล, ระบบคำนวณกลยุทธ์พื้นฐาน | ติดตั้งอัตโนมัติตอนตื่น |
| **ขั้นที่ 2** | ~30 | ระบบวัดสายสัมพันธ์เต็มรูปแบบ + สัมผัสสิ่งผิดปกติเริ่มทำงาน | จุดเปลี่ยนอารมณ์กับสแนปโซ |
| **ขั้นที่ 3** | ~90 | ระบบเริ่ม "ตอบโต้" กับเรย์ได้ (เตือนภัยเชิงรุก) | เริ่มสงสัยว่าระบบ "มีเจตนาบางอย่าง" |
| **ขั้นที่ 4** | ~150 | ระบบเผยว่าเป็นส่วนหนึ่งของระบบประสานงานจาก Arceus | รู้ความจริงว่าโลกนี้คือโลกจริง ไม่ใช่เกม |

## ข้อจำกัดสำคัญ (ป้องกันไม่ให้ตัวเอกเก่งเกินไป)

1. **ไม่ทำงานกับโปเกมอนผิดปกติ** — ข้อมูลจะขึ้นเป็นสีเทา ผิดเพี้ยน หรือเงียบสนิท
2. **ใช้ได้เฉพาะตอนที่ใจสงบ** — หากตื่นตระหนกหรือบาดเจ็บสาหัส ระบบจะดับทันที
3. **ไม่มีแผนที่บอกทางแบบละเอียด** — รู้เพียงทิศทางคร่าวๆ แต่ไม่รู้ว่ามีอันตรายอะไรรออยู่
4. **ไม่มีการเซฟหรือโหลดเกมใหม่** — หากตายในโลกนี้จะตายจริงๆ
5. **วิเคราะห์ได้เฉพาะตัวที่เคยเจอในเกม** — หากเจอโปเกมอนสายพันธุ์ใหม่ในโลกนี้ ข้อมูลจะไม่ขึ้น

---

## การเล่าเรื่องในนิยาย (เน้นการแสดงให้เห็น ไม่ใช่บอกตรงๆ)

- ไม่เขียนตรงๆ ว่า "ระบบปรากฏขึ้น" แต่จะบรรยายว่า "เรย์หลับตา ภาพแสงจางๆ ลอยขึ้นในความมืด เขาไล่ดูข้อมูลที่ไม่มีตัวตนจริง แต่สัมผัสได้ถึงความอุ่น"
- ไม่เขียนตัวเลขค่าสถานะเต็มหน้ากระดาษ แต่ให้ซ่อนอยู่ในการกระทำ (เช่น รู้ว่าต้องหลบท่าไหนเพราะวิเคราะห์ข้อมูลในหัวแล้ว)
- ใช้ความคิดของเรย์ตอนวิเคราะห์กลยุทธ์เพื่อสื่อสารกับคนอ่าน แทนการรายงานหน้าต่างระบบลอยๆ

## หมายเหตุ

- ระบบนี้ต้องใช้สมาธิสูง ไม่ได้ทำงานอัตโนมัติตลอดเวลา
- เรย์ต้อง "ตั้งใจใช้งาน" ถึงจะใช้งานได้
- นี่เป็นเครื่องมือสร้างความกดดันในเรื่อง: ตอนที่เรย์ต้องการระบบมากที่สุด (ตื่นตระหนก) จะเป็นตอนที่ระบบใช้งานไม่ได้
