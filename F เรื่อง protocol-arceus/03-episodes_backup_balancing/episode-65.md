# ตอนที่ 65: "เวเรียผู้ไม่ยอมแพ้"

แสงสีม่วงแกมเทาจากแท่งคริสตัลเงินขนาดยักษ์ ‘คริสตัลแกนสมดุลเงิน’ (Silver Core Crystal) ส่องสว่างวาบตัดกับความมืดมิดของห้องโถงเก็บกักแร่เงินโบราณใจกลางเหมือง ทว่าท่ามกลางเศษหินและฝุ่นสนิมเหล็กที่ตกร่วงหล่น เงาร่างสูงโปร่งของหญิงผมเงินนัยน์ตาสีม่วงเย็นชา ‘เวเรีย’ ยืนประจันหน้าอยู่เหนือแท่นหินแกรนิตสูง มือขาวถือสะพานต่อเชื่อมควบคุมระบบพลังงานสีดำสลัวกำลังดูดซับเกลียวแร่เงินเข้ากระเป๋าเก็บประจุ

“พวกเธอเดินทางล่าช้าเกินไปนะ... แชมเปี้ยนซีโร่” เวเรียกล่าวเอ่ยวาจาน้ำเสียงเรียบหยิ่งยะโสพลางหรี่ตามองเรย์และโนอาที่เพิ่งก้าวเท้าพ้นแนวรุกลุยซากเหล็กเข้ามา

“เวเรีย! หยุดมือเดี๋ยวนี้นะ!” โนอาตะโกนลั่นเตือนภัยชี้นิ้วสั่งขวาง “ถ้าเธอทำลายและดึงคริสตัลแกนนี้ออกไป... ความบิดเบี้ยวมิติมณฑลเหนือจะกวาดล้างภูมิภาคซิลเวอร์ฟิลด์นี้หายสาบสูญถาวรแน่!”

เวเรียหัวเราะเสียงแผ่วเบาปนเยือกเย็น “นั่นแหละคือเป้าหมายของนายท่านเอ็ดเวิร์ดและกลุ่มเสียงประสาน... การกวาดล้างข้อมูลเพื่อเตรียม ‘รีโหลด’ คืนชีวิตใหม่ให้แก่โลกจริงใบนี้ ข้าไม่ยินยอมให้เทรนเนอร์ที่ใช้แค่สูตรคำนวณอย่างเจ้ามาขัดขวางหรอกเว้ย! ออกรบประปะทะ โดคุเกล็ก (Toxicroak)!”

บอลสีดำสนิทขว้างรวดเร็วตกลงพื้น โปเกมอนกบมีพิษลายม่วงดำโดคุเกล็ก (Toxicroak) ปรากฏกายพ่นน้ำลายขุ่นมัวสีน้ำหมึกดำ ดวงตากลมโตสลับแสงกะพริบสลัวเป็นเงาภาพซ้อนบิดเบี้ยว ร่างกายของมันแผ่ไอสีม่วงทมิฬสลายกฎธาตุธรรมชาติรอบตัวอย่างสมบูรณ์แบบ — ‘โดคุเกล็ก Anomaly (ประเภทพิษ/Void)’!

หน้าจอทองคำจำลองของระบบประสาทช่วยรบในสายตาของเรย์เด้งรหัสวิเคราะห์เตือนภัยระดับอันตรายขั้นวิกฤตทันที:

[ตรวจพบสิ่งผิดปกติระดับสาม: โดคุเกล็กบิดเบี้ยว (Anomaly Toxicroak)]
[ค่าสเตตัสพิเศษ: การประสานงานกฎธาตุย้อนกลับ (Inverted Typing Matchup Mode)]
[กฎธรรมชาติที่ผิดปกติ: ความเสียหายของพลังงานจิตใจ (Psychic Damage) และพลังงานพืชดินจะโดนสะท้อนกลับด้านลดทอนเหลือศูนย์ร้อยเปอร์เซ็นต์!]

*‘กฎธาตุย้อนกลับ...’* เรย์ขบกรามแน่น สบสายตาสีเทาอันคมกริบวิเคราะห์สมการ *‘ตามหลัก VGC โดคุเกล็กธาตุพิษและต่อสู้จะแพ้ทางดาเมจธาตุพลังจิต (Psychic) สี่เท่าอย่างรุนแรงที่สุด ทว่าในกฎแปรปรวนนี้ การยิงพลังจิตอัดใส่มันจะถูกแปลงเปลี่ยนเป็นตัวฟื้นฟูเลือดและฟื้นฟูพลังชีวิตให้แก่ศัตรูแทน! เป็นกับดักทางสถิติที่ร้ายกาจมากถ้าเผลอสั่งการตามตำราเกมดั้งเดิม!’*

“โดคุเกล็ก! ใช้ Poison Jab (หมัดสลักพิษดำ) ยิงทะลวงเข้าเบียดหน้าอกของสแนปโซการ์ดีเดี๋ยวนี้!” เวเรียสั่งการคำรามเสียงกึกก้อง

กบพิษยักษ์ขยับขาพุ่งทะยานเป็นเส้นสีม่วงด้วยความเร็วสปีดมหาศาลข้ามระดับสะพานหิน ปลายกรงเล็บแหลมเคลือบด้วยเมือกเหลวหนืดเหนียวความว่างเปล่าที่ส่งไอเย็นกรด กลิ่นอับสะพรึงลอยข้ามแนวรังสี

“สแนปโซ! หลบหลีกทางขวาด้วยจังหวะหมุน! เกโควาชิ! พ่น Water Pulse ยิงต้านทิศกรงเล็บพิษเพื่อลดทอนแรงบีบอัดความรุนแรงของดาเมจปะทะ!” เรย์กู่สั่งประสาทระบบเฉียบคม

เกโควาชินินจาพ่นคลื่นมวลน้ำบีบอัดพุ่งชนเป้าหมาย ทว่ากรงเล็บพิษดำของโดคุเกล็กแหวกมวลน้ำตกสลายลงอย่างรวดเร็วเนื่องจากพลัง Void เจาะทำลายกฎสสารน้ำ กรงเล็บกระแทกโดนหัวไหล่ซ้ายของเกโควาชิจนร่างกบนินจากระเด็นหมุนชนผนังถ้ำหินร้าวละเอียด ร้องเสียงขุ่นเจ็บปวด ค่าน้ำหนักความเสียหายลดขีดแถบเลือดสีทองลดฮวบลงไปสามสิบเปอร์เซ็นต์ในพริบตาแรกของการปะทะ

“เกโควาชิ!” โนอาอุทานด้วยความตกใจอย่างรุนแรงพยายามใช้เครื่องสแกนประคองรักษาระยะ

เวเรียเผยรอยยิ้มเย้ยหยันเหนือแท่นหิน “สูตรคำนวณแชมป์โลก VGC ของเจ้าน่ะ... มันล้าสมัยไปแล้วล่ะเรย์ คืนนี้พวกเจ้าทั้งหมดจะต้องโดนลบข้อมูลทิ้งคาเหมืองเงินร้างแห่งนี้แน่นอน!”

เรย์พยักหน้าให้เกโควาชิประคองตัวกลับขึ้นมาอย่างมุ่งมั่น *‘ยังเหลือวิธีรับมือมิติกลับขั้ว... ศึกคำนวณขั้นสองเพิ่งเปิดฉากเท่านั้นเวเรีย!’*
