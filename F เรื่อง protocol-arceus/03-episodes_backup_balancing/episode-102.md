# ตอนที่ 102: "ศึกอสุรกายคิงดราและพายุคลื่น Void"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดกระแสน้ำพายุดำสลัวแผ่ความชื้นจัดปะทะเพดานถ้ำหินวิหารชั้นในสุด แผ่นศิลาทองแดงหักพังจมดิ่งลงใต้ผืนน้ำสีน้ำหมึกหนืดขรึมกลางห้องโถงวิหารสลักเหล็ก สมาชิกกลุ่มเสียงประสานแห่งความว่างเปล่าห้าคนยืนสวดภาวนาโบกแท่งควบคุมพลังงานแปรปรวน ส่งรังสีสีม่วงดำชาร์จเข้าใส่คริสตัลแกนกลางรอยแยกยักษ์

ท่ามกลางมวลพายุน้ำดำที่หมุนวนเป็นเกลียวคลื่นทะยานสูง เงาร่างมังกรทะเลมหึมาพุ่งเหินเลื้อยทะยานตัวขึ้นมาจากปากเหวน้ำทมิฬส่งเสียงคำรามแหลมหู

มันมีรูปพรรณคล้ายคลึงมังกรทะเลประเภทน้ำและมังกร ‘คิงดรา’ (Kingdra) ทว่าขนาดตัวสูงใหญ่ตระหง่านกว้างกว่าแปดเมตร แผงครีบหลังปะติดด้วย **เกล็ดแก้วผลึกใสสะท้อนประกายมิติมืด** และรอบดวงศีรษะมังกรยาวมีรูดวงตาสว่างแสบส้มสีส้มแดงฉานถึง **แปดดวง** กะพริบจ้องสแกนตรวจรูปขบวน — ‘คิงดรา Anomaly (ประเภทน้ำ/มังกร/Void)’!

*“โฮก...!!!”*

เสียงกรีดร้องแหลมหูของมันกู่เรียกกระแสมหาพายุคลื่นวารีทมิฬ *Spatial Wave* (คลื่นมิติถล่ม) ซัดถล่มรุนแรงล้อมรอบแท่นดินปูน กวาดพัดเสาหินทรายพังทลายสลายตัวเป็นแก้วใสกลวงเปล่าและสลายตัวจากความว่างเปล่าชั่วพริบตาเดียวปัดม่านเสียงรบกวนคลื่น

หน้าจอทองคำจำลองของระบบช่วยรบในสมองของเรย์เตือนภัยรหัสวิบัติตัวแดงปะทุ:

[ตรวจพบรหัสอสุรกายภูมิภาค: คิงดราบิดเบี้ยว (Anomaly Kingdra) | เลเวล 79]
[ค่าสถานภาพพิเศษ: การควบคุมสนามฝนคลื่นว่างเปล่า (Void Drizzle field)]
[กฎบิดเบี้ยว: ดาเมจพิเศษประเภทธาตุไฟและสายฟ้า จะโดนคลื่นพายุลดทอนความรุนแรงลง 75% ชั่วคราวชดเชยน้ำหนัก!]
[สิทธิ์ถอดรหัสความเสถียร: พร้อมทำงานควบคู่กุญแจทองเหลือง]

*‘ฝนคลื่นว่างเปล่าลดทอนดาเมจไฟ 75%...’* เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทเผชิญพายุน้ำดำซัดสอย *‘นี่คือเงื่อนไขกดดัน VGC สลับขั้วที่โกงมาก! ภายใต้ Drizzle Void เพลิงทองคำของสแนปโซจะโดนหักพลังจู่โจมแทบสิ้นซากจนสร้างดาเมจได้ล้นพ้นลำบาก หากเราไม่มีวิธีสลายสภาพอากาศฝนคลื่นเพื่อดึงความสว่างตะวันคืน หรือสลัดกรงเล็บนินจาเกโควาชิรบเข้าตีตรงจุดอ่อนกายภาพระยะประชิด จะโดนพายุน้ำดำบดขยี้แหลกแน่ในเทิร์นเริ่มต้นแรก!’*

“ไคริว! ใช้ Sandstorm (พายุทรายทลายฝน) พัดสลัดมวลทรายเหล็กขจัด Drizzle เดี๋ยวนี้! เกโควาชิ! โบกุเระ! ตั้งแถวคุมโซนป้องกันซ้ายดักทาง!” เรย์สั่งการเสียงเฉียบขาดเฉียบแหลมมั่นคง

ไคริวมังกรยักษ์สลัดปีกกู่ร้อง เรียกพายุทรายเหล็กสีทองเหลืองปลิวว่อนเข้าปะทะพายุน้ำดำเพื่อแย่งสิทธิ์สภาพอากาศในสนามรบประวัติศาสตร์ปะทุดวลเดือดทันทีอย่างหาญกล้า!
คิงดราแปรปรวนตัวยักษ์แปดตาแผ่ครีบเกล็ดเกราะแก้วแผดร้องคำราม สาดคลื่นมิติถล่ม Spatial Wave และกางฝนคลื่นว่างเปล่า Void Drizzle หักทอนดาเมจไฟทองของสแนปโซพินาศ 75% ไคริวเข้าแย่งชิงสิทธิ์ประจุฟิลด์สนามอากาศด้วยพายุทรายทลายฝน Sandstorm ประสานงานเรย์ปะทะสบจังหวะรบรุกดุเดือด
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
