# ตอนที่ 119: "เศษเสี้ยวความหวังและซากบอลของลูคัส"

ฝุ่นควันและละอองสนิมเหล็กดำร่วงหล่นปกคลุมซากโครงเหล็กหักพังใต้สะพานลาดหินลึก เรย์และโนอาก้าวเท้าย่ำฝุ่นขยับร่างเหยียบย่ำซากเศษปูนอย่างเชื่องช้าท่ามกลางไอความว่างเปล่าเย็นเฉียบเกาะผิวแก้ว แสงไฟตะเกียงน้ำมันสั่นสะท้อนไหวเผยรอยแผลใจสะสมสะทือนจากเหตุการณ์ลูคัสสูญหายพลัดตกเหวก่อนหน้านี้ประคองวิจัยรักษาใจร้าว

เรย์ทรุดตัวก้มลมคุ้ยค้นเศษดินโคลนเหลืองใต้แผงเสาปูนแหลม ค่อย ๆ ยื่นข้อนิ้วขาวหยิบวัตถุทรงกลมโลหะผุพังชิ้นหนึ่งขึ้นมาจากโคลน — มันคือ **ซากบอลเปลือกเหล็กฉีกขาดของลูคัส (Lucas's Broken Poké Ball)** อุปกรณ์สื่อสัญญาณสุดท้าย

บอลสีแดงสลับขาวมีรอยฉีกผ่ายักษ์ผ่ากลางกระเป๋าเกลียวรหัสสลัก แผ่ประจุไอพลังงาน Void จาง ๆ กะพริบถี่ริ้วรหัสเอเร่อเตือนระบบสแกนวิญญาณสั่นไหว

“ซากบอลของไฟแอโร่วิวัฒน์แล้วของลูคัส...” โนอาประคองวิ่งเข้ามายกมือทาบอกเสียงสั่นสั่นระริกน้ำตาคลอเบา ๆ “มันไม่มีรหัสสสารชีวิตค้างคาข้างในเลยเรย์... แสดงว่าบอลพังก่อนลูคัสตกหล่นจมลึก หมายความว่าตัวแกคงพยายามสั่งเรียกไฟแอโร่กลับเข้าบอลก่อนตกเหว หรือไม่ก็ปัดปล่อยโปเกมอนของแกออกมาเพื่อพยุงตัวก่อนร่วงหล่นลงน้ำทมิฬกู้ชีพ!”

โนอาสูดหายใจลึกขยับเช็ดหน้าตาแดงก่ำ “นั่นหมายความว่าลูคัสและไฟแอโร่อาจจะยังมีชีวิตรอดอยู่ในชั้นอุโมงค์น้ำลึกทิศเหนือที่เชื่อมต่อแกนหลักค่ะเรย์! รหัสของแกยังไม่สูญสลายหายสาบสูญจากแกนคอร์โลกเสมือนจริงถาวร! เศษเสี้ยวความหวังของพวกเรากู้คืน!”

เรย์จ้องมองดูซากบอลเหล็กฉีกขาดในอุ้งมือด้วยสายตาสีเทาสุขุมนิ่งสนิททว่าแฝงประกายความหวังเต็มพิกัด หน้าจอระบบช่วยรบ Tier 3 แจ้งรหัสตรวจสอบพิกัดละเอียด:

[ได้รับเบาะแสความมั่นคง: ซากบอลฉีกรหัสเปลือกเหล็ก (Traced Lucas Signal)]
[ผลสแกนขีดรหัสแฝง: ตรวจพบรังสีออร่าชีวิตของลูคัสห่างไป 1.5 กิโลเมตร ทิศทางเหนือลึก]
[ขีดวิเคราะห์ระบบรบ: พร้อมประมวลข้อมูลนำทาง]

*‘ลูคัสยังไม่หมดลมหายใจกู้ระบบ...’* เรย์กำซากบอลแน่นสะพายเก็บเข้ากระเป๋าสแนปโซคู่ยาก *‘เขาสู้ขาดใจเพื่อรักษาสัจธรรมของโลกเดิม... ฉันในฐานะแชมป์เปี้ยนผู้ประคองกุญแจทองคำจะไม่ยอมปล่อยมิตรแท้คนนี้ให้โดนความว่างเปล่าสลายลบทิ้งเด็ดขาด! ศึกถัดไปโรงถลุงเหล็กกล้าบดขยี้รอยแยกภูมิภาคที่ห้าจะเป็นเป้าการทลายปิดฉากวิบัติภัยของพวกเรา!’*

“เดินทางรุกคืบก้าวต่อไปตามรังรอยรหัสของลูคัสกันเถอะครับโนอา!” เรย์สั่งวาจาน้ำเสียงเฉียบขาดกระชับทรงพลังหนักแน่น

โนอาพยักหน้าเด็ดเดี่ยวมั่นใจ ขบวนกู้ภัยรุกขยับก้าวเดินฝ่าฝุ่นปูนมุ่งตรงสู่อุโมงค์เหมืองเหล็กชั้นลึกเพื่อตามล่าเบาะแสช่วยเหลือเพื่อนยากทันทีอย่างเด็ดขาด!
เศษซากเสาเหล็กถล่มทับซากบอลเปลือกเหล็กฉีกขาดของลูคัสที่กะพริบแจ้งชีพจรรังสีออร่าชีวิตห่างออกไป 1.5 กิโลเมตรทิศเหนือลึก โโนอายิ้มปาดเหงื่อพยุงเสถียรความหวังลูคัสยังมีดวงจิตกู้ระบบ เรย์กำบอลฉีดขาดเก็บสะพายพร้อมเดินทางก้าวรุกกอบกู้ปิดฉากลัทธิบดขยี้
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
