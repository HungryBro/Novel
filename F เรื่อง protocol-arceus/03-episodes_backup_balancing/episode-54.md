# ตอนที่ 54: "รอยกรงเล็บในเปลือกไม้"

ค่ำคืนลงเมื่อพวกเขาถอยกลับไปตั้งหลักที่ริมลำธาร มีนาหลับไปแล้วกอดโปจจามะไว้แน่น ร่างเล็กๆ สั่นเครือเป็นระยะแม้ในหลับ เรย์คลุมเสื้อคลุมสีเทาให้เธอ ใจจริงเขาไม่ได้นึกถึงเรื่องอุณหภูมิ — เขาแค่ไม่อยากให้เด็กตื่นขึ้นมาเห็นความกังวลในดวงตาของผู้ใหญ่

"เล่าให้ฟังอีกครั้ง" เรย์เอ่ยเสียงเบา นั่งไขว่ห้างข้างกองไฟเล็กที่โนอาจุดไว้ "ทุกอย่างที่มีนาเล่ามา แบบเป็นลำดับ"

โนอาเปิดสมุดบันทึก อ่านออกเสียงเบาๆ "คนเสื้อคลุมดำมาถึงก่อนพระอาทิตย์ตกดิน ใช้เวลาประมาณยี่สิบนาทีคุมหมู่บ้าน พวกเขามีโปเกมอนตัวใหญ่สีดำที่มีนาไม่รู้จักสายพันธุ์ เธอบอกว่ามัน 'เหมือนหมาป่าแต่ใหญ่กว่า มีเปลวไฟสีดำที่คอ' — "

เรย์ชะงัก "เปลวไฟสีดำที่คอ?"

"ใช่" โนอาพยักหน้า "มีนาบอกว่ามันเหมือนไฟแต่สีดำสนิท ไม่มีแสง ไม่มีความร้อนที่รู้สึกได้ปกติ"

เรย์หลับตาลง ความทรงจำเรียงตัวขึ้นในหัว — เปลวไฟสีดำ ใช่ สแนปโซก็มีพลังคล้ายกัน เมื่อเจอสิ่งผิดปกติในป่าสนเมื่อสองวันก่อน เขี้ยวของมันลุกเป็นเปลวไฟสีดำเล็กๆ เช่นกัน ความคล้ายคลึงนี้ไม่ใช่เรื่องบังเอิญ

"โปเกมอนที่พวกเขาใช้..." เรย์เอ่ยช้าๆ "อาจเป็น Anomaly เช่นกัน หรือเป็นโปเกมอนที่ถูกดัดแปลงด้วยพลัง Anomaly"

"เหมือนซาเมฮาดะ Anomaly ของหัวหน้าแคลร่าที่นอร์ธพอร์ต?" โนอาถาม

"ใช่ — แต่น่าจะอันตรายกว่า" เรย์เปิดตา "ถ้าพวกเขาควบคุม Anomaly ได้หลายตัว เราสองคนเข้าตีตรงๆ คือการฆ่าตัวตาย"

"แล้วเราจะทำยังไง?" โนอาเสียงจริงจัง

เรย์หยิบกิ่งไม้แห้งขีดเขียนลงบนพื้นดินเบาๆ เป็นแผนผัง "กล่องเก็บพลังงาน Anomaly ใต้เกวียนคันกลาง — ถ้าเราทำลายมัน พลังงานมิติที่สะสมอาจรั่วออกมาและส่งผลกระทบต่อ Anomaly ที่พวกเขาควบคุม คล้ายกับที่เกิดขึ้นที่อู่เรือนอร์ธพอร์ต — เมื่อแบตเตอรี่รั่ว Anomaly ที่ควบคุมไว้เริ่มคลั่ง"

"แต่ถ้าพลังงานรั่วในหมู่บ้าน มันอาจทำร้ายชาวบ้านที่ถูกจองจำด้วย" โนอาชี้ประเด็น

เรย์พยักหน้า "เพราะงั้นเราต้องแยกชาวบ้านออกมาก่อน โซตะและโปเกมอนในกรงต้องออกมาก่อนที่จะทำลายกล่อง"

โนอาจดบันทึก "เราต้องรู้ว่าชาวบ้านถูกคุมไว้ที่ไหน"

"โบกุเระ" เรย์เรียกเสียงเบา ตอไม้ลอยลอยเข้ามาใกล้ "นายช่วยเข้าไปสำรวจอีกครั้งตอนดึก — มองหาว่าชาวบ้านถูกกักไว้ที่ไหน มีคนเฝ้ากี่คน และที่สำคัญมองหาจุดที่คนเสื้อคลุมดำเผลอ — ทางเข้าออกที่ไม่มีคนเฝ้า"

โบกุเระหมุนวนหนึ่งรอบตอบรับแล้วจางหายไปในความมืด

"แล้วฉันล่ะ?" โนอาถาม

"เธอดูแลมีนา" เรย์ตอบ "และเตรียมยารักษา — ถ้าทุกอย่างเป็นไปได้ด้วยดี ชาวบ้านและโปเกมอนอาจบาดเจ็บ ฉันต้องการเธอพร้อมรักษา"

โนอาขมวดคิ้วเล็กน้อย "เธอจะเข้าไปคนเดียวอีก?"

"มีสแนปโซ เกโควาชิ และอีวุย" เรย์มองไปที่โปเกมอนสามตัวที่นอนล้อมวงอยู่ใกล้ "สามตัวพอแล้วสำหรับภารกิจแทรกซึม ถ้าเจอศัตรูมากเกินไป เราจะถอย"

โนอาเงียบไปครู่หนึ่ง แล้วยื่นมือมาจับแขนเรย์เบาๆ "เธอสัญญานะ ว่าจะไม่เสี่ยงเกินไป"

เรย์มองตาเธอสักครู่ แล้วพยักหน้า "สัญญา"

ขณะที่รอโบกุเระกลับ เรย์หยิบกิ่งไม้แห้งอีกมองอย่างครุ่นคิด เขาจำได้ว่าเคยเห็นรอยกรงเล็บที่เปลือกไม้ในป่าสนเมื่อสองวันก่อน — ตอนนั้นเขาคิดว่าเป็นรอยสัตว์ป่า แต่ตอนนี้เมื่อเทียบกับเครื่องหมายกรงเล็บดำบนอกเสื้อของคนเสื้อคลุม รอยเหล่านั้นอาจเป็นสัญลักษณ์ที่พวกเขาใช้ทำเครื่องหมายเส้นทาง

"โนอา" เรย์เอ่ย "สมาคมแคลร่าที่นอร์ธพอร์ตเคยสารภาพว่ารับงานจากกลุ่มเสียงประสานแห่งความว่างเปล่าใช่ไหม?"

"ใช่" โนอาค้นหาสมุดบันทึกเล่มเก่า "แต่เขาบอกแค่ว่าสมาคมแคลร่าเป็น 'สาขาเล็ก' ของกลุ่มใหญ่กว่า"

"ถ้างั้นสาขาใหม่ที่นี่ก็เป็นสาขาเล็กเช่นกัน" เรย์คิดเป็นเสียงดัง "แต่ภารกิจของพวกเขาเฉพาะเจาะจงมาก — หาคนที่สัมผัสพลัง Anomaly ได้ พวกเขาไม่ได้แค่จับโปเกมอนขายเหมือนสาขานอร์ธพอร์ต พวกเขาหาคน ทำไม?"

โนอาหยุดเขียน สายตาเธอจริงจัง "คนที่สัมผัสพลัง Anomaly ได้... จะเกี่ยวข้องกับอะไรที่กลุ่มเสียงประสานแห่งความว่างเปล่าต้องการรึ?"

"ฉันไม่รู้" เรย์ตอบตรงๆ "แต่โซตะเป็นเด็กที่เพิ่งได้ตราทดสอบแรก — ถ้าเขาสามารถควบคุม Anomaly ได้จริง นั่นคือความสามารถที่หายากมาก กลุ่มเสียงประสานอาจต้องการใช้ความสามารถนี้"

"เพื่ออะไรล่ะ?"

เรย์นิ่งไปนาน เขานึกถึงคำเตือนของเอ็ดเวิร์ดที่นอร์ธพอร์ต — "พวกเธอก็ทำได้เพียงแค่ซื้อเวลาจำลองความพังทลายของระบบธรรมชาติออกไปชั่วคราวเท่านั้น" — ถ้ามีคนที่สามารถควบคุมพลัง Anomaly ได้ อาจเป็นไปได้ว่ากลุ่มเสียงประสานต้องการคนแบบนั้นเพื่อ... เร่งการพังทลายของกฎธรรมชาติ?

"ยังไม่รู้" เรย์ตอบในที่สุด "แต่เราต้องช่วยโซตะออกมาก่อนที่จะรู้คำตอบ"

เวลาผ่านไปจนดวงจันทร์ขึ้นสูง โบกุเระกลับมาปรากฏตัวจางๆ ส่งภาพสำรวจผ่านพลังผีเข้ามาในหัวเรย์ เรย์เห็นภาพชัดเจน — ชาวบ้านประมาณยี่สิบคนถูกกักไว้ในโรงนาขนาดใหญ่ที่ขอบหมู่บ้าน มีคนเสื้อคลุมดำเพียงสองคนเฝ้าอยู่หน้าประตู ส่วนอีกหกคนกระจายอยู่รอบเกวียนสามคัน โซตะยังถูกมัดอยู่ที่เสาใกล้เกวียนคันขวา

"มีช่องโหว่" เรย์พึมพำเบาๆ มุมปากเผยอยิ้มเล็กน้อยเป็นครั้งแรกในวันนี้ "ทางทิศเหนือของหมู่บ้านไม่มีคนเฝ้าเลย เพราะติดกับแนวหน้าผา พวกเขาคงคิดว่าไม่มีใครเข้ามาทางนั้นได้"

เรย์หยิบกิ่งไม้วาดแผนการอีกครั้งบนพื้นดิน "เราจะเข้าไปทางหน้าผาเหนือ — ฉันและสแนปโซไปปลดปล่อยชาวบ้านที่โรงนาก่อน เกโควาชิไปช่วยโซตะที่เสา ส่วนอีวุย..." เขามองไปที่อีวุยตัวเล็กที่นอนขดอยู่ "อีวุยจะเป็นตัวล่อ — วิ่งออกไปทางใต้ให้ศัตรูเข้าใจว่ามีโปเกมอนป่าเข้ามา เพื่อดึงความสนใจ"

"แล้วกล่องเก็บพลัง Anomaly ล่ะ?" โนอาถาม

"ทำลายทีหลังสุด เมื่อชาวบ้านและโปเกมอนออกมาหมดแล้ว" เรย์ตอบ "ฉันจะใช้สแนปโซเป็นคนทำลาย — เปลวไฟสีดำของมันมีคุณสมบัติกำจัดมิติ ควรทำลายกล่องเก็บพลังงานได้"

โนอามองแผนบนพื้นดินอย่างตั้งใจ แล้วพยักหน้า "ดูดี แต่ถ้ามีอะไรผิดพลาดล่ะ?"

"ถอยทันที ไม่มีข้อยกเว้น" เรย์ย้ำ "ชีวิตคนสำคัญกว่าภารกิจ"

เรย์มองขึ้นไปที่ดวงจันทร์เต็มดวงที่ส่องแสงเงินลงมาบนป่าสนเหล็ก คืนนี้จะเป็นคืนที่ยาวนาน
