# Protocol Arceus / ผู้ถูกเลือกที่ไม่เคยเลือก

> โครงการนิยาย Pokémon ข้ามมิติต่างโลก + ระบบช่วยเหลือ + ไขปริศนา
> แนว Web Novel จีน-เกาหลี | All Ages | 150–200 ตอน

---

## ข้อมูลสรุปเร็ว

| หัวข้อ | รายละเอียด |
|--------|-----------|
| **ชื่อเรื่อง** | ผู้ถูกเลือกที่ไม่เคยเลือก (Protocol Arceus) |
| **ตัวเอก** | เรย์ (Rei) "Zero" — แชมป์โลกประลองโปเกมอน อายุ 24 ปี |
