# ตอนที่ 182: "ดินแดนว่างเปล่าและเงาของกิราติน่า"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดพังพินาศปฐพีรอยแยกสล้างละเอียดของเสาศิลามรดกใต้ดินดังกึกก้องกังวาน นำพาคณะเดินทางข้ามพ้นปากประตูมิติด่านแปดเข้าสู่ ‘ดินแดนสลับมิติความว่างเปล่า’ (Void Core Dimension) ดินแดนเสมือนจริงไร้น้ำหนักที่มีริ้วสีทองคำเหลวกะพริบตาเป็นระนาบตารางกริดสีม่วงลอยเคว้งขว้างสะกดใจสลัว

ท่ามกลางมวลไอพอร์ทัลความว่างเปล่าที่เต้นระริก ร่างมังกรกึ่งปีศาจยักษ์มีปีกหนาสีดำหกปีกขลิบแก้วคริสตัล ‘กิราติน่า’ (Giratina Anomaly) ปรากฏตัวกรีดร้องคำรามเสียงต่ำแหลมหูแยกเขี้ยวเนตรแปดตาแดงฉานส่องขวางหน้า — ‘กิราติน่า Anomaly (ประเภทผี/มังกร/Void)’!

มันมีรูปร่างบิดเบี้ยวปกคลุมด้วย **แร่กระจกเงาสะท้อนรังสีมลพิษ** และปล่อยกระแสคลื่นเหนี่ยวนำ *Shadow Force Void* (พลังกรงเล็บเงามืดสลายตัว) ปัดกระแทกเศษปูนดินทรายจนละลายกลายเป็นอากาศชั่ววิ

โนอาสูดน้ำลายปากเบิกตากว้างตื่นตระหนก “เรย์! กิราติน่า Anomaly บอสใหญ่ตัวนี้ใช้พลัง Shadow Force สลายสภาพป้องกัน VGC ทุกชนิดและกวาดลบคำนวณในประสาท Tier 3 ของเธอเกือบสิ้นซากแล้วค่ะ! ถ้าเราสั่งแก้รหัสพลาด ร่างกายของพวกเราจะโดนเงาสอยสลายตัวกลายเป็นฝุ่นผง!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทขยับแว่นโมโนเคิลวิเคราะห์ขวาช่วยรบประเมิน VGC ในประสาท Tier 3:

[ตรวจพบสิ่งผิดปกติระดับสุดท้าย: กิราติน่าแปรปรวน (Anomaly Giratina) | เลเวล 78]
[สเตตัสพิเศษ: กรงเล็บเงามืดสลายตัวกวาด (Shadow Force Void)]
[กฎธรรมชาติบิดเบี้ยว: ดาเมจพิเศษประเภทผีและไฟปกติจะโดนหักพลังสู้ 80% ชั่วคราว!]
[ขีดวิเคราะห์ความทรงจำชิบูย่า: เสถียร 90%]

*‘เงามืดกวาดดาเมจไฟและผีแปดสิบเปอร์เซ็นต์...’* เรย์หลับตาสแกนสมมาตรในความคิด VGC *‘กติกาสลับมิติบอสโกงมาก! ภายใต้เงาสลายตัว เพลิงทองของสแนปโซจะสูญเสียกำลังทำลายเกราะ หากเราไม่มีวิธีสอยสปีด Tailwind หนุนความเร็ว หรือดึงเจตจำนงรบสบตาเปล่าปิดจอระบบรุกกายภาพ จะโดนกรงเล็บฉีกร่างสลายไปสิ้นเชิง!’*

“สแนปโซ! เกโควาชิ! ตั้งแถวคุมโซนป้องกันซ้ายขวา! ลูคัสสั่งไฟแอโร่บินคุมขีด Tailwind ล่อปีกมัน!” เรย์ตะโกนชี้นิ้วสั่งการเสียงเฉียบกังวานลั่นมิติ

กบเกโควาชิและ Growsits สแนปโซกระโจนสปริงร่างฝ่าไอพายุคลื่น Void เปิดศึก decisions สุดท้ายอย่างองอาจกล้าหาญ!
รอบข้างแท่นศิลาปฐมกาลในดินแดนสลับมิติความว่างเปล่าลอยเคว้งคว้างไร้น้ำหนัก ท้องฟ้าเปลี่ยนเป็นสีม่วงทมิฬสลับกับริ้วเส้นตารางกริดสีทองคำเหลวของระบบระบบที่สั่นเตือนเตือนระเบิดถี่ โนอากอดกล่องวิจัยเฝ้ามองความผันผวนอย่างลุ้นระทึกดวงตาคลอน้ำใส ลูคัสและไฟแอโร่สแตนด์บายขบวนรบร่วมจูลส์อย่างเด็ดเดี่ยวเปี่ยมสัมผัสรอยต่อ เรย์เพ่งสบสายตาสีเทาสุขุมผ่านแว่นตาโมโนเคิลขวาประเมินสูตร VGC ในหัวประสาทสมอง Tier 3 พยายามควบคุมระดับความเสถียร Willpower เต็มขีดสัมบูรณ์ 100% ความเป็นแชมเปี้ยนชิบูย่าดั้งเดิมช่วยนำความอุ่นประสานชะล้างไอ Anomaly มลพิษโดยไม่ยอมพ่ายแพ้ต่อบทบาทลบล้างกติกาโลกเสมือนจริงสะสะสม
