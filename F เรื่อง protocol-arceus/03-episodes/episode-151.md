# ตอนที่ 151: "พายุหิมะกระหน่ำในเหวหิมะขาวสูญสิ้น"

*ครืน... ครืน... เพล้ง...!!!*

เสียงวายุหนาวกรีดร้องดังสนั่นสะเทือนทั่วแนวกำแพงน้ำแข็งยักษ์ของหุบเหว 'หิมะขาวสูญสิ้น' (White-Peak Abyss) ยอดผาลึกตอนเหนือสุดเขตมณฑลแกลร่า เกล็ดหิมะบางเบาและละอองน้ำแข็งร่วงหล่นละลิ่วลงมาถล่มทับระเบียงดินลาดขรุขระจนมืดสลัวเกือบสิ้นเชิง เรย์ โนอา และจูลส์ กระชับเสื้อคลุมกรูดเดินทางไกลฝ่ามลพิษรอยแยกไอแปรปรวนสีม่วงดำส่งกลิ่นอายคุกคาม

โนอาใช้คอมพิวเตอร์พกพาสวมแว่นรายงานน้ำเสียงสั่นสั่นระริกหน้าซีด “เรย์! ค่าความหนาแน่นมิติลัดวงจรกระเพื่อมสะสมขึ้นแตะร้อยละเจ็ดสิบสองแล้วค่ะ! รอยแยกมิติตามผนังน้ำแข็งกำลังขยายตัวแยกกว้างจนจะลบข้อมูลธรรมชาติรอบตัวหุบเหวนี้ทิ้งไปทั้งหมด!”

เรย์ยืนคุมรูปขบวนนิ่ง นัยน์ตาสีเทาสุขุมภายใต้โมโนเคิลวิเคราะห์ระบบรบ Tier 3 ที่ใช้โหมดสายตาจิตเจตจำนงรบกู้ภัยแสดงผลพิกัด:

[ตรวจพบรัศมีรอยแยกวิบัติภัย: กำแพงน้ำแข็งหุบเหว (White-Peak Abyss Ravine)]
[ระดับเหนี่ยวนำสสารความว่างเปล่า: พุ่งขึ้น 65% (Alert Tier)]
[ความทรงจำแชมเปี้ยนชิบูย่าเสถียร: 90% (ระบบกู้ภัย Firewall ปลอดภัย)]

*‘พายุหิมะหนาวจัดต้านสปีด VGC และล้างพิกัดคำนวณ...’* เรย์หลับตาลงรวบรวมรหัสสมบูรณ์ประสาทในหัวประสาทรวดเร็ว *‘ที่ใจกลางหุบเหวหิมะขาว... แรงลมพายุหนาวจะคอยพัดทรายแร่และน้ำแข็งกรดมาขัดขวางขีดสปีดคู่หู หากเราประคองแนวรบช้า จะโดนมวลหิมะถล่มทับจนหมดสเตตัสป้องกันสู้ต่อ! เราต้องอาศัยเส้นตารางกริดสีทองใต้หิมะนำทางและประสาน Tailwind พัดล้าง!’*

“สแนปโซ! ยิงไฟกงล้อ Golden Flame Wheel พุ่งส่องความร้อนละลายหิมะหนาขวางสะพานด้านหน้าเปิดทางลาด! เกโควาชิ! ตั้งท่าป้องกันขวาคุมโซน!” เรย์ชี้นิ้วสั่งการเสียงเฉียบกังวานหนักแน่น

Growlithe สแนปโซเห่ากู่คำราม ร่างหมุนควบเปลวอัคคีเพลิงทองคำเหลวเจิดจ้าพุ่งปาดสลายหิมะหนาเปิดทางเดินหินทรายสว่างวาบขจัดมลพิษ ละลายพายุเย็นเฉียบจนระเหยขึ้นสู่ฟ้า โโนอาและจูลส์พยุงตัวก้าวเท้าย่ำฝุ่นรบรุกรุดหน้าตามแนวพิกัดสแกนทันทีอย่างองอาจมั่นใจ!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
