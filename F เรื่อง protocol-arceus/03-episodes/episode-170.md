# ตอนที่ 170: "ม่านหนาทึบและห้องโถงสุดท้ายของวิหาร"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดกระแสมิติลำดับสี่จากใจกลางเจดีย์ปฐมกาลโบราณสูงสุดดังสนั่นเลื่อนลั่นสะเทือนสลับเสียงเสาศิลาถล่มร่วงราบจมดิ่งลงใต้ทะเลเดือดพลาสม่าเหล็กทรานสิชั่น เรย์ โนอา และลูคัส ก้าวเท้าหยุดเหยียบยืนอยู่บนระเบียงสะพานเหล็กกล้าสุดท้าย เผยให้เห็นทัศนียภาพห้องโถงวิหารชั้นในสุด — ‘ห้องโถงปฐมกาลสัมบูรณ์’ (Genesis Altar Room) ที่แผ่ไอรังสี Anomaly สีม่วงดำมหาศาลพวยพุ่งพีกร้อยเปอร์เซ็นต์สะกดดวงจิตน่าเกลียด

เบื้องหน้าของพวกเขา ม่านกักมิติความว่างเปล่าหนาทึบขนาดยักษ์พิกัด ‘ม่านพลังงาน Void แกนเจ็ด’ (Genesis Void Shield) ตั้งตระหง่านครอบคลุมแท่นศิลาปิดกั้นทางลาดกู้ภัย

เหนือแท่นบูชา บิชอปวาลด์ (Bishop Vald) แม่ทัพใหญ่ลัทธิยืนกอดอกกู่คำรามกร้าวแววตาเหี้ยมเกรียมสะใจท้าทายสงคราม “ฮ่า ๆ ๆ ! เจ้าแชมเปี้ยนซีโร่แรกรบ... แกนรอยแยกที่เจ็ดคุมพิกัดสูงสุดเปิดใช้งานชาร์จเต็มระบบแล้ว! หากนายกล้าแตะตรากุญแจทองเหลือง... มวลความว่างเปล่าจะย่อยสลายร่างของนายและกวาดลบล้างข้อมูลโลกธรรมชาติจริงทิ้งถาวร!”

เรย์ยืนมองดูม่านกัก Void แกนเจ็ดนิ่ง นัยน์ตาสีเทาสุขุมนิ่งสนิทภายใต้โมโนเคิลขวาประเมิน VGC ในประสาทช่วยรบ Tier 3:

[ตรวจพบรัศมีรอยแยกสุดท้าย: ห้องโถงปฐมกาลสัมบูรณ์ (Genesis Altar Room)]
[ระดับการกัดเซาะสสารความว่างเปล่าสะสม: ทะยานขึ้น 80% (Alert Tier)]
[ขีดวิเคราะห์ความทรงจำชิบูย่าคงค้าง: 90% (คงที่เสถียรดี)]
[สถานภาพความเสถียร Willpower: เต็มร้อยพิกัดรบเสถียร]

*‘ม่านหนาทึบกั้นรอยแยกสุดท้าย...’* เรย์หลับตาลงรวบรวมรหัสสมบูรณ์ประสาทสมองรวดเร็วดั่งมีดโกน *‘ศึกตัดสินสุดท้ายอาร์คเจ็ดอยู่ตรงหน้า... ไม่มีทางลัดสลับสูตรในจอคำนวณสแกนกบดาน แต่ความผูกพันและสัมผัสสายตาจริงจะชี้ทางสแนปโซการ์ดีทองคำทลายม่าน Void เจาะแกนกลางได้เด็ดขาดปราศจากหนทางพ่ายแพ้!’*

“ผมพร้อมจะทลายม่านและปิดแกนรอยแยกที่เจ็ดสยบภัยพิบัติครับบิชอปวาลด์!” เรย์ตะโกนสั่งเสียงกึกก้องลั่นสะท้านวิหาร

เรย์สั่งการเกโควาชิและสแนปโซการ์ดีทองคำร่วมรบพุ่งตัวนำศึกเดือดชี้ชะตามณฑลเหนือสุดเริ่มต้นขึ้นในทันทีอย่างหาญกล้ามั่นคงรบรุกฆาต!
รอบบริเวณระเบียงศิลายอดเจดีย์ปฐมกาลทองเหลืองยังคงอบอวลไปด้วยกระแสพลังงานทองคำหนาแน่นที่สั่นเตือนสะท้อนกังวานสลับกับไอเย็นจัดจากพายุหิมะภายนอก เกล็ดหิมะบางเบาสล้างปลิวว่อนผ่านลวดลายสลักโบราณประดับยอดเสา โโนอากุมกล่องวิจัยสแกนค่าความเสถียรมิติด้วยใบหน้าเคร่งเครียดปนตื่นตัว ลูคัสคุมขบวนรบร่วมคู่หูไฟแอโร่อย่างตั้งใจพึ่งพา เรย์ใช้สายตาสีเทาสุขุมเพ่งมองริ้วรอยพับกริดสีทองใต้พื้นด้วยระบบช่วยรบ Tier 3 ปรับใจเจตจำนงรบ Willpower คงที่ร้อยเปอร์เซ็นต์ ความเป็นผู้สมัคร Candidates คนนอกระบบชี้ทางให้เขาเข้าใจความผูกพันดั้งเดิมที่ล้ำค่ากว่าสูญสิ้นสถิติ VGC ทวีคูณสะสะสม
