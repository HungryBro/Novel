# ตอนที่ 150: "ประตูมุ่งสู่หุบเหวหิมะขาวสูญสิ้น"

*ครืน... ครืน... เพล้ง...!!!*

เสียงแยกตัวดังกังวานก้องปฐพีรอยร้าวของบานประตูหินศิลาทองเหลืองแกะลายสลักมังกรยอดเจดีย์หน้าทิศเหนือของหอวิหารทองเหลืองวิจิตรดังสะท้อนกังวานดึงจังหวะพายุหิมะพัดผ่า พวยพุ่งละอองหิมะป่าหนาวบางเบาลอยละลิ่วพัดปะทะใบหน้าเรย์ โนอา และจูลส์ ที่สวมเสื้อหนาวหนายืนตรงประตูด่านสุดท้าย

หลวงพ่อเทนซินกางมือซ้ายส่องรังสีออร่าสีทองคำเหลวเจิดจ้าประสานกุญแจทองเหลืองของเรย์ชี้ทางลาดดิ่งลงสู่หุบเหวพายุขัดสัญญาณมลพิษ

“พิกัดประตูรอยแยกหกยอดเขาหิมะขาวสูญสิ้น (White-Peak Abyss) เปิดใช้งานสำเร็จแล้ว...” เทนซินกล่าวเตือนน้ำเสียงจริงจังหนักแน่น “พวกลัทธิเสียงประสานบดขยี้กักลูคัสและไฟแอโร่ไว้กลางน้ำตกน้ำแข็งเพื่อประกอบพิธีเจาะรอยร้าวแก้ไขระบบคอร์... ศัตรูใช้ Tyranitar Anomaly โลหะพลังหนาต้านดาเมจเวทย์พิเศษสี่สิบชั้นคุมด่านรักษาการณ์อยู่เบื้องใน!”

“จงประคอง Willpower ประสาทโมโนเคิลขวาและสแนปโซการ์ดีสองสัจธรรมบุกเจาะระบบชะล้างปิดแกนนี้ด่วนก่อนสสาร Void ลบข้อมูลเมืองหลักพินาศ!”

เรย์พยักหน้าสลัดวิเคราะห์ VGC ประสาทรบ Tier 3 ประสานประสาทกู้ภัยแสดงผลพิกัด:

[ปลดล็อกพิกัดสงครามสุดท้าย อาร์ค 6: หุบเหวหิมะขาวสูญสิ้น (White-Peak Abyss Area)]
[ระดับการกัดเซาะสสารความว่างเปล่าสะสม: คงเหลือ 65% (Progress 65% sealed)]
[ระดับความจำแชมเปี้ยนชิบูย่าคงค้าง: 90% (เสถียรรหัสกู้ภัย)]
[สัญญาณลูคัสประสาทร่วมรบ: คาดว่าพร้อมช่วยสอยกบดาน]

*‘รอยแยกที่หก... จะเป็นศึกทดสอบใหญ่ที่บีบให้ฉันต้องปลดปล่อยเปลวไฟไฟอุ่นชะล้างไอเย็นจัด...’* เรย์จ้องมองดูเส้นตารางกริดสีทองใต้หิมะด้วยดวงตาสีเทาสุขุมนิ่งสนิทดุจเหล็กกล้า *‘ไม่มีหน้าจอคำนวณเอเร่อรอยต่อช่วยสแกน... แต่สัมผัสธรรมชาติและสายสัมพันธ์ร่วมศึกจะชี้ทางสยบ Tyranitar บอสลงได้เด็ดขาดปราศจากริ้วรอยพ่ายแพ้!’*

“ลุยรุกคืบเดินทางเข้าสู่หุบเหวปิดรอยแยกและช่วยลูคัสกลับบ้านกันเถอะครับโนอา! จูลส์!” เรย์สั่งน้ำเสียงเฉียบขาดเฉียบแหลมหนักแน่น

โนอาและจูลส์พยักหน้าเด็ดเดี่ยวมั่นใจ ขบวนกู้ภัยรุกขยับก้าวเท้าย่ำฝุ่นข้ามทางลาดด่านประตูดิ่งมุ่งตรงสู่หุบเหวหิมะขาวสูญสิ้นเพื่อประเดิมเปิดม่านรบอาร์คที่หกอย่างสง่างามทันที!
รอบบริเวณห้องโถงวิจัยและลานประลองแกรนด์ซัมมิทอารีน่ายังคงอบอวลไปด้วยไอหมอกเย็นชื้นปนกลิ่นกระแสไฟฟ้าจากท่อไอน้ำใต้ดินที่พ่นระเหยควันขาวฟุ้งกระจาย แสงไฟนีออนสีส้มเหลืองและเขียวสะท้อนผ่านผลึกเกล็ดหิมะบางเบาที่ร่วงหล่นลงมาทับถมแนวกำแพงศิลาหนาเตอะดึงทัศนียภาพหนาวเหน็บ โนอากระชับผ้าพันคอหนาสีกรมท่าคอยคีย์ปรับความเสถียรข้อมูลระบบวิจัยอย่างใส่ใจดูแลในขณะที่สแนปโซและเกโควาชิเฝ้าระแวดระวังแนวขอบเขตสนาม เรย์ใช้สายตาสีเทาสุขุมนิ่งสนิทจับจ้องวิเคราะห์ริ้วสัญญาณรหัสสีทองของตารางกริดจำลองกู้ภัยผ่านโมโนเคิลขวา พยายามประสานพลังเจตจำนงรบ Willpower ขั้นเต็มร้อยเปอร์เซ็นต์เพื่อชะล้างลบรอยร้าวมลพิษ Anomaly ข้ามขีดปวดล้าสะสม การเดินทางร่วมทาง VGC ในชีวิตจริงที่เกิดจากความผูกพันและสายสัมพันธ์ร่วมรบพยุงระดับความมั่นคงสมาธิประสาทช่วยรบ Tier 3 ให้คงที่เสถียรรหัสเต็มพิกัดโดยไม่มีการยอมจำนนต่อกฎบิดเบี้ยวมิติ
