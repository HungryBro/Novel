# ตอนที่ 138: "การพบเจอจูลส์และอดีตแชมเปี้ยนผู้สูญเสีย"

กลิ่นไอกระดาษเก่าและเขม่าถ่านหินฟืนไม้จากร้านกาแฟเล็กหลังโกดังลานประลองแผ่ความเงียบงันสลัวปกคลุม เรย์และโนอานั่งลงบนเก้าอี้ไม้โอ๊คเก่าตรงข้ามกับจูลส์ (Jules) เด็กหนุ่มแชมป์ VGC รุ่นพี่ที่เพิ่งก้าวพ้นรอบประลองที่สามเข้ามาเก็บตัวอย่างลับ ๆ โดยไม่มีคู่หูมังกรเหล็กโฮโดรอนคอยเคียงข้างแผ่ออร่าความแปรปรวน

จูลส์วางถ้วยชากระเบื้องดำลงบนโต๊ะช้า ๆ สบสายตาสีนิ่งสีเทาอมม่วงเข้มจ้องมองข้อมือของเรย์ที่กุมตรากุญแจทองเหลืองกู้ภัยประสาทช่วยรบ

“นายชื่อเรย์สินะ... แชมเปี้ยนซีโร่แรกรบกู้โลก...” จูลส์กล่าววาจาน้ำเสียงเรียบสุขุมแต่แฝงประกายดุดันปนอ่อนล้าลึกซึ้ง “ฉันก้าวเดินทางเข้ามาในโลกจำลองนี้ก่อนนายสามปีเต็ม... ในตอนแรกฉันมีระบบประสาทช่วยรบคำนวณกราฟแบบเดียวกันเปี๊ยบ... แต่ในการประลองศึก ‘Grand Trial’ เพื่อกู้รอยแยกแกนที่สี่ภูมิภาคหนาวครั้งก่อน... ข้อมูลพลังของคู่หูของฉันลัดวงจรสะกดพ่ายแพ้จนร่างมังกรคู่ยากโดนไอ Void กวาดลบล้างร่างสลายพินาศไปกับตา!”

จูลส์คำรามสั่นสั่นระริกกำถ้วยชาแน่นจนน้ำชากระฉอก “เมื่อคู่หูพ่ายสลาย... ระบบลบรหัสความทรงจำดั้งเดิม Memory Lock ก็เข้าปัดลบข้อมูลชิบูย่าของฉันทิ้งไปเกือบแปดสิบเปอร์เซ็นต์สะสะเทือน! ทิ้งฉันให้กลายเป็นอดีตแชมเปี้ยนผู้ล้มเหลวตาบอดประสาทที่เฝ้ากบดานพึ่งพาชีวิตรอดดั่งขยะสัญจรลีกสำรอง!”

โนอาสูดน้ำลายปากเบิกดวงตาสลัวตกตะลึงล้นพ้น “แสดงว่า... จูลส์สูญเสียคู่หูร่วมศึกและข้อมูลความทรงจำไปเกือบหมดสิ้นเพราะล้มเหลวในการชำระล้างรอยแยกมิติเหรอคะอาจารย์จูลส์!”

“ใช่แล้วแม่หนู...” จูลส์ตอบน้ำเสียงราบเรียบปนท้อถอยใจขัดรหัส “ตอนนี้ฉันมีสัญญาณรบแค่กู้ประคองระบบไปวัน ๆ เพื่อตามล่าล้างแค้นพวกคัลต์เสื้อคลุมเทาที่ลอบฆ่าโฮโดรอน... เรย์เอ๋ย นายกำลังเดินสะดุดบนเส้นทางกบดานหนทางเดียวกัน ระวังให้ดีเถอะ!”

หน้าจอกริชทองประสาทช่วยรบของเรย์แจ้งข้อมูลยืนยันความพ่ายแพ้ Candidates อดีต:

[ยืนยันประวัติความเป็นมา: Candidate ลำดับสองสูญสิ้นสิทธิ์ระบบ 80%]
[ระดับความทรงจำกู้คืนเรย์คงเหลือ: 90% (เสถียรรหัสเต็มขั้น)]
[เป้าหมายรหัสถัดไป: ปรับจูนระบบรบประสานงานร่วมกันเพื่อขัดขวางคัลต์]

เรย์มองสบตาจูลส์ด้วยประกายตาเทาสุขุมนิ่งสนิท “ผมจะสู้รบไม่ใช่เพียงเพื่อป้องกันสิทธิ์ระบบหรือรีเซ็ตเกม VGC ชิบูย่า... แต่ผมจะปิดผนึกรอยแยกช่วยชีวิตลูคัสและกู้คืนสัจธรรมของพวกเราทุกคนให้สมบูรณ์แบบร้อยเปอร์เซ็นต์โดยไม่มีการทิ้งเพื่อนแท้คนไหนเด็ดขาดครับจูลส์!”

จูลส์จ้องมองดูดวงตาสลัวแชมป์ซีโร่ช้า ๆ พยักหน้ารอยยิ้มยอมรับปนทึ่งในดวงใจเตรียมร่วมศึกปิดด่านต่อไปทันทีอย่างกล้าหาญ!
รอบบริเวณห้องโถงวิจัยและลานประลองแกรนด์ซัมมิทอารีน่ายังคงอบอวลไปด้วยไอหมอกเย็นชื้นปนกลิ่นกระแสไฟฟ้าจากท่อไอน้ำใต้ดินที่พ่นระเหยควันขาวฟุ้งกระจาย แสงไฟนีออนสีส้มเหลืองและเขียวสะท้อนผ่านผลึกเกล็ดหิมะบางเบาที่ร่วงหล่นลงมาทับถมแนวกำแพงศิลาหนาเตอะดึงทัศนียภาพหนาวเหน็บ โนอากระชับผ้าพันคอหนาสีกรมท่าคอยคีย์ปรับความเสถียรข้อมูลระบบวิจัยอย่างใส่ใจดูแลในขณะที่สแนปโซและเกโควาชิเฝ้าระแวดระวังแนวขอบเขตสนาม เรย์ใช้สายตาสีเทาสุขุมนิ่งสนิทจับจ้องวิเคราะห์ริ้วสัญญาณรหัสสีทองของตารางกริดจำลองกู้ภัยผ่านโมโนเคิลขวา พยายามประสานพลังเจตจำนงรบ Willpower ขั้นเต็มร้อยเปอร์เซ็นต์เพื่อชะล้างลบรอยร้าวมลพิษ Anomaly ข้ามขีดปวดล้าสะสม การเดินทางร่วมทาง VGC ในชีวิตจริงที่เกิดจากความผูกพันและสายสัมพันธ์ร่วมรบพยุงระดับความมั่นคงสมาธิประสาทช่วยรบ Tier 3 ให้คงที่เสถียรรหัสเต็มพิกัดโดยไม่มีการยอมจำนนต่อกฎบิดเบี้ยวมิติ
