# ตอนที่ 181: "รอยร้าวมิติสุดท้ายที่ประตูแปด"

*ครืน... ครืน...!!!*

เสียงคำรามกระหึ่มกึกก้องลื่นไหลข้ามหุบเขาและห้วงเวลาดังสะท้อนปฐพีเหนือยอดเขากำเนิด ทันทีที่เรย์ยอมรับหน้าที่ผู้พิทักษ์นิรันดร์ทางเลือกที่สามเพื่อพยุงรอยต่อโลกคอร์ ม่านพลังงานสีม่วงทมิฬขนาดมหึมาและรอยริ้วแหลมคม ‘ประตูด่านแปดห้วงมิติลึก’ (Eighth Gate Void Core) พุ่งปะทุเปิดตัวขึ้นกักลานเจดีย์โบราณสลัก ดึงเศษเกล็ดหิมะสีม่วงเข้มให้ลอยว่อนร่วงหล่นปะทะแนวกำแพงหินทรายจนสลายตัวแตกละเอียดกลายเป็นแก้วเปราะบาง

โนอาคลี่ยิ้มหวานสลับสแกนข้อมูลด้วยแว่นประคองวิจัยรายงานน้ำเสียงเครียดตื่นเต้นสั่นระริก “เรย์! ประตูด้านที่แปดซึ่งเป็นประตูด่านคอร์สุดท้ายพังทลายรหัสสะสมแล้วค่ะ! ค่า Anomaly พุ่งพีกแตะร้อยละเก้าสิบเก้าแล้ว! นี่คือภัยวิบัติลบข้อมูลโลกจริงระดับสูงสุดคุกคามดวงจิต!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทภายใต้แว่นตาโมโนเคิลขวาประเมิน VGC ในประสาทช่วยรบ Tier 3:

[ตรวจพบรัศมีรอยแยกสุดท้าย: ประตูด่านแปดความว่างเปล่ากางระบบ (Eighth Gate Void Core Active)]
[ระดับการกู้คืนความทรงจำคงเหลือ: 90% (คงที่เสถียร)]
[ขีดวิเคราะห์ความทรงจำชิบูย่า: เสถียรร้อยเปอร์เซ็นต์]
[ระดับความมั่นคง Willpower: พุ่งแตะระดับสัมบูรณ์สูงสุด]

*‘ประตูด้านแปดกวาดมลพิษสูงสุด...’* เรย์จดจ้องขวับวิเคราะห์ VGC สลับขั้วในประสาทประสาทช่วยรบ *‘นี่คือไฟนอลด่านรบที่จะทดสอบเจตจำนงรบแชมเปี้ยนชิบูย่าทั้งหมด! ภายใต้ห้วงดินมลพิษประตูด่านแปด หากเราพึ่งพาแต่คำนวณกราฟสแกนในจอคอมพิวเตอร์ จะโดนพลังสลายมวลสารบดชนสลบหมดสภาพการรบตั้งแต่เทิร์นแรก! เราต้องสั่งให้เกโควาชิและสแนปโซการ์ดีทองคำร่วมประสานงานรบข้ามมิติชะล้าง!’*

“ลูคัสครับ! จูลส์! เตรียมพร้อมประสานทีมรุกเพื่อพังม่านประตูด่านแปดกู้ภัยแกนคอร์เดี๋ยวนี้เลย!” เรย์กู่สั่งวาจาน้ำเสียงเฉียบขาดกระชับทรงพลังกึกก้องลั่นวิหารลึก

ขบวนกู้ภัยรุกขยับก้าวเท้าย่ำฝุ่นข้ามทางลาดประตูด่านแปดเปิดฉากศึกสงครามไคลแมกซ์สุดท้ายอย่างหาญกล้าเด็ดเดี่ยวในทันที!
รอบข้างแท่นศิลาปฐมกาลในดินแดนสลับมิติความว่างเปล่าลอยเคว้งคว้างไร้น้ำหนัก ท้องฟ้าเปลี่ยนเป็นสีม่วงทมิฬสลับกับริ้วเส้นตารางกริดสีทองคำเหลวของระบบระบบที่สั่นเตือนเตือนระเบิดถี่ โนอากอดกล่องวิจัยเฝ้ามองความผันผวนอย่างลุ้นระทึกดวงตาคลอน้ำใส ลูคัสและไฟแอโร่สแตนด์บายขบวนรบร่วมจูลส์อย่างเด็ดเดี่ยวเปี่ยมสัมผัสรอยต่อ เรย์เพ่งสบสายตาสีเทาสุขุมผ่านแว่นตาโมโนเคิลขวาประเมินสูตร VGC ในหัวประสาทสมอง Tier 3 พยายามควบคุมระดับความเสถียร Willpower เต็มขีดสัมบูรณ์ 100% ความเป็นแชมเปี้ยนชิบูย่าดั้งเดิมช่วยนำความอุ่นประสานชะล้างไอ Anomaly มลพิษโดยไม่ยอมพ่ายแพ้ต่อบทบาทลบล้างกติกาโลกเสมือนจริงสะสะสม
