# ตอนที่ 122: "การทำลายฐานข้อมูลวิจัยของอาจารย์ฌอน"

*ตูม... เพล้ง... บึ้ม...!!!*

เสียงระเบิดพลาสม่า Void ดังลั่นปะทุถล่มผนังฝั่งตะวันออกเฉียงเหนือของอู่ซ่อมรถร้างสล้างพังทลายสลายตัวเป็นละอองแก้วใส สมาชิกกลุ่มเสียงประสานเสื้อคลุมเทาสี่คนกระโจนพุ่งตัวพังแผงคุม สาดกระแสคลื่นรังสีความแปรวนแปรสีม่วงเข้มเข้าใส่ ‘เครื่องคอมพิวเตอร์เมนเฟรมฐานข้อมูลสำรอง’ (Mainframe Research Terminal) ของโนอาที่ตั้งประคองวิจัยอยู่กลางห้องส่งไอร้อนทมิฬ

กระแสไฟฟ้าลัดวงจรไฟฟ้าระเบิดสว่างจ้าปะทุช็อตเปรี้ยงเปรี้ยง ควันดำหนาทึบพุ่งอบอวลหลอมเหลวแผ่นเมนบอร์ดรหัสและละลายชิปข้อมูลสัจธรรมจนแตกละเอียดร้าวหลุดพ่นเศษแก้วร่วงกวาด

“อย่านะ! เครื่องข้อมูลของพวกเรา! ฐานข้อมูล 15 ปีของอาจารย์ฌอน!” โนอาอ้าปากกรีดร้องเสียงหลงกระโจนตัวกรูดไปพยายามพยุงปกป้องเครื่องคอมพิวเตอร์ แต่มวลพลาสม่าร้อนจัดดีดปัดปัดร่างบางกระเด็นลื่นไถลไปบนลานกระเบื้องส่งสัญญานระเบิดใหญ่ร้าวลึก

หน้าจอสแกนข้อมูลวิจัยแสดงข้อความสีแดงเตือนภัยดับสลายสลายแกน:

[รหัสวิกฤตความเสถียรข้อมูล: ตรวจพบรังสีความว่างเปล่าล้างแกนสแกน (Data Cleanse Protocol Active)]
[ฐานข้อมูลการวิจัยรอยแยกมณฑลเหนือสิบห้าปี (Professor Sean's Legacy Database): ถูกทำลายล้าง 100% (Deleted Permanent)]
[ระบบประสาทสัมพัทธ์วิจัย: สลายขีดจำกัดประมวลผล]
[สถิติสแกนจุดบกพร่องบอส: ปิดตัวถาวร]

“ไม่... ไม่จริง... ฐานข้อมูลวิจัยสิบห้าปีของอาจารย์ฌอน...” โนอาทรุดตัวลงคุกเข่าเอาหน้าผากแตะพื้นดินทรายทุบเศษเหล็กเบา ๆ เสียงสั่นสั่นระริกน้ำตาหลั่งไหลโฮเป็นสายเดือดพล่าน “ข้อมูลประวัติศาสตร์ Firewall และแผนรอยแยกทั้งหมดที่อาจารย์กู้รวบรวมกู้ชีวิตจนร่างสลาย... โดนพวกมันล้างลบรหัสทิ้งหายสาบสูญไปหมดสิ้นแล้วค่ะเรย์... ฉัน... ฉันไม่เหลือปัญญาที่จะช่วยพยุงเธอสู้รุกอีกแล้ว... แผนวิจัยพังพินาศหมดสิ้น”

โนอาร้องไห้สะอึกสะอื้นตัวสั่นเทาอย่างบอบช้ำล้นพ้นจิตวิญญาณ ความหวังลบล้างวิบัติภัยที่ประคองวิจัยมาตลอดชีวิตสลายวับกลายเป็นเศษขี้เถ้าถ่านหินดำเกลื่อนกลาดพื้นอู่ร้างสลัว

เรย์ก้าวเท้าสั้น ๆ เดินเข้ามาทรุดตัวยืนกอดอกกุมหลังโนอาช้า ๆ นัยน์ตาสีเทาสุขุมนิ่งสนิทภายใต้โมโนเคิลขวาประกายตาเฉียบคมเด็ดเดี่ยว *‘พวกมันล้างข้อมูลการวิจัยเพื่อบีบให้พวกเราหมดสิ้นหนทางประมวลผลประสาท... แต่มันไม่รู้หรอกว่า... สถิติวิจัย VGC สิบห้าปีทั้งหมดได้รับการสลักรหัสบันทึกสเตรตจีลงลึกในสมองประสาท Tier 3 ของฉันสำเร็จเสร็จสิ้นร้อยเปอร์เซ็นต์แล้ว! ตราบใดที่ฉันยังยืนหยัดรบ... วิถีการพิทักษ์ของฌอนก็ไม่มีวันสลายสิ้น!’*

“โนอาครับ...” เรย์เอ่ยน้ำเสียงเรียบนิ่งอบอุ่นมั่นคงกราบ “เงยหน้าขึ้นเถอะครับ ฐานข้อมูลของอาจารย์ฌอนไม่ได้สูญสหายไปไหน... แต่มันฝังอยู่ในสายตาและหัวใจรบของพวกเราทุกคนแล้วในวินาทีนี้! ผมจะปราบพวกมันเอง!”

เรย์สบประสานนัยน์ตาเรียบสุขุมมั่นคงกับโนอาพร้อมรับมือศึกสุดท้ายอย่างน่าทึ่งเด็ดเดี่ยวทันทีอย่างมีสติสัมปชัญญะ!
เครื่องเมนเฟรมฐานข้อมูลสำรองวิจัยรอยแยกมณฑลเหนือสิบห้าปีของอาจารย์ฌอนโดนคัลต์ลอบวางระเบิดพลาสม่า Void ละลายละเอียดสลายข้อมูลถาวร 100% โโนอาทรุดเข่าร้องไห้สิ้นหวังหมดปัญญาประมวลผลประคับประคอง เรย์ก้าวรับหน้าที่ผู้พิทักษ์แบกรหัสฌอนกู้โลกธรรมชาติจริง
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
