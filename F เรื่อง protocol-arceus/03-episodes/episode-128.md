# ตอนที่ 128: "การสบสัมผัสกริดแปรปรวนด้วยตนเอง"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดพลาสม่าดาวตก *Meteor Mash* ของเมทากรอส (Anomaly Metagross) พุ่งซัดถล่มรุนแรงสะเปรี้ยวปีกเหล็กแตกกระจาย เศษศิลาทรายและละอองเหล็กเหลวลอยละลิ่วตลบพังทับระเบียงศิลา เรย์ก้าวเท้าพริ้วไหวหรี่นัยน์ตาสีเทาสุขุมนิ่งสนิทจ้องมองฝ่าเขม่าดำหนาทึบด้วยใจจริง

โดยไม่มีแผงหน้าจอดิจิทัลสแกนสถิติช่วยเหลือ... ทว่าดวงตาสีเทาของเรย์ในก้นบึ้งประสาทกลับจับริ้วประกายสีม่วงและสีทองคำเหลวที่สั่นกริบรอบตัวได้เด่นชัดอย่างน่าอัศจรรย์ — **เขาสามารถมองเห็นแนวกริดมิติตำแหน่งบิดเบี้ยว (Void Grid Lines) ของโลกจำลองใบนี้ได้ด้วยสัมผัสประสาทสายตาของตนเองอย่างสมบูรณ์แบบเสถียร!**

รอยแยกมิติสีทองริ้วสะกดชีพจรกะพริบแจ้งรหัสโชว์ตรงข้อต่อสะโพกขวาของเมทากรอสยักษ์สี่แขนกบดานรหัสกู้ชีพ

“สแนปโซ! ยิงกงล้อไฟทองคำ Golden Flame Wheel โฉบกระแทกข้อต่อขวาเพื่อสลายแกนพลังจิต! อีวุย! ใช้ Quick Attack (จู่โจมว่องไวฉับพลัน) พุ่งเป็นเส้นตรงตัดผ่านกริดสีทองขวาด้านล่างเบี่ยงพิกัดดักทาง!” เรย์สั่งการประสาทเสียงเฉียบขาดกระชับลั่นกึกก้องสะท้าน

สแนปโซการ์ดีทองคำกู่คำราม ทะยานตัวหมุนประกายกงล้อไฟอัคคีทองคำเหลวบริสุทธิ์เจิดจ้าพุ่งชนกระแทกเข้าใส่แกนคริสตัลม่วงดำที่สะโพกขวาของเมทากรอสอย่างแม่นยำรวดเร็วปานไฟฟ้าผ่า เสียงแตกเพล้งใหญ่ของแกนสสารละลายไอความว่างเปล่าลอยฟุ้งกระจาย ในขณะเดียวกัน อีวุยตัวจ้อยซอกวิ่งความเร็วตัดขวางมิติช่วยสกัดจังหวะก้าวสะดุดของมันจนเมทากรอสยักษ์สี่แขนเสียหลักลื่นไถลทรุดคุกเข่าล้มคว่ำราบไปบดกระแทกเสาหลักร้องเสียงหลงเจ็บปวดเลือดทองลดฮวบดิ่งเหลือก้นรังพินาศสลาย!

*ตูม... บึ้ม...!!!*

แรงระเบิดชะล้างไอแปรปรวนสีม่วงทมิฬสลายตัวสลายหายวับสัจธรรม เมทากรอส Anomaly คืนร่างโลหะปกติไร้ประกายบ้าคลั่งหลับสนิทคว่ำหน้าไปบนดินทรายอย่างหมดสภาพการรบอย่างงดงามและราบคาบสิ้นเชิง

บิชอปวาลด์ยืนหน้าถอดสีซีดเผือดสะดุดรนรานตื่นตระหนกขยับถอยหลังพิงปากเหวระเบิด “เป็นไปไม่ได้!... สบสัมผัสกริดรอยร้าวมิติด้วยตาเปล่ากู้ปิดระบบได้โดยไม่มี Firewall สนับสนุนช่วยเหลือรบงั้นรึ! เจ้าเด็กนี่มัน... ปีศาจเหนือมหาเทพจำลองชัด ๆ !”

เรย์ก้าวเท้ายืนตรงอย่างสง่างามเคียงคู่สแนปโซการ์ดีทองคำและคู่หูร่วมรบด้วยแววตาสีเทาคมกล้าพิทักษ์โลก *‘ชัยชนะในการประลองโดยปราศจากระบบจำลองช่วยสแกน... บทพิสูจน์แห่งเจตจำนงแชมเปี้ยนโลกในชีวิตจริงเสร็จสมบูรณ์ร้อยเปอร์เซ็นต์แล้ว! ลูคัส... ฉันกำลังบุกไปช่วยแกแล้ว!’*
สัมผัสสายตาจริงของเรย์มองเห็นขอบเขตแนวกริดมิติตำแหน่งบิดเบี้ยว Void Grid Lines กะพริบทองขวา สั่งสแนปโซกงล้อเพลิงทองคำชนข้อสะโพกขวาล้างไอพลังจิต เมทากรอส Anomaly เสียจังหวะลื่นไถลทรุดเข่าราบราบคาบสลบล้มคว่ำราบสิ้นเชิง บิชอปวาลด์ตะลึงลนลานตื่นตระหนกขวัญบิน
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
