# ตอนที่ 89: "อาจารย์กึ่งเวลาและทางชี้แนะ"

ค่ำคืนคืบคลานลื่นไถลปกคลุมทิวเขาไผ่ทองเหลืองท่ามกลางมวลหมู่ดาวระยิบระยับลอยเกลื่อนน่านฟ้ากว้าง แสงไฟสปอตไลท์จากแท็บเล็ตวิจัยของโนอาสะท้อนส่องโครงสร้างข้อมูลรหัสสมดุลมิติกะพริบถี่เป็นแนวกราฟสีฟ้า เรย์ โนอา และลูคัสนั่งล้อมรอบโต๊ะไม้ไผ่โบราณในบ้านพักศิลา โดยมีคีธอดีตแชมเปี้ยนยืนชี้พิกัดแผนที่กระดาษเก่าสลักขริบทองเหลืองนำเสนอทิศรบรุกอย่างตั้งมั่นในดวงใจ

"แกนหลักรอยแยกภูมิภาคที่สี่ (Origin Core Node) ตั้งอยู่ที่ใจกลาง ‘วิหารป่าไผ่สลักเหล็ก’ (Steel-Etched Temple) ทิศตะวันออกเฉียงเหนือลึกสุด..." คีธขยับจัดแว่นตาโมโนเคิลอธิบายน้ำเสียงหนักแน่นเป็นจังหวะมีพลัง “ที่ตรงนั้นมีอสุรกายความว่างเปล่าระดับสี่ ‘คิงดรา’ (Kingdra) ตัวเปิดพายุคลื่น Void ประจำการเฝ้าระวังภัยรอยต่อมวลธาตุอยู่เพื่อคอยสะกัดปิดกั้นผู้บุกรุก”

“คิงดรา Anomaly งั้นเหรอครับ?” ลูคัสถามเสียงตื่นตระหนกขมวดคิ้ว “ในตำรา VGC คิงดรามีสเตตัสธาตุน้ำและมังกรที่ได้ผลดีเยี่ยมในสภาพอากาศฝนตก Drizzle รวดเร็วปานสายฟ้าฟาดกวาดพลังชีวิตล้นพ้น!”

“ใช่แล้วเจ้าหนุ่มลูคัส...” คีธพยักหน้าและหันสบตาเรย์อย่างจดจ่อด้วยประกายดวงตาแหลมคม “อสุรกายคิงดราตัวนี้ใช้กระแสคลื่น Void ยืดหยุ่นต้านทานดาเมจพิเศษและสปีดสูงลื่นไหลเหนือขีดระบบสแกน หากพวกเธอไม่มีวิธีตรึงสภาพเหนี่ยวนำแรงโน้มถ่วงหรือสลัดล้างสสารห้วงว่างเปล่าชำระล้าง ยุทธวิธีรบแบบ VGC ธรรมดาจะล้มเหลวราบคาบเหมือนพายเรือทวนกระแสน้ำเชี่ยวพังทลายรหัสรบหมดสิ้นเชิง!”

คีธก้าวเท้ายื่นตราสลักทองเหลืองโบราณส่งต่อให้แก่เรย์อย่างเงียบเชียบ — มันคือ **กุญแจถอดรหัสความเสถียรป่าไผ่ (Bamboo Stabilizer Core Key)** อุปกรณ์เสริมกู้ระบบที่ฌอนทิ้งไว้

“ข้าจะร่วมศึกเดินทางก้าวเข้าไปเป็น ‘เมนเทอร์’ (อาจารย์สอนแผนรบ) นำทางสกัดสสารแปรปรวนให้พวกเธอชั่วคราวในวิหารใต้ดินลึก...” คีธเอ่ยน้ำเสียงเด็ดเดี่ยวเด็ดขาดมั่นคง “ข้าจะคอยใช้ไคริวมังกรยักษ์เปิดทัพคุมกระแสลมสภาพอากาศพายุทรายต้านฝนคลื่น Void ส่วนเรย์... เธอต้องใช้สแนปโซและเกโควาชิปัดกวาดล้างรอยแยกปิดประตูคริสตัลให้เสถียรร้อยเปอร์เซ็นต์”

หน้าจอทองคำระบบประสาทในสมองของเรย์เตือนรหัสข้อมูลยกระดับความเชื่อมั่น:

[ปลดล็อกสถานะร่วมรบพิเศษ: เมนเทอร์คีธ (Mentor Keith Joined)]
[พลังสมาธิประมวลผลระบบ VGC Tier 3: เสถียร 90% (ขีดการรับรู้สิ่งบิดเบี้ยวรอบกายกว้างขวางขึ้น)]

เรย์กำตราสลักทองเหลืองแน่น สัมผัสอุ่นชะล้างไอเหนี่ยวนำขัดขวางขีดประสาท *‘การต่อสู้ประสานสองแชมป์เปี้ยน VGC... นี่คือแผนการจัดทัพที่แข็งแกร่งที่สุดในระดับภูมิภาคตะวันออกเฉียงเหนือ! คิงดราน้ำมังกรแปดตา... เตรียมตัวหมดสภาพการรบรุกรานได้เลย!’*

"ตกลงครับคุณคีธ..." เรย์ตอบน้ำเสียงนิ่งสุขุมมั่นคงกราบ “พวกเราจะบุกฝ่าพายุไปถล่มแกนรอยแยกและปิดวิกฤตความเสื่อมมณฑลเหนือนี้ให้เด็ดขาดร่วมกันครับ!”

โนอาและลูคัสพยักหน้าพร้อมสบตากระชับเป้ร่วมเดินทางก้าวหน้าลุยรบ!
แสงไฟตะเกียงน้ำมันวิจัยสะท้อนพิกัดเส้นตารางกริดสีทองและแผนที่ทองเหลือง คีธตกลงใจร่วมเดินทางนำทัพในฐานะเมนเทอร์เพื่อสยบ Anomaly Kingdra นกยักษ์ไคริวจะคอยปล่อยทรายเหล็ก Sandstorm คุมสภาพอากาศฝนคลื่น Void ส่วนเรย์ต้องสั่งคู่หูรบสอยแกนวิหารลึกใต้ดิน เรย์ประสานกุญแจทองเหลืองพร้อมสัมประสิทธิ์ประสาท Tier 3 ที่เสถียร 90% เตรียมลุยประจันหน้าศึกสุดท้ายภูมิภาคที่สี่
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
