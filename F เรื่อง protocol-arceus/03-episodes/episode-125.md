# ตอนที่ 125: "ประจันหน้าเมทากรอส Anomaly สี่แขนยักษ์"

*ครืน... ครืน... เพล้ง...!!!*

เสียงแยกถล่มร้าวละเอียดของเพดานศิลาเหล็กกล้าใต้ดินวิหารก้องกังวานสะเทือนสะท้านทิศเหนือ ท่ามกลางกระแสความร้อนระอุเดือดของบ่อเหล็กถลุงเหลวสีส้มแดงฉานกว้างกว่าร้อยเมตร แม่ทัพใหญ่ลัทธิเสียงประสาน ‘บิชอปวาลด์’ (Bishop Vald) ในเสื้อคลุมสลักขริบหน้ากากเหล็กสีม่วง Void ยืนเยือกเย็นตบเท้าอยู่หน้าเสาส่งกำลังแกนกลางรอยแยกยักษ์สบตาตรวจรบ

บิชอปวาลด์สลัดปล่อยกูบโปเกบอลสีดำขลิบทองเหลืองลงดินทรายลานเหล็กเกิดประกาย Anomaly

*“กิ๊ง... โฮก...!!!”*

เงามวลเหล็กกล้ายักษ์สี่แขนโผบินลอยตัวต่ำเหนือทะเลบ่อเหล็ก — โปเกมอนประเภทโลหะและพลังจิต ‘เมทากรอส’ (Metagross) ทว่าร่างกายโลหะหนาของมันกลับมีขนาดใหญ่โตมหึมาตระหง่านกว้างกว่าเจ็ดเมตร แขนเหล็กหนาทั้งสี่ข้างหุ้มเคลือบด้วย **แร่คริสตัลแก้วสีดำอมม่วงเข้มแผ่ไอคลื่นรังสีความว่างเปล่าเย็นยะเยือก** บิดกวนอากาศร้อนรอบตัวให้ตกผลึกเป็นแก้วเปราะบางฉลุย — ‘เมทากรอส Anomaly (ประเภทโลหะ/พลังจิต/Void)’! ดวงตาสีแดงฉานรูปตัวเอกซ์ (X) สว่างโร่สแกนตรวจขบวนรุกเรย์ดุร้าย

“ยินดีต้อนรับแชมเปี้ยนไร้พ่ายซีโร่...” บิชอปวาลด์สั่งเสียงแหบกึกก้องลั่นสะท้าน “จงเตรียมพร้อมยอมโดนลบข้อมูลระบบที่นี่ซะ! เมทากรอส! เปิดใช้งานคลื่นพลังจิตทำลายล้างมิติ Psychic Void Field (ทุ่งคลื่นพลังจิตว่างเปล่า) สยบล้างระบบประสาทของมันเดี๋ยวนี้!”

สิ้นเสียงสั่ง เมทากรอสยกสี่แขนเหล็กประกบกันแผ่คลื่นรังสีสะเทือนพลังจิตแปรปรวนสีม่วงเข้มเป็นวงแหวนมิติหมื่นรัศมีกวาดคลุมรอบลานรบ ส่งผลให้แผงหน้าจอทองคำระบบประสาทช่วยรบ Tier 3 ในหัวประสาทสมองของเรย์ เกิดอาการกลายสภาพสลายกระแสไฟฟ้าลัดวงจรเสียงดังวิ้ง ๆ ช็อตรหัสกะพริบตาถี่ยิบแทบมองไม่เห็นตัวเลขวิเคราะห์ข้อมูลจุดอ่อนบอส!

[รหัสวิกฤตสูงสุดระบบช่วยรบ: ตรวจพบคลื่นพลังจิตแร้นแค้นรบกวนขีดประสาท (Psychic Interference Alert)]
[ขีดเสถียรระบบช่วยรบ Tier 3: ดิ่งลงเหลือเพียง 12% (ลัดวงจรเกือบสมบูรณ์)]
[กฎบิดเบี้ยว: หน้าจอแสดงผลพารามิเตอร์ VGC สถิติและจุดอ่อนบอสปิดตัวชั่วคราวร้อยเปอร์เซ็นต์!]
[ความทรงจำแชมเปี้ยน: ล็อกประสาทหนาแน่นคงค้าง 90%]

*‘คลื่นพลังจิตสลายสัญญาณหน้าจอช่วยรบ...’* เรย์กิตเขี้ยวแน่นหรี่ดวงตาสีเทาสุขุมนิ่งสนิทเผชิญหน้าอสุรกายยักษ์สี่แขน *‘บิชอปวาลด์จงใจใช้สนามพลังจิตทำลายสิทธิ์วิเคราะห์สมอง เพื่อบีบให้ผมต้องรบในสภาวะตาบอดปราศจากการคาดสถิติ VGC ดั้งเดิม! สเตตัสและดาเมจของเมทากรอสตัวนี้ทรงพลังทั้งรับและรุกขีดสุด... หากก้าวเท้าพลาดแม้แต่เสี้ยววิเดียว เกโควาชิและสแนปโซจะโดนกรงเล็บอัดกระแทกสลายร่างหายวับทันที!’*

“เกโควาชิ! สแนปโซ! โโบกุเระ! ตั้งแถวข้ามระนาบรูปขบวนป้องกันขั้นสูงสุดด่วน!” เรย์สะบัดข้อมือสั่งการน้ำเสียงเรียบนิ่งเด็ดเดี่ยวเปี่ยมประกายตาคมกริบเผชิญหน้าดวลเดือดขั้นประวัติศาสตร์ทันทีอย่างหาญกล้า!
บิชอปวาลด์แม่ทัพหน้ากากเหล็กคัลต์เสือกโปเกบอลปล่อยอสุรกายยักษ์สี่แขนเมทากรอส Anomaly โลหะพลังจิตกางทุ่งพลังจิตว่างเปล่า Psychic Void Field ทำลายระบบแสดงผลพารามิเตอร์ VGC หน้าจอรบประสาทช่วยรบ Tier 3 ของเรย์ดับร่วงสลัวกะพริบเอเร่อรหัสลัดวงจรเกือบสิ้นเชิง
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
