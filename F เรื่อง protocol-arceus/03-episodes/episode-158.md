# ตอนที่ 158: "การประสานเจตจำนงรบสยบแรงกดดัน"

*วิ้ง... ติ๊ด... เปรี๊ยะ...!!!*

เสียงกระแสคลื่นมิติลัดวงจรกระสับกระส่ายเกร็งสั่นในหัวประสาทสมองของเรย์กะพริบถี่ยิบระเบิดช็อต แผงหน้าจอโฮโลแกรมทองคำระบบช่วยรบ Tier 3 ดับสนิทลงอย่างสมบูรณ์แบบถาวรชั่วคราว ดับค่าวิเคราะห์พารามิเตอร์ VGC สลัดสถิติคำนวณสลายวับไปกับตาท่ามกลางมวลพายุทรายสีม่วงเข้มระเบิดถล่มพัดฝ่าลานเหล็กน้ำแข็งร้าวละเอียดกวาดมลพิษ

เรย์ขยับก้าวเท้ายื่นมั่นคง ยื่นมือขวาถอดกดปิดการแสดงผลหน้าจอโมโนเคิลขวาลงสลัดพิกัด ดึงสายตาเปล่าธรรมชาติจริงเพ่งสบจ้องมองฝ่าหมอกดำหนาทึบประคองเจตจำนงแชมเปี้ยนโลกพิทักษ์มิตรภาพข้ามขีดความล้าสะสม

“ปิดระบบปิดกั้นหน้าจอช่วยรบ...” เรย์พึมพำกระซิบเสียงเรียบนิ่งสุขุมนิ่งสนิทดวงตา *‘ตอนแข่งขัน VGC ที่ชิบูย่า... ยามหน้าต่างกราฟสถิติจางหายลบแกนขัดสนวิเคราะห์ ผมคิดอย่างเดียวคือการถอนตัวยอมสละเกมสลบแพ้... แต่ในโลกธรรมชาติจริงใบนี้... ความรักและความไว้วางใจของเพื่อนแท้ลูคัสคือคีย์สำคัญที่ไม่อาจยอมแพ้ลบข้อมูลได้เด็ดขาดชั่วยาม!’*

เรย์จดจ้องมองแนวพิกัดขยับตัวประมวลผลฟิสิกส์ธรรมชาติจริงของบันกิรัส (Anomaly Tyranitar) ยักษ์สี่ตาหินด้วยสายตาเปล่าละเอียดถี่ถ้วนดุจมีดโกน:

*กรวดทรายแร่หมุนปลิวเฉียงใต้ขวาด้วยสปีดแรงกดลม...*
*เกล็ดแก้วขวาของมันตรงรอยพับสะโพกกะพริบสว่างประจุสีม่วงเข้มเป็นจุดอ่อนคอร์พลังงาน Void!*
*สแนปโซแผ่ประกายไฟทองคำเหลวหมุนนำอากาศ Tailwind พัดชะล้างไอสนิมเหล็ก!*

นี่คือร่องรอยสัจธรรมรบสบจังหวะปิดเกมที่ไม่มีเครื่องสแกนจำลองใดมองเห็นได้แม่นยำประสาน!

“เกโควาชิครับ! สปริงตัวเหยียบเสาหินลม Tailwind หนุนความเร็ว! Aerial Ace ปาดดาบน้ำกายภาพสับเปรี้ยงเข้าข้อต่อสะโพกขวานั่น! สแนปโซ! พ่นเปลวอัคคีเพลิงทองคำเหลวพุ่งสลายประจุ Void สกัดล้างพายุดักทาง!” เรย์สั่งชี้นิ้วขวาน้ำเสียงเฉียบขาดกระชับทรงพลังกึกก้องลั่นเจดีย์ทองเหลือง

กบนินจาเกโควาชิและ Growlithe สแนปโซดีดตัวร่างสีทองแดงกระโจนขึ้นเวหาประสานงานร่วมรุก ชะล้างไอพายุทรายสีม่วงจนกระจายตัวลอยฟุ้งหลอมเหลวเป็นละอองทองคำระยิบระยับลอยระเหยสลัดรังสีมืดดับสูญสิ้นสลายพลังรบรุกเดี่ยวชี้ชะตากู้โลก!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
