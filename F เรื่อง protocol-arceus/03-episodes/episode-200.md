# ตอนที่ 200: "ผู้คุ้มกันนิรันดร์และเวลาที่ไม่เคยหยุดหมุน"

*ครืน... ครืน... เพล้ง...!!!*

เสียงแยกตัวดังกังวานสลับกระแสมิติลำดับสี่จากประตูเจดีย์ปฐมกาลทองเหลืองดังสะท้อนกังวานดึงจังหวะระทึกขวัญ ประตูรอยแยกแปดปิดผนึกสำเร็จสมบูรณ์กู้ชีพสะเสร็จสิ้นลบล้างรอย Anomaly ปลุกประจุความเสถียรโลก Firewall 100% คืนความเงียบสงบอุ่นคืนสู่ทวีปเหนือมณฑลแกลร่าถาวร

เรย์ก้าวเท้าเหยียบยืนอยู่บนลานศิลาแท่นเจดีย์สูงสุดปฐมกาล สบสายตากับหลวงพ่อเทนซินร่างจำลองมหาพิทักษ์และสแนปโซการ์ดีทองคำคู่ยากด้วยดวงตาสีเทาสุขุมนิ่งสนิทเปี่ยมสัจธรรมกู้โลก

“ผมขอยอมรับหน้าที่ ‘ผู้คุ้มกันนิรันดร์’ (Eternal Protector) ประคองระบบ Firewall คุ้มกันชีวิตโนอา ลูคัส และคู่หูสแนปโซของผมทุกคน...” เรย์ประกาศวาจาน้ำเสียงราบเรียบนิ่งสงบเด็ดเดี่ยวหนักแน่น “แต่ผมจะใช้พลังเจตจำนงรบ Willpower คนนอกคอร์ตรึงค้างรักษาระดับความจำ 90% และประคองนาฬิกาพกทองเหลืองเก่าแก่ของคุณพ่อไว้หมุนเดินหน้าบอกเวลาที่ไม่เคยหยุดหมุนชั่วนิรันดร์กาล!”

หลวงพ่อเทนซินคลี่ยิ้มกว้างกางขาสลัดยอมรับเจตจำนงแชมเปี้ยนซีโร่ “สมศักดิ์ศรีของแชมเปี้ยน VGC ที่มีหัวใจพิทักษ์มิตรแท้คนนอกระบบ... มหาเทพยอมรับทางเลือกที่สามแก้ไขแกนคอร์ของเจ้าอย่างสง่างามดีเสถียร!”

*วิ้ง... ติ๊ด... เปรี๊ยะ...!!!*

หน้าจอโฮโลแกรมทองคำระบบช่วยรบ Tier 3 เด้งตัวแจ้งข้อมูลปลดล็อกประจุรอยต่อสมดุลโลกเต็ม 100%:

[รายงานผลลัพธ์เลือกชะตา: ยืนยันหน้าที่ผู้คุ้มกันนิรันดร์ทำงาน (Eternal Protector Activated)]
[ระดับความคืบหน้า Progress ประสานแกนคอร์: คงที่คงเหลือ 100% (World System Stabilized)]
[ระดับความทรงจำชิบูย่าของเรย์: เสถียรรหัสเต็มขั้น 90% (Safety Memory Lock preserved)]
[สเตตัสความปลอดภัยเพื่อนร่วมเดินทาง: สมบูรณ์ปลอดภัยร้อยเปอร์เซ็นต์ถาวร]

โนอาวิ่งพุ่งกระโจนโอบกอดคอเรย์น้ำตาไหลพรากยิ้มหวานสดชื่นยินดีล้นพ้นขอบเวที “เรย์! เธอทำได้แล้วเว้ย! ขอบคุณนะเว้ยที่ไม่ยอมทิ้งพวกเรากลับโลกเดิมทอดทิ้งมิตรแท้! ฉันจะอยู่เคียงข้างเธอพิทักษ์โลกใบนี้ตลอดไป!”

ลูคัสยกนิ้วหัวแม่มือขวากางยิ้มกว้างปาดเหงื่อ “สุดยอดแชมเปี้ยนผู้ไม่เคยเลือกแต่กำหนดชะตาตนเองได้สมบูรณ์แบบมากเพื่อนรักเรย์! พวกเราจะอยู่ร่วมรบพิทักษ์แกนทวีปปิดอาร์คแปดสำเร็จดีเสถียร!”

เรย์คลี่ยิ้มกว้างละมุนอบอุ่นดวงจิตลูบหัวสแนปโซการ์ดีสองสัจธรรมกู่เห่าต้อนรับตะวันใหม่ ร่วมเดินทางก้าวพ้นวิหารแกนกลางกุมเป้าหมายสู่อนาคตสดใสพิทักษ์โลกธรรมชาติ Pokémon ถาวรชั่วนิรันดร์กาลอย่างกล้าหาญสงบสุขเด็ดเดี่ยวสัจธรรมปิดฉากบทประพันธ์เรื่อง ‘ผู้ถูกเลือกที่ไม่เคยเลือก (Protocol Arceus)’ อย่างสง่างามสมบูรณ์แบบเสร็จสิ้นร้อยเปอร์เซ็นต์บริบูรณ์!
เปลวแดดสีทองของตะวันใหม่ส่องสว่างเรืองรองเหนือยอดเจดีย์ปฐมกาลละลายกองหิมะหนาเหน็บให้ละลายระเหยกลายเป็นไออุ่นสลายมลพิษ Anomaly ทล้างสลายมิติความว่างเปล่าให้คืนเสถียรภาพถาวรร้อยเปอร์เซ็นต์ โโนอายิ้มหวานสดชื่นตาสว่างโอบแขนกอดรอบไหล่เรย์ด้วยหัวใจรักใคร่และรอยยิ้มปรีดา ลูคัสส่งเสียงหัวเราะอย่างโล่งใจประคองไฟแอโร่ร่วมยินดีในความก้าวหน้าสะสม เรย์หยิบนาฬิกาพกทองเหลืองเก่าแก่ของคุณพ่อขึ้นมาประคองจ้องมองเข็มวิจัยบอกเวลาที่ไม่เคยหยุดหมุนประสาน เจตจำนงรบแชมเปี้ยนที่ยอมรับหน้าที่ Eternal Protector ทางเลือกที่สามช่วยพยุงระบบ Firewall และกู้ชีวิตมิตรแท้ทุกคนไว้อย่างงดงามชั่วนิรันดร์
