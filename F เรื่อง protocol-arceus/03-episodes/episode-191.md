# ตอนที่ 191: "รหัสปลดล็อกความทรงจำร้อยเปอร์เซ็นต์"

*วิ้ง... ติ๊ด... เปรี๊ยะ...!!!*

เสียงกระแสกลไกจักรกลในนาฬิกาพกทองเหลืองเก่าแก่ที่คุณพ่อเหลือทิ้งไว้ในมือขวาของเรย์เริ่มหมุนย้อนกลับอย่างรวดเร็ว กะพริบตาเป็นแสงแสงส้มทองเรืองรองสอดประสานเข้ากับเส้นกริดสีทองคำเหลวใต้แผ่นศิลาเจดีย์ปฐมกาล ดึงเอาประจุกระแสอารมณ์ความทรงจำสุดท้ายที่ถูกระบบกรองล็อกไว้สิบเปอร์เซ็นต์สุดท้ายปลดปล่อยพวยพุ่งพีกร้อยเปอร์เซ็นต์ประสานระบบประสาทสมอง Tier 3

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิท ภายใต้หน้าจอวิเคราะห์โมโนเคิลขวาประเมินสเตตัสความเสถียรข้อมูลกู้ภัยสากล:

[ตรวจพบกุญแจถอดรหัสความทรงจำคุณพ่อ: สำเร็จ 100% (Absolute Memory Restored)]
[ขีดวิเคราะห์ความทรงจำแชมเปี้ยนชิบูย่าเรย์: คงเหลือเต็มขีด 100% (ไม่มีรอยลบข้อมูล)]
[ระดับการซ่อมแซม Firewall ระบบคอร์โลก: ปลดผนึกกู้ภัย Progress 98%]
[สถานภาพความเสถียร Willpower: เต็มพิกัดรบเสถียรดี]

*‘ความทรงจำทั้งหมดล้านเปอร์เซ็นต์กลับมาแล้ว...’* เรย์จ้องสบตาเกโควาชิและสแนปโซการ์ดีทองคำร่วมรบอย่างอบอุ่นดวงจิตลึกซึ้ง *‘คุณพ่อไม่ได้กักเก็บความจำของฉันไว้เพราะจงใจลบข้อมูล... แต่ฝังรหัสรักษาความปลอดภัย Safety Code ไว้ในนาฬิกาพกกระปุกเหล็กนี้ เพื่อรอให้ฉันปลุกค่า Willpower ประสานคู่หูสำเร็จจนถึงด่านสุดท้าย! กติกาทุกอย่างของระบบกู้ชีพนี้ประคองไว้ด้วยความผูกพันดั้งเดิมรอยยิ้มไม่ใช่ตัวเลข VGC คะแนน!’*

“โนอาครับ! ลูคัส! ความทรงจำของคุณพ่อปลดล็อกกุญแจทางเลือกที่สามเสร็จสมบูรณ์แล้วครับ!” เรย์ตะโกนชี้นิ้วขวาน้ำเสียงราบเรียบนิ่งสงบเด็ดเดี่ยวเปี่ยมประกายตาคมกล้า

โนอาและลูคัสคลี่ยิ้มหวานกู่ร้องยินดีล้นพ้นขอบสนามรบ ขบวนกู้ภัยรุกขยับก้าวเท้าย่ำฝุ่นฝ่าหิมะหนาบุกเจาะวิหารกลางสุดยอดเจดีย์ปิดผนึกทันทีอย่างหาญกล้ามั่นคงกู้โลก!
เปลวแดดสีทองของตะวันใหม่ส่องสว่างเรืองรองเหนือยอดเจดีย์ปฐมกาลละลายกองหิมะหนาเหน็บให้ละลายระเหยกลายเป็นไออุ่นสลายมลพิษ Anomaly ทล้างสลายมิติความว่างเปล่าให้คืนเสถียรภาพถาวรร้อยเปอร์เซ็นต์ โโนอายิ้มหวานสดชื่นตาสว่างโอบแขนกอดรอบไหล่เรย์ด้วยหัวใจรักใคร่และรอยยิ้มปรีดา ลูคัสส่งเสียงหัวเราะอย่างโล่งใจประคองไฟแอโร่ร่วมยินดีในความก้าวหน้าสะสม เรย์หยิบนาฬิกาพกทองเหลืองเก่าแก่ของคุณพ่อขึ้นมาประคองจ้องมองเข็มวิจัยบอกเวลาที่ไม่เคยหยุดหมุนประสาน เจตจำนงรบแชมเปี้ยนที่ยอมรับหน้าที่ Eternal Protector ทางเลือกที่สามช่วยพยุงระบบ Firewall และกู้ชีวิตมิตรแท้ทุกคนไว้อย่างงดงามชั่วนิรันดร์
