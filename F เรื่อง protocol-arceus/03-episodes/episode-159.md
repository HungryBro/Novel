# ตอนที่ 159: "อัคคีเพลิงทองคำชะล้างบันกิรัสและกู้ภัยลูคัส"

*โฮก...!!! ครืน...!!!*

เสียงคำรามต่ำแหลมหูแหบแห้งของบันกิรัส (Anomaly Tyranitar) ดังสนั่นลานวิหารศิลา มันยกสองแขนหินพ่นพายุทรายสีม่วงพยายามรวบรวมไอสสารความว่างเปล่าลบข้อมูลกู้ภัยทิ้ง เรย์ก้าวเท้ายื่นมั่นคง แสงรังสีสีทองจากตรากุญแจทองเหลืองกู้ระบบกะพริบตาเจิดจ้าดุจตะวันสว่างขจัดมลพิษ

“สแนปโซ! ปลดปล่อยพลังอัคคีเพลิงทองคำแปรปรวนขั้นสูงสุด Golden Flame Wheel (กงล้ออัคคีเพลิงทองคำเหลว) ชะล้างพายุน้ำแข็งทรายทมิฬมันเดี๋ยวนี้เลย!” เรย์ตะโกนชี้นิ้วสั่งการเสียงกึกก้องสะท้านปฐพี

สแนปโซการ์ดีทองคำกู่เห่ากังวานดุจสิงโตทอง ร่างดีดกระโจนหมุนเป็นกงล้อเพลิงทองคำเหลวบริสุทธิ์เรืองรองขนาดยักษ์สิบสองเมตรพุ่งทะลายตรงดิ่งลงมากระแทกแหกแนวรบหินมลพิษของบันกิรัสอย่างรุนแรงลื่นไหลปัดไอเย็นและทรายดำสลายวับสลายตัวสิ้นเชิง!

*ตูม... บึ้ม...!!!*

แรงปะทะระเบิดแสงทองคำเหลวเจิดจ้าลบล้างพัดพารอยแยกมิติทมิฬและไอหมอกทมิฬสีม่วงดำสลายร่างอสูรหินยักษ์ Tyranitar คืนเป็นละอองทองระยิบระยับปลิวระเหยสลัดรังสีคัลต์ดับหายวับคืนสภาพวิหารปกติสัจธรรม รอยแยกประตูหกปิดผนึกประสานรอยต่อคืนเสถียรภาพร้อยเปอร์เซ็นต์สมบูรณ์ดี

เรย์กระโจนร่างพุ่งตัวข้ามสะพานหินที่ละลาย คุ้ยทรายหิมะขยับใช้ตรากุญแจกระแทกปัดเศษผลึกแก้วน้ำแข็งม่วง ‘Void Ice Cage’ จนระเบิดแตกเพล้งใหญ่ ดึงประคองร่างลูคัสที่นอนสลบหนาวเหน็บพยุงอ้อมขึ้นพาดไหล่อย่างปลอดภัยช่วยชีวิตลุล่วงสะใจ!

[รายงานผลลัพธ์มหาเทพ: ปราบอสุรกาย Anomaly Tyranitar และช่วยเหลือลูคัสสำเร็จเสร็จสิ้น]
[ระดับความทรงจำกู้คืนสำเร็จ: สะสมรวมคงเหลือ 90% (ระบบ Firewall ปลอดภัย)]
[แกนรอยแยกมิติระดับหก: ปิดผนึกเรียบร้อยสะสม Progress แตะระดับ 80%]
[สเตตัสลูคัสและไฟแอโร่: ปลอดภัยรอยเชื่อมต่อกู้คืนสำเร็จ]

โนอาและจูลส์ทรุดวิ่งเข้ามายกมือทาบอกเสียงสั่นสั่นระริกตาคลอน้ำตาไหลยิ้มกว้างกางขากู้ภัย “ปิดแกนรอยแยกที่หกสำเร็จและช่วยลูคัสกลับมาได้แล้วเรย์! ชัยชนะครั้งนี้ยิ่งใหญ่ที่สุดล้านเปอร์เซ็นต์!”

เรย์คลี่ยิ้มกว้างอ่อนละมุนอบอุ่นใจเป็นครั้งที่สี่ ประคองพยุงไหล่เพื่อนรักลูคัสเคียงข้างคู่หูสแนปโซและมิตรแท้ทุกคนก้าวพ้นวิหารน้ำแข็งขาวมุ่งตรงกลับคืนสู่เมืองหลวงทันทีอย่างเด็ดเดี่ยวเปี่ยมสุข!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
