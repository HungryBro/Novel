# ตอนที่ 162: "เจดีย์โบราณและพิกัดแกนที่เจ็ด"

*ครืน... ครืน...!!!*

เสียงกระแสพายุมลพิษมิติมืดพัดกระหน่ำปะทะผนังศิลายอดเจดีย์โบราณสูงสุด ‘เจดีย์ปฐมกาลทองเหลือง’ (Genesis Brass Pagoda) ที่ตั้งตระหง่านสลัวอยู่เหนือน้ำแข็งยอดเขากำเนิด เรย์ โนอา และลูคัส ย่ำฝุ่นขรุขระก้าวเดินหยุดยืนอยู่หน้าปากประตูด่านหินศิลายักษ์แผ่รังสี Anomaly สีม่วงดำหนาแน่นขัดขวางขีดสแกนสัญญาณ

รอบแท่นศิลา หลวงพ่อเทนซินร่างจำลองมหาเทพยืนกอดอกประคองตรากุญแจทองเหลืองกู้ระบบเฝ้ารอรอบประลอง

“ยินดีต้อนรับกลับมาเจ้าหนุ่มเรย์... และผู้สมัคร Candidates ร่วมศึก...” เทนซินกล่าวเสียงทุ้มแฝงประกายเปี่ยมเมตตา “เบื้องหน้าประตูดินทรายหินนี้คือ ‘แกนรอยแยกมิติระดับเจ็ด’ (Seventh Core Rift) จุดเหนี่ยวนำใหญ่ที่ลัทธิทมิฬเริ่มรวมประจุสสารความว่างเปล่าลบข้อมูลธรรมชาติจริงทั้งหมด!”

โนอาพยักหน้าเครียดลงตรวจสแกนเครื่องสแกน “ระดับไอ Anomaly สลายแกนพุ่งขึ้นพีกร้อยละเจ็ดสิบหกสะสมแล้วค่ะเรย์! ระบบช่วยรบ Tier 3 และหน้าต่างจำลอง Error กะพริบถี่ยิบเกือบสลบหมดสภาพรักษารอยต่อ! ถ้าเราปิดรอยร้าวไม่ทันในเย็นวันนี้ ทวีปเหนือและโลกเสมือนจริงทั้งหมดจะดับวูบหายสาบสูญถาวร!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทภายใต้แว่นตาโมโนเคิลขวาประเมินสูตรระบบ:

[เข้าสู่แกนรอยแยกมิติระดับเจ็ด: เจดีย์ปฐมกาลทองเหลือง (Genesis Core Area)]
[ระดับการสั่นลัดวงจรของระบบคอร์ธรรมชาติ: เพิ่มขึ้นเป็น 35%]
[ค่าความเสถียร Willpower เจตจำนงรบเรย์: เต็มพิกัด 100%]
[ขีดวิเคราะห์ความทรงจำแชมเปี้ยนชิบูย่า: เสถียร 90%]

*‘แกนรอยแยกสุดท้ายที่เจ็ดคุมพิกัดสูงสุด...’* เรย์หลับตานิ่งสะกดสมาธิรบประเมิน VGC ในหัวประสาทรวดเร็วดั่งลูกศร *‘นี่ไม่ใช่เกม VGC เอาชนะทั่วไป แต่เป็นศึกตัดสินชะตากรรมเพื่อนพ้องคู่หู หากเราใช้อารมณ์ความอยากเอาชนะที่โลภโมโทสันบุกรบ จะโดนพลังสะท้อนลบล้างแกนชีวิตสลายร่างกลายเป็น Void Matter! เราต้องปิดระบบสแกนสถิติพึ่งพาเจตจำนงสายตาเปล่าชะล้างร่วมกับสแนปโซการ์ดีสองสัจธรรม!’*

“ผมจะบุกเจาะระบบสลายภัยพิบัติและช่วยทุกชีวิตร่วมทางปิดรอยแยกนี้ให้สวยงามที่สุดครับหลวงพ่อ!” เรย์กล่าวปักธงน้ำเสียงราบเรียบนิ่งสงบเด็ดเดี่ยว

หลวงพ่อเทนซินพยักหน้ายิ้มคลี่กางขาสั่งสลัดเปิดประตูด่านหินบุกศึกอาร์คเจ็ดรังบอสใหญ่ Tyranitar ทันทีอย่างองอาจมั่นใจ!
รอบบริเวณระเบียงศิลายอดเจดีย์ปฐมกาลทองเหลืองยังคงอบอวลไปด้วยกระแสพลังงานทองคำหนาแน่นที่สั่นเตือนสะท้อนกังวานสลับกับไอเย็นจัดจากพายุหิมะภายนอก เกล็ดหิมะบางเบาสล้างปลิวว่อนผ่านลวดลายสลักโบราณประดับยอดเสา โโนอากุมกล่องวิจัยสแกนค่าความเสถียรมิติด้วยใบหน้าเคร่งเครียดปนตื่นตัว ลูคัสคุมขบวนรบร่วมคู่หูไฟแอโร่อย่างตั้งใจพึ่งพา เรย์ใช้สายตาสีเทาสุขุมเพ่งมองริ้วรอยพับกริดสีทองใต้พื้นด้วยระบบช่วยรบ Tier 3 ปรับใจเจตจำนงรบ Willpower คงที่ร้อยเปอร์เซ็นต์ ความเป็นผู้สมัคร Candidates คนนอกระบบชี้ทางให้เขาเข้าใจความผูกพันดั้งเดิมที่ล้ำค่ากว่าสูญสิ้นสถิติ VGC ทวีคูณสะสะสม
