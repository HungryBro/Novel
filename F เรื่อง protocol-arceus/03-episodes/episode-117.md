# ตอนที่ 117: "เปลวพลาสม่า Void ของบูเบิร์นแปรปรวน"

*ครืน... ครืน... บึ้ม...!!!*

เสียงระเบิดพลาสม่าร้อนจัดปนกระแสมะเกลือสีม่วงดำแผ่ความชื้นไอแปรปรวนเย็นพวยพุ่งพัดผ่าข้ามแนวกรงระบายอากาศเหล็กกล้าโบราณตระหง่านทิศเหนือ เผยร่างอสูรเพลิงขนาดมหึมาตระหง่านตวัดกรงเล็บกระบอกปืนสลักลายดวงตาร้องไห้ — โปเกมอนราชาเตาหลอม ‘บูเบิร์น’ (Magmortar) ทว่าทั่วผิวหนังหนาสีแดงอมดำของมันกลับเคลือบด้วย **แผ่นแก้วเหลวเดือดปะทุสะท้อนไอความว่างเปล่า** ปลายแขนปืนพ่นไอร้อนประจุกระแสสสารสีม่วงเข้มปะทุวุ่นวาย — ‘บูเบิร์น Anomaly (ประเภทไฟ/Void)’!

*“โฮก...!!!”*

บูเบิร์นยักษ์กู่ร้องแผดเสียงต่ำ ปลายแขนปืนทั้งสองกระชับสั่งปล่อยพลาสม่าความว่างเปล่า *Void Plasma Burst* (ลำแสงแกนบิดเบี้ยว) ขนาดสี่เมตรยิงทะลายตรงถล่มสะพานลาดหินจนละลายกลายเป็นแก้วเหลวร่วงลงน้ำเสียงดังระงมเกรียวกราวสอยรหัส

โนอาปาดฝุ่นเขม่าถอนใจใบหน้าซีดเผือด “พลาสม่า Void ของบูเบิร์นตัวนี้แผ่รังสีความร้อนเหนี่ยวนำขัดขวางขีดสแกนขึ้นสูงสี่สิบเปอร์เซ็นต์เรย์! ตัวเลขสถิติ VGC และระบบช่วยรบ Tier 3 ลัดวงจรกะพริบไม่หยุดนิ่งเลยค่ะ! ถ้าเราไม่รีบสลายแกนปืนพลาสม่า ระบบความปลอดภัย Firewall จะล่มสลายล้างฐานข้อมูล!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทขยับแว่นตาโมโนเคิลขวาประเมินสเตตัสรบผ่านคลื่นเสียงระบบรบประสาทกู่เตือน:

[ตรวจพบสิ่งผิดปกติระดับสี่: บูเบิร์นแปรปรวน (Anomaly Magmortar) | เลเวล 67]
[คุณสมบัติพิเศษการโจมตี: ลำแสงสลายกฎสสาร (Void Plasma Burst)]
[กฎธรรมชาติบิดเบี้ยว: ดาเมจพิเศษประเภทคลื่นน้ำจะโดนพลาสม่าเดือดสะท้อนลบล้างดาเมจ 50% ชั่วคราว!]
[ขีดเชื่อมประสาทช่วยรบ Tier 3: เสถียร 90%]

*‘ลำแสงสลายกฎน้ำห้าสิบเปอร์เซ็นต์...’* เรย์จดจ้องมองจังหวะขยับตัวประมวลสมการในหัวประสาทรวดเร็วดั่งมีดโกน *‘ภายใต้สภาพแวดล้อมพลาสม่าเดือด ดาเมจธาตุน้ำของเกโควาชิจะสูญพลังเหนี่ยวนำล้นพ้นลำบาก หากเราไม่มีวิธีตรึงสภาพเหนี่ยวนำแรงโน้มถ่วง หรือสั่งดาเมจพิเศษล้างฟิลด์สนามด้วยไฟทองคำของสแนปโซสะเทินพลังงานชนิดเดียวกัน จะโดนบูเบิร์นยิงถล่มหมดสภาพการรบพังยับเยินแน่!’*

“สแนปโซ! ยิงไฟกงล้อ Golden Flame Wheel พุ่งชนต้านพลาสม่า Void ตรงปากกระบอกปืนของมันเพื่อล้างประจุพลังความว่างเปล่า! โโบกุเระ! หมุนระเบิดพื้นที่ Trick Room (พื้นที่บิดมิติ) บิดระนาบความเร็วขัดจังหวะการยิงพลาสม่าเดี๋ยวนี้!” เรย์สั่งการเสียงกึกก้องลั่นประจบขวา

สแนปโซการ์ดีทองคำกู่เห่ากังวาน ทะยานตัวหมุนเป็นวงล้อเพลิงทองคำเหลวเจิดจ้าพุ่งปะทะตรงเข้าใส่ปากกระบอกปืนของบูเบิร์น ความร้อนสีทองคำเหลวล้างและละลายไอความว่างเปล่าสีม่วงดำสลายปลิวว่อนกับตา โบกุเระปล่อยพลังมิติชมพูบิดกลับสปีดขวางจังหวะยิงพลาสม่าจนบูเบิร์นยักษ์ชะงักลื่นไถลทรุดเอียงตัวล้มคว่ำราบไปบดกระแทกเสาเหล็กร้องเสียงหลงเจ็บปวดล้นพ้น!

“ประสานการรบสลายพลาสม่า Void ได้ดีเยี่ยมมากเรย์! แกนปืนพลาสม่าของมันละลายพังทลายแล้ว!” โนอาตะโกนลั่นชี้นิ้วพยุงสแกนรังสีชื่นใจ

เรย์กระชับเสื้อคลุมกวาดตาสีเทาสุขุมมั่นใจ *‘การปะทะยังคงดุเดือด... เราต้องรีบเผชิญหน้าปราบมันและสกัดคอร์เตาหลอมให้เร็วที่สุดก่อนมิติทลวงลบข้อมูล!’*
บูเบิร์น Anomaly ยักษ์สี่แขนพลาสม่าเดือด Void Plasma Burst พ่นเกลียวพลาสม่าร้อนจัดละลายสะพานลาดหินเดือดพล่านลดดาเมจน้ำ 50% สแนปโซพุ่งกงล้อไฟอัคคีทอง Golden Flame Wheel ต้านสลายประจุปากกระบอกปืน โโบกุเระกากเกณฑ์ Trick Room สลับความเร็วถล่มทำลายดาเมจบอส
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
