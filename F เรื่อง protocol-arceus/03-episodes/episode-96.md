# ตอนที่ 96: "การบุกโจมตีของคัลต์และไคริว"

*ครืน... ครืน... บึ้ม...!!!*

เสียงระเบิดคลื่นมิติดังลั่นปานฟ้าผ่าหักพังถล่มจากเพดานหินปากอุโมงค์ทางใต้ลึกดึงความสงบเงียบยามดึกสลายตัวลงในพริบตาเดียว เรย์ โนอา และคีธ สปริงเข่าดีดตัวยืนขึ้นกุมโปเกบอลไว้แน่นหนา ฝุ่นปูนเหลืองและหินแร่ทองเหลืองถล่มกองลงปิดกั้นช่องทางหลัก เผยให้เห็นสมาชิกกลุ่ม ‘เสียงประสานแห่งความว่างเปล่า’ ในเสื้อคลุมเทาลายดวงตาฉิบหายเกือบยี่สิบคนก้าวเท้าย่ำฝุ่นฝ่าไอรังสี Void ดับสว่างดาหน้าเข้ามาปิดล้อมสิบด้านพร้อมสั่งปล่อยไอพลังงานบิดเบี้ยว

“ค้นหารอยเชื่อมสมองและลบข้อมูลวิจัยของพวกมันทิ้งเดี๋ยวนี้เลย!” นักพรตลัทธิผู้คุมด่านคำรามสั่งเสียงกร้าวชี้นิ้วสั่งขวางทางกู้ภัย

โปเกมอนประเภทพลังจิตแปรปรวนเนนดอล (Claydol) และสุนัขยมโลกเฮลการ์ (Houndoom) นับสิบตัวถูกปล่อยออกมาแยกเขี้ยวพ่นเปลวไฟมืดสะท้อนประกายความตายและยิงคลื่นพลังจิตทำลายล้างรอบลานศิลาโพรงหินกว้างจนเกิดไอร้อนปะทุตู้สแกนวิจัยสั่นคลอนลัดวงจรช็อตดังวิ้ง

“พวกสารเลวขัดขวางเควสหลัก!” ลูคัสตะโกนกู่สั่งยาโคมาบิบินกเพลิงบินฉีกสอยควบคุมขวาทันทีสกัดพิกัดป้องกันหลังหนาแน่น

คีธก้าวเท้ายืนสลัดผ้าคลุมเปลือกไผ่พลาดยืนประจันหน้าบังเรย์และโนอาไว้ ดวงตาคมภายใต้แว่นโมโนเคิลซ้ายประกายความเด็ดเดี่ยวระดับแชมป์เปี้ยนลีกของแท้ “เรย์! โนอา! คุมเครื่องบันทึกและประคองโซต้าถอยหลบเข้าไปในอุโมงค์ชั้นในด่วน! ตรงนี้ข้าจะกางด่านรักษาการณ์รับดาเมจสกัดกั้นพวกมันไว้เอง!”

“ไคริว (Dragonite)! ออกศึกรุกรับคำท้าประลองสงคราม!” คีธชี้นิ้วสั่งการตะโกนกังวานดุจฟ้าผ่าสะท้านปฐพีหินลึก

มังกรบินยักษ์เขียวสว่างไคริวแยกเขี้ยวคำรามลั่นสลัดปีกกางขากระโจนยืนบังรั้วทางลาดหินถ้ำ แผ่ไอออร่าหนาแน่นเกล็ดเกราะป้องกันหนาสิบชั้นส่องสว่างสะท้อนประกายข้อมูลทองคำ — ‘ไคริว เลเวล 78 โหมดตั้งรับสงคราม (War Defense Mode)’!

“ไคริว! เปิดใช้งาน Hurricane (วายุพัดทะลวง) ควบคุมสภาพอากาศพัดกระหน่ำฝูงเฮลการ์ให้ปลิวกระเด็นถอยหลังไปซะ! ตามด้วย Thunderbolt (สายฟ้าฟาดทลวง) ผ่าควบคุมสลายรังสีความว่างเปล่าขัดการเชื่อมประจุข้อมูลของลัทธิทมิฬ!” คีธสั่งการขยับจังหวะสปีดสูงสุดเฉียบขาด

ไคริวกู่มังกรกระพือปีกลอยสูงเหนือน่านฟ้าถ้ำ เรียกเกลียววายุพายุพัดหมุนวนขนาดใหญ่อบอวลฝุ่นดินปูนถล่มพัดกวาดชนร่างสมุนคัลต์และเฮลการ์กระเด็นลื่นไถลกระแทกผนังศิลาหินร้าวละเอียดร้องโหยหวนเจ็บปวดล้มคว่ำราบไปสี่ห้าตัว ในจังหวะประสานงาน ไคริวปล่อยกระแสสายฟ้าฟาดสีเหลืองสว่างเจิดจ้าผ่าเปรี้ยงเปรี้ยงลงถล่มทำลายกล่องปล่อยคลื่น Anomaly ของลัทธิระเบิดขึ้นปังใหญ่ทำลายไอสีม่วงทมิฬสลายหายวับไปกับตาชะล้างสนามรบ!

“สุดยอดไปเลยครับคุณคีธ! ประสานควบคุมดาเมจต้านล้างฟิลด์ได้หมดจดมาก!” ลูคัสกู่ร้องขณะยิงไฟสกัดพราดแนวรับขวา

เรย์จ้องมองดูขีดความสามารถการสั่งการประสานรหัสของคีธด้วยประกายตาหรี่สุขุมปราบทึ่ง *‘ระดับดาเมจกายภาพและการคุมเทมโปสนามของไคริวสะท้อนระดับคูณสองสเตตัสปกติชั่วคราว... เป็นบทเรียนการจัดรูปขบวนรุกรับระดับสูงของจริงที่ช่วยเพิ่มสมองวิเคราะห์ประสาท Tier 3 ของฉันให้พุ่งข้ามระดับขีดจำกัด!’*

“รีบก้าวเท้าเดินถอยนำแผนวิจัยหลบหนีเข้าวิหารชั้นในเถอะเรย์! คีธจะสกัดขวางด้านหน้าให้ปลอดภัยเอง!” โนอาตะโกนลั่นชี้นิ้วนำทางอุโมงค์ตะวันออก

เรย์พยักหน้าสั่งคู่หูเกโควาชิและสแนปโซการ์ดีทองคำขยับขึ้นกางปีกคุ้มกันขนาบซ้ายขวาก้าวเดินฝ่าฝุ่นปูนมุ่งตรงสู่วิหารป่าไผ่สลักเหล็กชั้นในสุดทันทีอย่างแน่วแน่!
การบุกจู่โจมยามดึกสงัดของกองทัพคัลต์เสียงประสานในอุโมงค์ถ้ำรากไผ่ คีธกางกั้นป้องรับดาเมจหลังส่งเรย์และโนอาถอยหลบ ไคริวมังกรยักษ์ปลดปล่อย Hurricane พัดเกลียววายุเหล็กและ Thunderbolt ผ่าทำลายเครื่อง Anomaly คัลต์ระเบิดปังใหญ่เสียงสะเทือนก้องกังวานป่า ลูคัสสั่งไฟแอโร่สู้รบเคียงคู่คุมโซนทิศทางเดิน
