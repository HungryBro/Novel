# ตอนที่ 93: "ความจริงเรื่องการลบความทรงจำ"

ภาพโฮโลแกรมสว่างสีครามของวิศวกรวิจัยของ System Architect ย่อยตัวดับวูบลงช้า ๆ ทิ้งไอลวดลายวงจรไฟฟ้าสีฟ้าจาง ๆ เต้นระริกกะพริบถี่ไปตามผนังศิลารากไผ่ทองเหลืองใต้ดินลึก เรย์และโนอาทรุดตัวลงตรวจเอกสารไฟล์สถิติในกระเป๋าแล็บวิจัยของอาจารย์ฌอนอย่างจดจ่อ โดยมีโบกุเระเคียงข้างคอยส่องไฟช่วยลดประกายความมืดมนสลัวเหนี่ยวนำ

โนอาขยับจัดเอกสารกระดาษเปื่อยกรอบแล้วเอามือทาบอกขยับแว่นสายตารายงาน “เรย์... บันทึกวิจัยแรกรหัสสแกนตัวนี้ขยายความลับเพิ่มอีกขีดนึงค่ะ! ที่จริงแล้ว... การลบล้างและกรองความทรงจำยี่สิบเปอร์เซ็นต์ของเธอไม่ใช่แค่เรื่องลดแรงเหนี่ยวนำจิตใจดั้งเดิมประสาท...”

“แต่เพื่อสกัดกั้นไม่ให้แชมเปี้ยนก้าวข้ามไปปรับเปลี่ยน ‘แกนคอร์ระบบหลัก’ (Core Kernel Editor) ของโลกใบนี้ต่างหาก!”

เรย์หรี่ดวงตาสีเทาสุขุมนิ่งสนิทจับจ้องวิเคราะห์ตัวอักษรรหัสระบบช่วยรบ Tier 3 ที่แจ้งยืนยันสมการประสาทสว่างวาบ:

[ยืนยันสอดรับความเข้าใจรหัส: แกนแก้ไขระบบหลัก (Core Kernel Editor Restriction Mode)]
[ระดับการกักรหัสประสาทสมอง: กีดกันการแทรกแซงรหัสประเภทผู้ควบคุมระบบ (Admin Privilege Blocked)]
[ขีดวิเคราะห์สปีดสลายตัว: ปลอดภัยรหัสคงค้าง 90%]

*‘ปรับแกนคอร์ระบบหลัก...’* เรย์หลับตานิ่งคิดประมวลผล VGC ในห้วงความคิดลึกล้ำอบอุ่น *‘แกนคอร์ของโลกนี้คือระบบกติกาพื้นฐานธรรมชาติ เช่น กฎแรงดึงดูด กฎธาตุไฟหญ้าน้ำ และระบบต่อสู้จำลอง VGC หากแชมเปี้ยนที่มีความทรงจำเดิมครบถ้วนสมบูรณ์เข้าสลับแก้ไขรหัสตรง ๆ ด้วยอารมณ์ความอยากเอาชนะที่โลภโมโทสัน ระบบโลกจะลัดวงจรเกิดสสารบิดเบี้ยว Anomaly มหาศาลทำลายธรรมชาติจริงให้พังพินาศปานไฟลามทุ่ง! ระบบจึงจำเป็นต้องสร้าง "ข้อจำกัดสเตตัสความทรงจำลบออก" เพื่อกักรหัสตัวแก้แกนคอร์ไว้ไม่ให้ถูกดัดแปลงตามใจชอบ’*

“นี่มันเหมือนระบบควบคุม firewall ของบริษัทซอฟต์แวร์ยักษ์ใหญ่เลยนะเรย์...” โนอาพูดวาจาน้ำเสียงเคร่งเครียดปนทึ่งในทฤษฎีประสาท “อาจารย์ฌอนวิจัยพบว่า... กลุ่มเสียงประสานพยายามกู่เรียกตัวแทนแชมเปี้ยนที่มีความทรงจำครบลถ้วนและสแกนเขียนรหัสสลับ เพื่อแหกคีย์กุญแจทองเหลืองเจาะเข้าไปล้างและดัดแปลงแกนคอร์ลบรหัสธรรมชาติทั้งหมดทิ้งไป แล้วเขียนโค้ดธรรมชาติบิดเบี้ยวตามความเชื่อว่างเปล่าอันบ้าคลั่งของพวกมันเพื่อลบล้างระบบกู้ชีพ!”

ลูคัสตบข้อมือซ้ายก้าวเท้ายืนตรงปานมีดตัดเด็ดเดี่ยว “ก็นั่นแหละเว้ยเพื่อนยาก! พวกมันจงใจลบสติสัมปชัญญะของโซตะและเวเรียเพื่อใช้เป็นตัวเชื่อมต่อเจาะระบบคอร์! วิธีการโสโครกแบบนั้นพวกเราจะปล่อยให้ทำสำเร็จไม่ได้เด็ดขาดชั่ววิ!”

เรย์พยักหน้าเด็ดเดี่ยวมั่นใจในจิตวิญญาณแชมเปี้ยน *‘ความรู้เดิมและมิตรภาพที่ประคองมา... เราจะปกป้องระบบคอร์ธรรมชาติจริงนี้ไว้ให้มั่นคงที่สุดโดยไม่ยอมจำนนแก่กฎบิดเบี้ยวมิติว่างเปล่า!’*

“เราจะมุ่งหน้าตรงเข้าสู่วิหารป่าไผ่สลักเหล็กชั้นในและสกัดพิธีกรรมปัดกวาดล้างรอยแยกของคิงดราทันทีในเช้าพรุ่งนี้ครับ!” เรย์กล่าวปักธงน้ำเสียงราบเรียบนิ่งสุขุมเปี่ยมประกายตาคมกริบหนักแน่น!
เอกสารวิจัยของอาจารย์ฌอนเฉลยปมการจำกัดสิทธิ์ Admin Privilege Blocked เพื่อป้องกันแชมเปี้ยนเข้าไปปรับแก้ไขแกนคอร์ระบบหลัก (Core Kernel Editor) ของโลกจำลองกติกาธรรมชาติจนพังทลาย ลัทธิคัลต์เสียงประสานพยายามกู้แชมเปี้ยนรหัสเต็มเพื่อแหก firewall ลบธรรมชาติจริงทิ้ง เรย์ ลูคัส และโนอา ร่วมประสานอุดมการณ์พิทักษ์รหัสโลกดั้งเดิม
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
