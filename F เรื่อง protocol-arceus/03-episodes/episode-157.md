# ตอนที่ 157: "พายุทรายสีม่วงทลายแท่นศิลา"

*ตูม... ตูม... เพล้ง...!!!*

กระแสพายุทรายสีม่วงทมิฬ *Void Sandstorm* ปนกรวดหินมลพิษของบันกิรัส (Anomaly Tyranitar) พุ่งซัดกระแทกม้วนตัวถล่มรอยร้าวมณฑลเหนือรุนแรงสะสะเทือนปฐพีรอยแยก บดหักสะบั้นคานเสาหินและพัดเสาวิหารเก่าสล้างละลายกลายเป็นแก้วเหลวร่วงลงน้ำตกน้ำแข็งเสียงดังเกรียวกราวสอยรหัสลัดวงจรไฟฟ้า

จูลส์และโโฮโดรอนมังกรเหล็กขวางกุมแนวป้องข้างหน้าต่างโดนพายุทรายอัดกระแทกจนเกราะเหล็กส่งเสียงแตกเปรี๊ยะใหญ่หนาเตอะ ดับแถบเลือดลดฮวบลงเจ็บปวดพีดจนล้มทรุดตัวลงคุกเข่าบนน้ำแข็ง ค่าน้ำหนักแถบเลือดสีทองหมดเกลี้ยงหมดสภาพการรบทรุดร่วงล้มคว่ำ

“โโฮโดรอน!” จูลส์ตะโกนลั่นใบหน้าเผือดสีซีดสะกดใจพยายามพยุงมังกรคู่ยากกลับเข้าบอลอย่างถนอมวิญญาณรบ

เรย์ยืนมองดูผลลัพธ์การสลายตัวของแนวรับข้างหลังด้วยดวงตาสีเทาสุขุมนิ่งสนิทขยับแว่นโมโนเคิลขวาประเมินสเตตัสรบประสาทช่วยรบ Tier 3:

[รายงานผลลัพธ์ระบบ: โโฮโดรอนหมดสภาพการรบ (Duraludon fainted)]
[ระดับการกัดเซาะสสารความว่างเปล่าสะสม: ทะยานขึ้น 75% (Alert Tier)]
[ขีดวิเคราะห์ความทรงจำแชมเปี้ยนชิบูย่า: 90% (คงที่เสถียร)]
[สัมผัสรหัสบันกิรัสแปรปรวน: ค่าป้องกันทางกายภาพพุ่งพีกสูงสุดขอบ]

*‘โโฮโดรอนช่วยพยุงรับดาเมจพัดทรายเหล็กจนสลบหมดก๊อก...’* เรย์จดจ้องมองจุดเปราะบางรอยพับขวาล่างใต้สะโพกของบันกิรัสที่สั่นกริบกู้ภัย *‘พายุทรายสีม่วงบล็อกกระสุนเวทย์พิเศษไฟและน้ำสะสม หากเราไม่มีวิธีตรึงมิติบิดสลับ หรือกระโดดสั่งรบรุกฆาตกายภาพประชิดสัมผัสใกล้เพื่อหักจุดแกนบดทลายเหล็ก จะโดนพายุทรายขยี้จนหมดสภาพรบแน่ในเทิร์นประลองสุดท้าย!’*

“เกโควาชิครับ! สแนปโซ! สบสายตากบประสานใจพร้อมทลายด่านตัดสินรังบอสใหญ่เดี๋ยวนี้!” เรย์กู่ร้องตะโกนสั่งพุ่งตัวนำศึกปักธงน้ำเสียงเรียบเรียบเด็ดเดี่ยวเปี่ยมสายตาคมกริบ

เรย์สั่งการสแนปโซและเกโควาชิก้าวเท้ายื่นมั่นคงฝ่าฝุ่นปูนทรายมลพิษเข้าปะทะดวลเดือดขั้นประวัติศาสตร์เปิดศึกในทันทีอย่างกล้าหาญ!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
