# ตอนที่ 124: "มุ่งหน้าสู่โรงเตาหลอมใหญ่เพียงลำพัง"

ไอความร้อนอบอวลปนกลิ่นเหล็กหลอมเหลวและเศษเขม่าถ่านหินดำหนาทึบแผ่รังสีร้อนสะเปรี้ย ๆ พัดผ่าข้ามแนวกะบังลมปากทางอุโมงค์รอยร้าวทิศเหนือ เรย์ก้าวเท้าเหยียบก้าวเหยียบย่ำลงบนแผ่นเหล็กกล้าร้าวขรุขระ ก้าวเดินเดี่ยวเดี่ยวสะพายกระเป๋าปืนพก Stabilizer ตรงดิ่งมุ่งหน้าเข้าสู่ห้องโถงยักษ์ของ ‘โรงถลุงเหล็กแกนกลาง’ (Central Foundry Room) ข้ามขีดปวดล้า

ทิ้งโนอาและโซต้าไว้เบื้องหลัง ณ อู่ซ่อมรถร้างในพิกัดปลอดภัยที่ตั้งจุดตรึงตราไว้มั่นคงเพื่อให้ปลอดภัยจากไอ Anomaly ลัดวงจร

ที่ใต้ฝ่าเท้า ลายเส้นตารางกริดสีทองกะพริบตาถี่ริ้วปะทุเป็นละอองไฟสีทองเหลวสลับม่วงดำแปรปรวน เสียงระเบิดตูมตามของกระแสเครื่องปฏิกรณ์แผดเสียงก้องกังวานน่ากลัวสะกดประสาทหูสั่นสะท้านกริบ

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทขยับคาดแว่น Veteran Analyzer Monocle ตรวจจับวิเคราะห์ระบบรบประสาท Tier 3 สว่างวาบ:

[ยืนยันการก้าวเข้าสู่พื้นที่รบสูงสุด: โรงถลุงเหล็กแกนกลางชั้นใน (Central Foundry Core)]
[ระดับการสลายตัวของมวลธรรมชาติรอบตัว: 62% (ระดับพอร์ทัลกางตัววิบัต)]
[เป้าหมายรุกฆาต: ชะล้างรอยแยกปิดผนึกแกนที่ห้าและกู้ตัวประกันลูคัส]
[ขีดวิเคราะห์ข้อมูลความจำชิบูย่า: 90% (รหัส Firewall เสถียรดี)]

*‘พวกมันเหนี่ยวนำให้กระแสเตาหลอมลัดวงจรรุนแรง...’* เรย์หลับตาลงรวบรวมจิตเจตจำนงรบสถิติ VGC *‘ศึกเดี่ยวครั้งนี้... ปราศจากวิศวกรวิจัยคอยสแกนวิถีทางพิกัด และไม่มีไคริวคุมอากาศพายุทราย ทว่าข้อมูลกริดรอยต่อเตาหลอมทั้งหมดได้รับการคำนวณสำรองในสายตาขวาของฉันแล้ว ฉันจะสั่งเกโควาชิและสแนปโซการ์ดีทองคำเจาะทะลวงเข้าสอยแกนกลางด้วยความมุ่งมั่นมั่นคงร้อยเปอร์เซ็นต์!’*

ที่ปลายลานเหล็กหลอมเดือดพวยพุ่งพลาสม่า เงาร่างสูงใหญ่ของแม่ทัพลัทธิเสื้อคลุมเทาปักลายดวงตาร้องไห้ยิ้มกว้างกางขายืนต้อนรับเคียงข้างเตาหลอมเหลวเดือดปะทุ ดักขวางขบวนประลองรบประวัติศาสตร์ขวางหน้าท้าทาย

*“โฮก...!!!”*

สแนปโซการ์ดีทองคำแยกเขี้ยวส่งเสียงคำรามขู่กวาด เกโควาชิหมอบลงกบพรางทิศตื่นรุกว่องไวเตรียมพร้อมเปิดสัญญารบรุก

เรย์กำตรากุญแจทองเหลืองพยุงกายหนักแน่นเด็ดเดี่ยวเผชิญพายุกระแสเหล็กหลอมเดือด *‘ลูคัส... เพื่อนยากเอ๋ย... ฉันยืนอยู่หน้าด่านทักษะประลองสุดท้ายแล้ว ฉันจะทลายคุกทมิฬปิดผนึกรอยแยกกู้ชีวิตแกกลับคืนมาให้เสถียรดั่งเดิม!’*

เรย์ก้าวเท้าพุ่งร่างเปิดศึกสงครามเงากวาดล้างรอยร้าวมณฑลเหนือทวีปขั้นสูงสุดในทันทีอย่างองอาจมั่นใจรบรุกฆาต!
เรย์เดินเดี่ยวก้าวพ้นประตูดักพ้นโนอาพำนักปลอดภัย เข้าสู่ห้องโถงยักษ์โรงถลุงเหล็กแกนกลางพลาสม่าบ่อเหล็กเหลวเดือดพวยพุ่งระอุเดือด 62% สสาร Void มุ่งปิดปากรอยแยกปิดประตูห้ากอบกู้ลูคัสเพื่อนยากร่วมรบ แว่นโมโนเคิลวิเคราะห์พร้อมลั่นกระสุนพลังชะล้างปราบภัย
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
