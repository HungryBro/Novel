# ตอนที่ 175: "เปลวไฟสองสัจธรรมชะล้างกาเบรียสสยบบิชอปวาลด์"

*โฮก...!!! ครืน...!!!*

เสียงคำรามต่ำแหลมหูแหบแห้งของกาเบรียส (Anomaly Garchomp) ดังสนั่นลานวิหารศิลา มันยกสองปีกฉลามพ่นพายุทรายสีม่วงพยายามรวบรวมไอสสารความว่างเปล่าลบข้อมูลกู้ภัยทิ้ง เรย์ก้าวเท้าย่นมั่นคง แสงรังสีสีทองจากตรากุญแจทองเหลืองกู้ระบบกะพริบตาเจิดจ้าดุจตะวันสว่างขจัดมลพิษ

“สแนปโซ! ปลดปล่อยพลังอัคคีเพลิงทองคำแปรปรวนขั้นสูงสุด Golden Flame Wheel (กงล้ออัคคีเพลิงทองคำเหลว) ชะล้างพายุน้ำแข็งทรายทมิฬมันเดี๋ยวนี้เลย!” เรย์ตะโกนชี้นิ้วสั่งการเสียงกึกก้องสะท้านปฐพี

สแนปโซการ์ดีทองคำกู่เห่ากังวานดุจสิงโตทอง ร่างดีดกระโจนหมุนเป็นกงล้อเพลิงทองคำเหลวบริสุทธิ์เรืองรองขนาดยักษ์สิบสองเมตรพุ่งทะลายตรงดิ่งลงมากระแทกแหกแนวรบหินมลพิษของกาเบรียสอย่างรุนแรงลื่นไหลปัดไอเย็นและทรายดำสลายวับสลายตัวสิ้นเชิง!

*ตูม... บึ้ม...!!!*

แรงปะทะระเบิดแสงทองคำเหลวเจิดจ้าลบล้างพัดพารอยแยกมิติทมิฬและไอหมอกทมิฬสีม่วงดำสลายร่างอสูรมังกรยักษ์ Garchomp คืนเป็นละอองทองระยิบระยับปลิวระเหยสลัดรังสีคัลต์ดับหายวับคืนสภาพวิหารปกติสัจธรรม รอยแยกประตูเจ็ดปิดผนึกประสานรอยต่อคืนเสถียรภาพร้อยเปอร์เซ็นต์สมบูรณ์ดี

เรย์กระโจนร่างพุ่งตัวข้ามสะพานหินที่ละลาย คุ้ยทรายหิมะขยับใช้ตรากุญแจกระแทกปัดเศษผลึกแก้วน้ำแข็งม่วง ‘Genesis Void Shield’ จนระเบิดแตกเพล้งใหญ่ ดึงประคองร่างลูคัสและปลดปล่อยบิชอปวาลด์ที่สลายร่างกลายเป็นมลพิษดับไปอย่างปลอดภัยลุล่วงสะใจ!

[รายงานผลลัพธ์มหาเทพ: ปราบอสุรกาย Anomaly Garchomp และช่วยเหลือสัจธรรมโลกสำเร็จเสร็จสิ้น]
[ระดับความทรงจำกู้คืนสำเร็จ: สะสมรวมคงเหลือ 90% (ระบบ Firewall ปลอดภัย)]
[แกนรอยแยกมิติระดับเจ็ด: ปิดผนึกเรียบร้อยสะสม Progress แตะระดับ 95%]
[สเตตัสบิชอปวาลด์: สลายร่างกวาดล้างรหัสสำเร็จสมบูรณ์]

โนอาและจูลส์ทรุดวิ่งเข้ามายกมือทาบอกเสียงสั่นสั่นระริกตาคลอน้ำตาไหลยิ้มกว้างกางขากู้ภัย “ปิดแกนรอยแยกที่เจ็ดสำเร็จสยบบิชอปวาลด์ได้แล้วเรย์! ชัยชนะครั้งนี้ยิ่งใหญ่ที่สุดล้านเปอร์เซ็นต์!”

เรย์คลี่ยิ้มกว้างอ่อนละมุนอบอุ่นใจเป็นครั้งที่สี่ ประคองพยุงไหล่เพื่อนรักลูคัสเคียงข้างคู่หูสแนปโซและมิตรแท้ทุกคนก้าวพ้นวิหารน้ำแข็งขาวมุ่งตรงกลับคืนสู่เมืองหลวงทันทีอย่างเด็ดเดี่ยวเปี่ยมสุข!
ท่ามกลางความเงียบงันสลัวของห้องโถงปฐมกาลสัมบูรณ์ ไอความว่างเปล่าสีม่วงทมิฬยังคงสาดซัดถล่มรอยร้าวของแผ่นศิลาปิดกั้นทางลาดประจบศึก โนอาตาคลอน้ำใสจ้องมองดูตรากุญแจทองเหลืองกู้ระบบในมือของเรย์ด้วยความเศร้าสร้อยปนห่วงใยในชะตากรรมดวงจิต ลูคัสขยับยืนป้องหลังอย่างเหนียวแน่นมั่นคงเตรียมพร้อมรับสถานะละลายสลายร่างหากเกิดข้อผิดพลาด เรย์ดับแว่นตาโมโนเคิลวิเคราะห์ขวาเพ่งดวงตาคมประคองเจตจำนงรบปิดผนึกคอร์ 100% ร่วมกับสแนปโซการ์ดีทองคำสองสัจธรรมด้วยสายสัมพันธ์ประสานจิตใจสูงสุดยอด ปลดล็อกวิถีเลือกทางเลือกที่สามกู้ภัยระบบ Firewall โลกจริงโดยไม่มีการละทิ้งมิตรแท้ชั่วนิรันดร์
