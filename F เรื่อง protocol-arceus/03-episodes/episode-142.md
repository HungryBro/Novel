# ตอนที่ 142: "การประสานพลังชะล้างร่วมกับจูลส์"

*ตูม... บึ้ม...!!!*

กระแสน้ำพายุดำปนเกล็ดหิมะสีม่วงทมิฬสาดกระแทกเข้าใส่แนวเสาเหล็กของวิหารลานประลองเสียงดังสนั่นสะเทือนรอยแยก ยูคิโนโอ (Anomaly Abomasnow) ขยับร่างยักษ์ยกกิ่งไม้น้ำแข็งสปริงขว้างกระสุนน้ำแข็งว่างเปล่า *Icicle Crash Void* (หอกน้ำแข็งความว่างเปล่า) ขนาดสี่เมตรนับสิบหอกยิงปะทะกระหน่ำดิ่งลงมาสอยเรย์และจูลส์อย่างรุนแรงปานป้อมปืน

จูลส์ก้าวเท้ายืนตรงพยุงมังกรคู่ยาก “โฮโดรอน! เปิดใช้งาน Iron Defense ตั้งแผงเกราะเหล็กบังรั้วด้านขวาไว้ด่วนเว้ย! อย่าให้สะเก็ดหอกน้ำแข็งเข้ากัดเซาะตัวสแกนกู้ระบบแชมป์ซีโร่!”

*เคร้ง... เคร้ง...!!!*

หอกน้ำแข็ง Void กระแทกแผงโลหะของโฮโดรอนจนสะท้อนประกายไฟเหลืองว่อน รั้งสะกดกั้นทางลาดหินถล่มอย่างสุดกำลังรบ

เรย์ยืนมองดูแนวป้องกันของจูลส์ด้วยดวงตาสีเทาสุขุมนิ่งสนิทภายใต้โมโนเคิลขวาประเมิน VGC ในประสาท Tier 3:

[สัมผัสพิกัดหอกน้ำแข็งบอส: โครงสร้างประจุกระแสสสาร Void อยู่ตรงแกนหัวใจสีม่วงอกของมัน]
[ทางเลือกรุกฆาต: ประสานเปลวไฟ Golden-Void Fire ของสแนปโซพุ่งชนละลายจุดแกนน้ำแข็งในแนวตรง]
[สมาธิความก้าวหน้าประสาทช่วยรบเรย์ Tier 3: เสถียร 100% ( Willpower ปลดล็อก)]

*‘โฮโดรอนช่วยดักรับความเสียหายและสกัดสปีด Icicle Crash...’* เรย์จดจ้องมองจังหวะขยับตัวประสานสมการในหัวดั่งมีดโกน *‘เทิร์นรบประสาน Double Candidate... นี่คือแผนรุกฆาตที่ลงตัวที่สุด! หากจูลส์กางม่านเหล็กต้านล้างฟิลด์สนาม แล้วฉันสั่งสแนปโซกระโจนสปริงเปิด Golden Flame Wheel หมุนตัวเหินข้ามกรงเกราะเหล็กทะลวงตรงเข้าหัวใจอกมัน ไอความว่างเปล่าหิมะจะระเหยสลายตัวทันที!’*

“สแนปโซ! ชิงเปิดจังหวะรุกด้วยสปีดลมหาง Tailwind ปีกจูลส์หนุนนำตัวลอยข้ามแผงเกราะเหล็ก! ปลดปล่อยกงล้อไฟอัคคีทองคำเหลว Golden Flame Wheel พุ่งชนสับตรงแกนหัวใจอกยูคิโนโอเดี๋ยวนี้เลย!” เรย์ตะโกนชี้นิ้วสั่งการเสียงเฉียบกังวานหนักแน่น

สแนปโซการ์ดีทองคำแยกเขี้ยวเห่ากังวานดุจสิงโตทอง ทะยานร่างหมุนวนเป็นลูกบอลเพลิงสีทองคำเหลวเจิดจ้ากระโจนแหวกพายุหิมะพุ่งชนเปรี้ยงชนขวับเข้ายอดอกยูคิโนโอ Anomaly อย่างรุนแรงสะเด็ดขาด!

*ตูม... บึ้ม...!!!*

เสียงระเบิดไอความร้อนสีทองชะล้างน้ำแข็งมืดสลายปลิวว่อน พายุหิมะสีม่วงทมิฬสลายตัวดับวูบกลายเป็นไอละอองทองคำระยิบระยับลอยฟุ้งกระจายชะล้างมณฑลลานให้สว่างจ้าดังเดิม อสุรกายยูคิโนโอกรีดร้องโหยหวนเสียจังหวะล้มคว่ำราบสลบหมดสภาพการรบ คลื่นม่านกัก Void Dome แตกเพล้งใหญ่พังพินาศสลายตัวสำเร็จเสถียร!

“ทลายพายุหิมะและม่านกักมิติได้หมดจดมากเรย์! ความร่วมมือของอดีตแชมเปี้ยนสุดยอดไปเลยเว้ย!” โนอาตะโกนลั่นยิ้มกว้างปาดเหงื่อกางชัยชนะ

เรย์ก้าวเท้าสบสายตากับจูลส์หนักแน่นมิตรภาพ *‘ม่านกักสลายตัว... หนทางเข้าสืบประวัติตัวตนใต้ดินวิหารอารีน่าเปิดออกแล้ว!’*
รอบบริเวณห้องโถงวิจัยและลานประลองแกรนด์ซัมมิทอารีน่ายังคงอบอวลไปด้วยไอหมอกเย็นชื้นปนกลิ่นกระแสไฟฟ้าจากท่อไอน้ำใต้ดินที่พ่นระเหยควันขาวฟุ้งกระจาย แสงไฟนีออนสีส้มเหลืองและเขียวสะท้อนผ่านผลึกเกล็ดหิมะบางเบาที่ร่วงหล่นลงมาทับถมแนวกำแพงศิลาหนาเตอะดึงทัศนียภาพหนาวเหน็บ โนอากระชับผ้าพันคอหนาสีกรมท่าคอยคีย์ปรับความเสถียรข้อมูลระบบวิจัยอย่างใส่ใจดูแลในขณะที่สแนปโซและเกโควาชิเฝ้าระแวดระวังแนวขอบเขตสนาม เรย์ใช้สายตาสีเทาสุขุมนิ่งสนิทจับจ้องวิเคราะห์ริ้วสัญญาณรหัสสีทองของตารางกริดจำลองกู้ภัยผ่านโมโนเคิลขวา พยายามประสานพลังเจตจำนงรบ Willpower ขั้นเต็มร้อยเปอร์เซ็นต์เพื่อชะล้างลบรอยร้าวมลพิษ Anomaly ข้ามขีดปวดล้าสะสม การเดินทางร่วมทาง VGC ในชีวิตจริงที่เกิดจากความผูกพันและสายสัมพันธ์ร่วมรบพยุงระดับความมั่นคงสมาธิประสาทช่วยรบ Tier 3 ให้คงที่เสถียรรหัสเต็มพิกัดโดยไม่มีการยอมจำนนต่อกฎบิดเบี้ยวมิติ
