# ตอนที่ 152: "มานิวลาแปรวนและกรงเล็บคมความว่างเปล่า"

*ตูม... บึ้ม...!!!*

สะเก็ดน้ำแข็งสีม่วงและไอรังสี Void คมกริบระเบิดปะทุขัดพิกัดขวางหน้าลานหินทรายรอยแยก เผยร่างอสูรนักล่าด่านพายุ ‘มานิวลา’ (Weavile Anomaly) โปเกมอนประเภทน้ำแข็งและมืด ทว่าทั่วร่างกายเพรียวบางของมันกลับหุ้มด้วย **แร่เกราะแก้วผลึกดำสลักดวงตาร้องไห้** และปลายเล็บกรงเล็บทั้งสองข้างยาวชะลูดแผ่ประกายกระแสสสารไอสีม่วงเข้มเย็นยะเยือก — ‘มานิวลา Anomaly (ประเภทน้ำแข็ง/มืด/Void)’!

*“โฮก...!!!”*

มานิวลายักษ์กู่ร้องแผดเสียงต่ำแยกเขี้ยวพุ่งกระโจนตัวด้วยสปีดมหาศาล ยิงกรงเล็บคมความว่างเปล่า *Void Slash* (กรงเล็บสลายเกราะ) ปัดปาดตัดเฉือนสะพานเหล็กจนขาดกระเด็นร่วงราบเสียงดังระงมสอยรหัส

โนอาปาดหยดน้ำตาซีดเผือด “กรงเล็บ Void Slash ของมานิวลาตัวนี้ขยับปัดสอยสปีดลื่นน้ำแข็งขึ้นสูงพีกร้อยละเก้าสิบแล้วค่ะเรย์! ตัวเลข VGC และระบบช่วยรบ Tier 3 ประเมินผล Error กะพริบไม่หยุดนิ่งเลยค่ะ! ถ้าโดนกรงเล็บฟันตรงจุดป้องกัน ข้อมูลชีวิตของพวกเราจะสลายตัวกลายเป็นเศษแก้ว!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทขยับแว่นโมโนเคิลขวาวิเคราะห์ขบวนรบผ่านคลื่นเสียงช่วยรบ Tier 3:

[ตรวจพบสิ่งผิดปกติระดับห้า: มานิวลาแปรวน (Anomaly Weavile) | เลเวล 73]
[คุณสมบัติพิเศษการโจมตี: กรงเล็บตัดผ่านดาเมจป้องกัน (Void Slash)]
[กฎธรรมชาติบิดเบี้ยว: ดาเมจพิเศษประเภทธาตุไฟปกติจะโดนเล็บน้ำแข็งสกัดสะท้อนกลับ 100% ชั่วคราว!]
[สัมประสิทธิ์รหัสคู่หูเสถียร: 90% (พร้อมชะล้าง)]

*‘เล็บน้ำแข็งสะท้อนไฟปกติ 100%...’* เรย์จดจ้องมองจังหวะเงากวัดแกว่งคำนวณในความคิดสมมาตร VGC *‘ภายใต้สภาพรอยแยกหนาวเหน็บ ดาเมจไฟส้มจะโดนหักพลังจู่โจมแทบสิ้นซาก หากเราสั่งรุกช้าหรือปล่อยให้อีวุยปะทะตรง ๆ จะโดนเล็บ Void Slash สับร่างจนหมดสภาพการรบพังยับเยินในเทิร์นเริ่มต้นแรก! เราต้องสั่งให้เกโควาชิใช้สปีดนินจา Aerial Ace สอยสกัดขัดจังหวะเท้าเพื่อหักล้างดาเมจศัตรู!’*

“เกโควาชิ! ชิงเปิดจังหวะรุกด้วยความเร็วกายภาพ Aerial Ace (เหินเวหาจู่โจมฉับพลัน) ปาดดาบน้ำปัดสอยข้อต่อกรงเล็บมานิวลาตัวซ้ายดักพิกัด! สแนปโซ! พ่นเปลวเพลิงสีทองคำเหลวคุมทิศทางป้องกันขวา!” เรย์สั่งเสียงเฉียบพลันเด็ดเดี่ยวหนักแน่นเปี่ยมพลัง

เกโควาชิขยับเงานินจาทะยานตัวข้ามพายุน้ำแข็ง สับดาบวารีกายภาพฟันปะทะหักกรงเล็บ Void Slash ของมานิวลาจนแตกละเอียดระเบิดไอความเย็นกระจาย อสุรกายมานิวลาร้องโหยหวนหลบถอยเสียหลักล้มคว่ำราบสิ้นขีดป้องกันศึก!

เรย์พยักหน้าให้คู่หูร่วมศึกประสานขบวนรบรุดรังบอสใหญ่ Tyranitar ทันทีอย่างมั่นใจพิทักษ์มณฑลเหนือ!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
