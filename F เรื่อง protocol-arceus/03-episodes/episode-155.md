# ตอนที่ 155: "วิหารแกนกลางและกรงขังน้ำแข็งลูคัส"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดกระแสมิติลำดับสี่จากใจกลางลานศิลายักษ์ใต้ดินวิหารก้องกังวานเลื่อนลั่นสะเทือนสลับเสียงน้ำแข็งถล่ม เรย์ โนอา และจูลส์ ก้าวเท้าหยุดเหยียบยืนอยู่บนขอบระเบียงศิลายักษ์สูงสุด เผยให้เห็นทัศนียภาพศูนย์กลางแท่นทำพิธีรอยแยกกว้างกว่าร้อยเมตร — ‘วิหารแกนกลางน้ำแข็งขาว’ (Central Ice Altar Room) ที่แผ่ไอรังสี Anomaly สีม่วงดำมหาศาลพวยพุ่งขึ้นพีกร้อยเปอร์เซ็นต์สะกดใจ

ที่ใจกลางแท่นบูชาหินทราย **เงามวลผลึกแก้วน้ำแข็งสีม่วงเข้มขนาดยักษ์พิกัด ‘คุกน้ำแข็งความว่างเปล่า’ (Void Ice Cage) ตั้งตระหง่านกักขังลูคัสที่นอนหมดสติหลับสนิทไว้เบื้องในเกล็ดแก้ว** โดยมีกรงเหล็กแฝงประจุ Void ขังไฟแอโร่นกเหยี่ยวนอนพับปีกสลัดความล้าอยู่ขนาบข้างดักพิกัดกู้ภัย

เหนือแท่นบูชา บิชอปวาลด์ (Bishop Vald) แม่ทัพใหญ่คัลต์ในหน้ากากเหล็กสีทองแดงยืนกอดอกกู่หัวเราะคำรามกร้าวอย่างบ้าคลั่งท้าทายสงคราม

“ฮ่า ๆ ๆ ! เจ้าแชมเปี้ยนซีโร่และคู่ Candidates อดีต...” บิชอปวาลด์แผดเสียงหัวเราะก้องสะท้อน “ลูคัสได้รับการเชื่อมต่อประสาทเป็นสะพานชาร์จแกนรอยแยกที่หกเสร็จสิ้นแล้ว! หากนายกล้าก้าวเท้าแตะแท่นศิลา... อสูรบันกิรัสความว่างเปล่าจะย่อยสลายร่างของนายและกวาดลบข้อมูลธรรมชาติจริงนี้ทิ้งไปเสียสิ้น!”

เรย์ยืนจ้องมองดูคุกน้ำแข็งของลูคัสด้วยสายตาสีเทาสุขุมนิ่งสนิท กำหมัดซ้ายที่สลักกุญแจทองเหลืองกู้ระบบกางระบบช่วยรบ Tier 3 ปรับความพร้อม Willpower 100%:

[ตรวจพบรหัสเป้าหมายหลัก: คุกน้ำแข็งความว่างเปล่ากักขังลูคัส (Void Ice Cage Identified)]
[ระดับการกัดเซาะสสารความว่างเปล่าสะสม: คงเหลือ 65% (Progress 65% sealed)]
[ระดับความทรงจำชิบูย่าคงค้าง: 90% (เสถียรรหัส Firewall ปลอดภัย)]
[ขีดวิเคราะห์สปีดรบ: พร้อมทำงาน]

*‘ลูคัสโดนกักรหัสชีวิตเป็นตัวเชื่อม...’* เรย์หลับตานิ่งสะกดสมาธิรบประเมิน VGC ในประสาทสมองรวดเร็วดั่งสายฟ้า *‘การสู้รบรังสุดท้ายในสภาพแวดล้อมน้ำแข็งบิดเบี้ยว จะได้รับการปะทะแลกหมัดด้วยดาเมจกายภาพความเร็วสูงปานเงา หากเราประสานงานไฟทองคำสแนปโซและดาบนินจาวารีของเกโควาชิพังทลายคุกน้ำแข็ง สสาร Void จะสลายร่างลบล้างไปสิ้นเชิง!’*

“ผมจะปิดผนึกแกนรอยร้าวและช่วยเหลือลูคัสออกมาอย่างสง่างามที่สุดครับบิชอปวาลด์!” เรย์กู่ร้องตะโกนสั่งพุ่งตัวนำศึกเดี่ยวปักธงรบกอบกู้อาร์คหกทันทีอย่างมั่นใจ!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
