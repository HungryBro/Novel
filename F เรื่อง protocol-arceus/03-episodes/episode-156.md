# ตอนที่ 156: "ประจันหน้าบันกิรัส Anomaly อสูรหินยักษ์"

*ครืน... ครืน... เพล้ง...!!!*

เสียงแยกถล่มร้าวละเอียดของเพดานศิลาและผลึกน้ำแข็งใต้วิหารลึกดังลั่นเลื่อนลั่นสะเทือนปฐพีรอยแยก บิชอปวาลด์แม่ทัพใหญ่คัลต์ยกข้อมือขวาสลัดปล่อยกูบโปเกบอลสีดำขลิบทองแดงปะทุรังสีความแปรปรวนสีม่วงเข้มขนาดใหญ่มหึมาระเบิดลงลานหินทรายกวาดคลุมแนวสายตาจนมืดสลัว

ท่ามกลางมวลพายุทรายสีน้ำหมึกดำหมุนวนเป็นเกลียวสูง เงาร่างอสุรกายโลหะและหินหนาตระหง่านพุ่งเหินเลื้อยทะยานตัวขึ้นมาจากปากเหวน้ำแข็งทมิฬส่งเสียงคำรามกังวานดุจฟ้าผ่าสะท้านปฐพี

มันมีรูปพรรณคล้ายคลึงอสูรหินยักษ์ประเภทหินและมืด ‘บันกิรัส’ (Tyranitar) ทว่าขนาดตัวสูงใหญ่ตระหง่านกว้างกว่าเก้าเมตร แผ่นหลังเหล็กหนาปะติดด้วย **เกล็ดแก้วผลึกดำสะท้อนประกายมิติมืด** และรอบดวงศีรษะมังกรยาวมีรูดวงตาสว่างแสบส้มสีส้มแดงฉานถึง **แปดดวง** กะพริบจ้องสแกนตรวจรูปขบวน — ‘บันกิรัส Anomaly (ประเภทหิน/มืด/Void)’!

*“โฮก...!!!”*

เสียงคำรามต่ำแหลมหูของมันกู่เรียกกระแสมหาพายุทรายทมิฬ *Void Sandstorm* (พายุทรายความว่างเปล่า) พัดซัดถล่มรุนแรงล้อมรอบแท่นดินปูน กวาดพัดเสาหินพังทลายสลายตัวเป็นแก้วใสกลวงเปล่าและสลายตัวจากความว่างเปล่าชั่วพริบตาเดียวปัดม่านเสียงรบกวนคลื่น

หน้าจอทองคำจำลองของระบบช่วยรบในสมองของเรย์เตือนภัยรหัสวิบัติตัวแดงปะทุ:

[ตรวจพบรหัสอสุรกายภูมิภาคเหนือ: บันกิรัสบิดเบี้ยว (Anomaly Tyranitar) | เลเวล 75]
[ค่าสถานภาพพิเศษ: การควบคุมสนามพายุทรายมิติมืด (Void Sandstorm field)]
[กฎบิดเบี้ยว: ดาเมจพิเศษประเภทธาตุไฟและน้ำจะโดนพายุทรายหักทอนพลังรบรุนแรงลง 75% ชดเชยน้ำหนัก!]
[สิทธิ์ถอดรหัสความเสถียร: พร้อมทำงานควบคู่กุญแจทองเหลือง]

*‘พายุทรายมิติมืดลดทอนดาเมจไฟและน้ำ 75%...’* เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทเผชิญพายุน้ำแข็งซัดสอย *‘นี่คือเงื่อนไขกดดัน VGC สลับขั้วที่โกงมาก! ภายใต้พายุทรายมิติมืดนี้ เพลิงทองคำของสแนปโซและดาบน้ำเกโควาชิจะสูญเสียพลังจู่โจมแทบสิ้นซากจนสร้างดาเมจได้ล้นพ้นลำบาก หากเราไม่มีวิธีสลายสภาพอากาศหรือดึงเจตจำนงรบสบจังหวะปิดจอระบบประจันรบกายภาพ จะโดนทรายบดกระแทกหมดสภาพการรบพังยับเยินแน่!’*

“โโฮโดรอน! ใช้ Iron Defense กางโล่เหล็กสะกดรับดาเมจป้องกันแนวซ้ายเดี๋ยวนี้! เกโควาชิ! โโบกุเระ! ตั้งแถวคุมโซน!” เรย์สั่งการเสียงเฉียบขาดเฉียบแหลมมั่นคง

จูลส์สั่งมังกรยักษ์กู่ก้องสลัดปีกกางเกราะเหล็กเข้าปะทะพายุทรายมลพิษเปิดศึก decisions แรกทันทีอย่างองอาจมั่นใจ!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
