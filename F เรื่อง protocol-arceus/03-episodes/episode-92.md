# ตอนที่ 92: "บันทึกของสถาปนิกและระบบป้องกัน"

ประกายสะเก็ดไฟร้อนอุ่นจากเตาไฟต้มเครื่องมือวิจัยสลัวช่วยละลายมวลหมอกน้ำค้างที่ไหลซึมจากโคนรากไผ่เหล็กใต้ดินลึก เรย์ โนอา และคีธขยับนั่งเรียงล้อมอยู่หน้าตู้เหล็กนิรภัยโบราณผุกร่อนสลักลายวงแหวนอาร์เซอุส คีธประคองนิ้วมือขยับคีย์สลับเปิดแท่นข้อมูลรหัสประจุประวัติศาสตร์ช้า ๆ สัญญาณไฟกะพริบรหัสตอบรับดังติ๊ด ๆ เป็นระเบียบเรียบร้อยนำเสนอข้อมูล

หน้าจอฉายภาพโฮโลแกรมแสงสีครามสว่างขึ้น เผยร่างชายฉกรรจ์สวมชุดวิศวกรวิจัยระบบภูมิภาคตะวันออก — เขาคือ **สถาปนิกของระบบ (The System Architect)** หรือหัวหน้านักเขียนโปรแกรมควบคุมมิติมหาเทพในอดีตผู้ก่อตั้ง!

“ฌอนแอบสำรองข้อมูลรหัสส่วนประวัตินี้ไว้ใต้ดินวิหารก่อนแกจะสลายร่างวิญญาณ...” คีธกล่าวพลาดยืดดวงตาจ้องภาพโฮโลแกรมสลัวเหนี่ยวนำ “ฟังข้อความเสียงนี้ให้ดีเถอะเรย์... นี่คือจุดกำเนิดระบบจำลองที่อยู่ในประสาทสมองของเธอที่ล็อกรอยแผลเป็นความทรงจำชิบูย่าของเธอไว้เพื่อความปลอดภัยขั้นสูงสุด”

เสียงแผ่วเบาปนซ่าปะทุในช่องโฮโลแกรมฉายดังขึ้นช้า ๆ น้ำเสียงสุภาพแต่แฝงประกายเด็ดเดี่ยวหนักแน่นดังสะท้อนกังวาน:

*‘บันทึกรหัสล็อกความปลอดภัยของสถาปนิกสลักไว้ (สิบห้าปีก่อน) — โครงข่ายโลก Pokémon ใบนี้ถูกพัฒนาขึ้นกู้สมดุลด้วยระบบจำลอง Firewall แชมเปี้ยนระบบจำลอง VGC เพื่อประคองขอบเขตมิติต้านแรงว่างเปล่า... แต่ตัวแปรที่น่ากังวลที่สุดคือการเข้าแทรกแซงรหัสแกนหลักของแชมเปี้ยนที่ใช้ทางลัดสถิติแห้งคำนวณเอาชนะอย่างเอาแต่ใจโดยปราศจากอารมณ์ร่วม มหาเทพจึงล็อกความทรงจำยี่สิบเปอร์เซ็นต์ส่วนคุณค่าความทรงจำครอบครัวและชีวิตธรรมดาไว้เพื่อป้องกันไม่ให้รหัสสมองดึงพลังกอบกู้โลกมาใช้ในทางลัดที่อันตรายล้างข้อมูลระบบสัญจร’*

*‘หากแชมเปี้ยนสามารถปลดสายสัมพันธ์ประสานจิตใจคู่หูแตะระดับสัมบูรณ์ร้อยเปอร์เซ็นต์ได้... ทางเลือกที่สามจะถูกเปิดใช้งานชำระล้าง Void โดยไม่ต้องสอยจิตวิญญาณแชมเปี้ยนมาเป็นแกนกึ่งสลายตัวประจุชาร์จเครื่องกางสนามพลังรักษาความเสถียรของโลกธรรมชาติจริง!’*

โนอายกมือขวาปาดหยดน้ำตาคลอเบา ๆ สบตากับภาพโฮโลแกรมอย่างซาบซึ้งใจสะเทือนใจ “ที่แท้... ระบบจำลองไม่ได้มีเจตนาทำร้ายหรือลักความทรงจำของเรย์... แต่เป็นระบบป้องกันภัยลัดวงจรมิติที่อาจารย์ฌอนและสถาปนิกตั้งขึ้นเพื่อช่วยแชมเปี้ยนให้พ้นทางตันความโดดเดี่ยวเปลี่ยวประสาทและรักษารหัสชีวิตให้ปลอดภัย!”

เรย์นั่งฟังนิ่งเงียบ ดวงตาสีเทาสุขุมประกายความเข้าใจสถิติพารามิเตอร์ขึ้นเด่นชัด หน้าจอระบบประสาทช่วยรบแจ้งยกระดับรหัสเสถียร:

[ยืนยันสอดรับความก้าวหน้าประวัติ: ปลดล็อกความทรงจำแชมเปี้ยน 90% (เสถียรรหัสเต็มพิกัด)]
[ความคืบหน้าระบบ Firewall Tier 3: เพิ่มระดับการกู้คืนสมมุติฐาน 3%]

*‘คุณพ่อรู้วิธีปลดล็อกทางเลือกที่สามนี้... แกจึงแฝงรหัสอัคคีเทพทองคำไว้ในตัวสแนปโซการ์ดีทองคำรอฉันย้ายมิติมาสืบสาน!’* เรย์หลับตาลงนิ่งสงบ ความอบอุ่นและรักใคร่ประสานในสัจธรรมสแนปโซการ์ดีทองคำทวีความมั่นคงขึ้นเป็นร้อยเท่า

“ผมจะปลดล็อกทางเลือกที่สามเพื่อช่วยทุกชีวิตและกู้คืนความทรงจำที่เหลือครับคุณคีธ” เรย์กล่าววาจาน้ำเสียงเด็ดเดี่ยวเปี่ยมประกายอุ่นใจหนักแน่น

คีธยิ้มกว้างกางไหล่พยักหน้ารับทึ่งในตัวเจ้าหนุ่มแชมเปี้ยนซีโร่ พร้อมพาลุยประดมศึกสุดท้ายต่ออย่างเด็ดขาด!
ภาพโฮโลแกรมฉายของ System Architect อธิบายเจตจำนงการสร้างระบบนิรภัย Memory Lock ป้องกันไม่ให้สมองแชมเปี้ยนลัดวงจรลบล้างสัญญารหัสดั้งเดิม โนอาตื้นตันใจน้ำตาคลอเบา ๆ เมื่อรู้ว่าระบบไม่ได้ทำร้ายเรย์แต่เพื่อปกป้องกู้ชีวิต เรย์พร้อมปลดล็อกทางเลือกที่สามชะล้างรอยแยกมณฑลเหนือด้วยพลังสแนปโซการ์ดีสองสัจธรรมร่วมรุกฆาต
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
