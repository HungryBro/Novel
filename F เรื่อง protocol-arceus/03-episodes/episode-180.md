# ตอนที่ 180: "สัญญาสัมบูรณ์และการยอมรับผู้คุ้มกันนิรันดร์"

*ครืน... ครืน... เพล้ง...!!!*

เสียงปิดประตูด่านรอยแยกสุดท้ายที่เจดีย์ปฐมกาลทองเหลืองดังสนั่นสะเทือนปฐพีรอยแยกก้องกังวาน ปลดปล่อยละอองทองคำระยิบระยับลอยชะล้างละลายฝุ่นเกล็ดหิมะสีม่วงทมิฬสลายตัวดับวูบกลายเป็นไอระเหยสว่างจ้าดุจวันใหม่รุ่งเช้า ท้องฟ้าสีครามสดใสปลอดโปร่งปราศจากริ้ว Anomaly ลัดวงจรพัดกวาดคืนสู่เมืองหลวงแกรนด์ซัมมิทพิทักษ์ธรรมชาติจริงสำเร็จ

เรย์ก้าวเท้าเหยียบยืนอยู่บนลานศิลาแท่นเจดีย์ปฐมกาลสัมบูรณ์ สบสายตากับภาพร่างจำลองมหาพิทักษ์อาร์เซอุสและหลวงพ่อเทนซินอย่างนิ่งสุขุมดวงตาสีเทาอันแสนคมกริบกู้โลก

“ผมขอเลือกทางเลือกที่สาม...” เรย์กล่าววาจาน้ำเสียงราบเรียบนิ่งสงบเด็ดเดี่ยวหนักแน่น “ผมจะยอมรับบทบาท ‘ผู้พิทักษ์นิรันดร์’ (Eternal Protector) พำนักอยู่ในโลกธรรมชาตินี้ตลอดกาลเพื่อประคองระบบ Firewall คุ้มกันชีวิตโนอา ลูคัส และคู่หูสแนปโซของผมทุกคน... แต่ผมจะใช้พลังเจตจำนงรบ Willpower คนนอกคอร์ ตรึงปฏิเสธระบบ wipe ลบความจำชิบูย่าของผมคงเหลือเก้าสิบเปอร์เซ็นต์นี้ไว้เป็นเหรียญกุญแจทองเหลืองกู้ระบบถาวรชั่วนิรันดร์!”

หลวงพ่อเทนซินคลี่ยิ้มกว้างกางขาสลัดยอมรับเจตจำนงแชมเปี้ยนซีโร่ “สมศักดิ์ศรีของแชมเปี้ยน VGC ที่มีหัวใจพิทักษ์มิตรแท้คนนอกระบบ... มหาเทพยอมรับทางเลือกที่สามรหัสแก้ไขแกนคอร์ของเจ้าอย่างสง่างาม!”

*วิ้ง... ติ๊ด... เปรี๊ยะ...!!!*

หน้าจอโฮโลแกรมกู้ระบบจำลองเด้งตัวสีทองสว่างจ้า ปลุกประจุรอยต่อสมดุลโลกเต็มร้อยเปอร์เซ็นต์:

[รายงานผลลัพธ์เลือกชะตา: ยืนยันทางเลือกที่สามกู้ชีพระบบ (Alternative Protocol Chosen)]
[ระดับความคืบหน้า Progress ประสานแกนคอร์: คงที่คงเหลือ 100% (World System Stabilized)]
[ระดับความทรงจำชิบูย่าของเรย์: เสถียรรหัสเต็มขั้น 90% (Safety Memory Lock preserved)]
[สเตตัสผู้คุ้มกันนิรันดร์: เริ่มต้นทำงาน (Eternal Protector Active)]

โนอาวิ่งพุ่งกระโจนโอบกอดคอเรย์น้ำตาไหลพรากยิ้มหวานสดชื่นยินดีล้นพ้นขอบเวที “เรย์! เธอทำได้แล้วเว้ย! ขอบคุณนะเว้ยที่ไม่ยอมทิ้งพวกเรากลับโลกเดิมทอดทิ้งมิตรแท้! ฉันจะอยู่เคียงข้างเธอพิทักษ์โลกใบนี้ตลอดไป!”

ลูคัสยกนิ้วหัวแม่มือขวากางยิ้มกว้างปาดเหงื่อ “สุดยอดแชมเปี้ยนผู้ไม่เคยเลือกแต่กำหนดชะตาตนเองได้สมบูรณ์แบบมากเพื่อนรักเรย์! พวกเราจะอยู่ร่วมรบพิทักษ์แกนทวีปปิดอาร์คเจ็ดสำเร็จดีเสถียร!”

เรย์คลี่ยิ้มกว้างละมุนอบอุ่นดวงจิตลูบหัวสแนปโซการ์ดีสองสัจธรรมกู่เห่าต้อนรับตะวันใหม่ ร่วมเดินทางก้าวพ้นวิหารแกนกลางกุมเป้าหมายสู่อนาคตสดใสพิทักษ์โลกธรรมชาติ Pokémon ถาวรชั่วนิรันดร์กาลอย่างกล้าหาญสงบสุขเด็ดเดี่ยวสัจธรรมปิดฉากอาร์คที่เจ็ดอย่างสมบูรณ์แบบร้อยเปอร์เซ็นต์!
ท่ามกลางความเงียบงันสลัวของห้องโถงปฐมกาลสัมบูรณ์ ไอความว่างเปล่าสีม่วงทมิฬยังคงสาดซัดถล่มรอยร้าวของแผ่นศิลาปิดกั้นทางลาดประจบศึก โนอาตาคลอน้ำใสจ้องมองดูตรากุญแจทองเหลืองกู้ระบบในมือของเรย์ด้วยความเศร้าสร้อยปนห่วงใยในชะตากรรมดวงจิต ลูคัสขยับยืนป้องหลังอย่างเหนียวแน่นมั่นคงเตรียมพร้อมรับสถานะละลายสลายร่างหากเกิดข้อผิดพลาด เรย์ดับแว่นตาโมโนเคิลวิเคราะห์ขวาเพ่งดวงตาคมประคองเจตจำนงรบปิดผนึกคอร์ 100% ร่วมกับสแนปโซการ์ดีทองคำสองสัจธรรมด้วยสายสัมพันธ์ประสานจิตใจสูงสุดยอด ปลดล็อกวิถีเลือกทางเลือกที่สามกู้ภัยระบบ Firewall โลกจริงโดยไม่มีการละทิ้งมิตรแท้ชั่วนิรันดร์
