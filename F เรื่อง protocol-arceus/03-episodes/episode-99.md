# ตอนที่ 99: "ส่วนผสมของอัคคีเทพและสสารว่างเปล่า"

*ครืน... ครืน...!!!*

เสียงระเบิดพังพินาศปนกระแสมิติลำดับสี่จากใจกลางป่าไผ่ด้านนอกแผ่คลื่นหนาหูสลับเสียงดัง *เคร้ง... เคร้ง...* ของลำต้นไผ่ทองเหลืองที่กระทบกันดังสะท้อนกังวานทวีคูณ เรย์และคีธยืนประคองสแนปโซการ์ดีทองคำไว้กลางแท่นวิจัยพลังงานศิลาหินทราย โดยมีเกโควาชิและโบกุเระคอยเฝ้าระแวดระวังแนวปากห้องแล็บไม่ให้ไอแปรวนแปรเข้ามาขัดจังหวะระบบวิจัย

คีธคีย์สลับเปิดแท่นควบคุมประสาทช่วยรบ Tier 3 เพื่อสแกนวิเคราะห์ลวดลายสสารบนเกราะอกของสแนปโซช้า ๆ เผยให้เห็นเส้นใยสีม่วงทมิฬสลับกับสีทองคำเหลวหมุนวนคล้ายเกลียว DNA นำเสนอความลับรหัส

“ดูสิเรย์...” คีธขยับจัดโมโนเคิลชี้รอยประกายแสงสว่างวับ “รหัสสสารความว่างเปล่า Pure Void Matter แฝงตัวอยู่ตรงรากคอร์วิญญาณของสแนปโซสลับกับประจุอัคคีเทพอย่างสมานดุล... นี่ไม่ใช่ภัยพิบัติแกลร่า แต่คือ ‘การรังสรรค์แบบผสมผสานพิเศษ’ (Composite Creator Code) ที่หาผู้เลียนแบบไม่ได้ในโลกจำลองเสมือนจริงใบนี้ที่คุณพ่อเธอสร้างไว้”

เรย์ก้มมองดูคู่หูสแนปโซ นัยน์ตาสีเทาสุขุมนิ่งสนิทลึกซึ้งห้วงคิดวิเคราะห์ VGC *‘มวลสสารสีม่วง Void มีน้ำหนักต้านดาเมจพิเศษและสปีดกวัดแกว่ง ส่วนอัคคีสีทองคำมีขีดล้างไอ Anomaly หากทั้งสองประจุทำงานประสานร่วมกันอย่างสมบูรณ์แบบโดยไร้แรงสะดุดกะพริบลัดวงจร... สแนปโซจะสามารถพุ่งชนทำลายเกราะตาข่าย Void และสลายสเตตัสบอสโกโดราหรือคิงดราได้เด็ดขาดปราศจากการพึ่งพาคำสั่งระบบช่วยรบ!’*

“คุณคีธครับ... ช่วยผมปรับความถี่สนามพลังงานทองเหลืองของวิหารเพื่อปรับสมานสมดุลโค้ดของสแนปโซให้เสถียรทีครับ” เรย์หันขอคำแนะนำเสียงเรียบนิ่งเด็ดเดี่ยว

คีธยิ้มกว้างกางขาสั่งไคริวมังกรยักษ์เปิดพัดคลื่นลมคุมทิศ “ได้เลยเจ้าหนุ่ม! ไคริวเปิดใช้งาน Tailwind (ลมหางหนุนสปีด) คุมทิศเหนี่ยวนำกระแสลมเหล็ก! โนอากางตัวเชื่อมต่อควบคุมประจุรหัสสแกนดึงคลื่นทองเหลืองสมาน!”

มังกรบินไคริวขยับปีกลม Tailwind พัดหางหนุนมวลความร้อนส้มสีทองกวาดคลุมรอบโต๊ะวิจัยสแนปโซลอยนิ่ง โนอาคีย์ปุ่มปรับความเสถียรกุญแจทองเหลืองของเรย์ชำระไอฝุ่นปูน สแนปโซแยกเขี้ยวหลับตารวมประจุเกลียวไฟสีทองคำเหลวและม่วงดำเข้าด้วยกันเกิดระเบิดแสงประกายทองเหลืองนวลกระจายรอบทิศสลายความสั่นสะท้านลงราบคาบสมบูรณ์แบบไร้ออร่าลัดวงจร!

หน้าจอทองคำระบบประสาทจำลองในสมองของเรย์แจ้งข้อมูลยืนยันความเสถียรขั้นสูงสุด:

[สัมผัสสายสัมพันธ์คู่หูรหัสสแนปโซ: ก้าวหน้าสะสม 60% (ระดับความเสถียรสสารสูงสุด)]
[ปลดล็อกขีดรบ Tier 3: ระบบสแกนหาขีดจำกัดความต่างของคิงดราสำเร็จร้อยเปอร์เซ็นต์]

*‘รหัสกู้ภัยสองสัจธรรมทำงาน...’* เรย์ลูบขนแผงขนสีทองอันแสนอบอุ่นของสแนปโซการ์ดีพลางเผยรอยยิ้มจางเด็ดเดี่ยวในดวงใจ *‘เช้าวันนี้ ศึกใหญ่ปิดฉากวิบัติภัยระดับสี่จะเปิดศึกกอบกู้หุบเขาไผ่ทองเหลืองนี้กลับคืนมาให้สำเร็จ!’*

คณะเดินทางรุกคืบก้าวออกจากห้องแล็บใต้ดินมุ่งตรงสู่วิหารป่าไผ่สลักเหล็กชั้นในสุดพร้อมรบเคียงคู่ทันทีอย่างมั่นใจ!
การปรับประสานรหัส DNA สองสัจธรรมของสแนปโซร่วมกับเมนเทอร์คีธและโนอาที่แท่นศิลาวิหาร สัญญาณ Tailwind ของไคริวช่วยเหนี่ยวนำประจุไฟกงล้อ Golden Flame Wheel ให้สมบูรณ์แบบไร้อาการสะดุดกะพริบขัดข้องมิติระบบช่วยรบ เรย์ลูบแผงขนสีทองคำอุ่นจัดของคู่ยากด้วยสายสัมพันธ์รบเต็มพิกัดเตรียมสู้คิงดราแปดตา
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
