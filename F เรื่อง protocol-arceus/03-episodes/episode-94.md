# ตอนที่ 94: "คำเตือนของพ่อในเครื่องฉายความรู้"

ประกายไฟสีครามสว่างจ้าจากขอบตู้เหล็กนิรภัยโบราณลอยพวยพุ่งขึ้นมาประสานระบบในห้วงความคิดของเรย์กะพริบถี่เป็นระนาบม่านแสงทอง เกิดริ้วคลื่นเสียงวิ้ง ๆ ในขีดประสาทช่วยรบ Tier 3 จิตใจของเรย์จมดิ่งก้าวลึกลงสู่นิมิตความทรงจำย้อนอดีตครั้งใหญ่ที่สลัดล็อกรหัสสะสม 90% ดั้งเดิมกระพริบฉายชัดดุจภาพถ่ายสดใสเรืองรองท่วมท้นจิตใจ:

ในภาพนิมิต... ร้านเกมการ์ดและตู้ตู้เกมเก่าของพ่อในย่านชิบูย่าเต็มไปด้วยแสงไฟสีเหลืองนวลอันแสนอบอุ่น กลิ่นกระดาษการ์ดใหม่และเสียงหัวเราะพุดคุยของเด็ก ๆ ดังสะท้อนกังวานอบอุ่นดวงจิต คุณพ่อนั่งลูบเข่าลูบหัวเรย์ในวัยเยาว์เคียงข้างเครื่องคอมพิวเตอร์เขียนโค้ดเก่าส่องสว่าง

“เรย์ลูกรัก...” พ่อเอ่ยเอ่ยวาจาน้ำเสียงอ่อนโยนแฝงประกายตาเด็ดเดี่ยวลึกซึ้งห่วงใย “พ่อจำยอมตั้งระบบ ‘กลไกนิรภัยล็อกความจำ’ (Safety Memory Lock) ไว้ในสมองของลูกเพื่อความปลอดภัยสูงสุดยามลูกก้าวเดินทางข้ามระนาบมิตินี้ในอนาคต... จำไว้นะ หากเกิดรอยแตกแยกวิบัติภัยธรรมชาติสสารว่างเปล่าลบข้อมูล สิ่งกั้นระบบที่จะช่วยให้ลูกชะล้างไอ Void โดยไม่ต้องสอยวิญญาณสลายคือสายสัมพันธ์ร่วมรบกับเพื่อนพ้องข้างกาย”

“อย่าลืมทางเลือกที่สามนะเรย์... ทางเลือกที่ลูกยอมสลัดความเย่อหยิ่งของแชมเปี้ยนไร้พ่าย VGC เพื่อปกป้องชีวิตเพื่อนร่วมทาง พลังไฟทองคำของสแนปโซที่พ่อเขียนฝังประสาทไว้คือรหัสนิรภัยที่จะช่วยกู้ชีวิตลูกและพากลับบ้านเดิมอย่างปลอดภัย!”

*วิ้ง... วิ้ง...!!!*

ม่านแสงนิมิตสีทองคำเหลวสลายวับปลิวหายไปดั่งฝุ่นผงกระจาย คลื่นเสียงเตือนภัยดับวูบลงช้า ๆ เรย์สะดุ้งตัวเบิกดวงตาสีเทาอันแหลมคมลุกโพลงขึ้นมาสบตากับโนอาและคีธที่จ้องมองมาด้วยความกังวลล้นพ้น หยาดเหงื่อเย็นผุดขึ้นเต็มแก้มขาวของเรย์พพยุงร่างยืนทรงตัวนิ่งสงบ

“เรย์! เธอเป็นอะไรไปน่ะ! จู่ ๆ ก็ยืนนิ่งเงียบสั่นเกร็งกะพริบตาเป็นสิบวิเลยนะ!” โนอาเอ่ยถามน้ำเสียงสั่นสะเทือนปนร้อนใจขยับดึงผ้าซับเหงื่อให้แก้อย่างรวดเร็ว

เรย์ระบายลมหายใจยาว สบตาสีเทาสุขุมนิ่งสนิทกับโนอาและคีธหนักแน่น “ผมปลดล็อกภาพรหัสความทรงจำของพ่อได้เสร็จสมบูรณ์แล้วครับ... พ่อเตือนผมเรื่อง Safety Memory Lock และยืนยันรหัสหนทางเลือกที่สามเพื่อความเสถียรโลกสอดคล้องไฟทองคำของสแนปโซ สแนปโซไม่ได้เกิดขึ้นตามความคลั่งมิติแร้นแค้น แต่แกเป็นรหัสคีย์ป้องกันกู้ชีพที่พ่อตั้งไว้ตั้งแต่เริ่มต้นแรกแล้วเพื่อช่วยพวกเรา!”

[ระบบรายงานความก้าวหน้าความทรงจำแชมเปี้ยน: กู้คืนสำเร็จสะสม 90% (เสถียรรหัสเต็มระบบ)]
[ปลดล็อกขีดสุดระดับสามระบบช่วยรบ Tier 3: กำลังสแกนหาขีดสุดความเหนี่ยวนำคิงดรา]

คีธพยักหน้ามั่นใจลูบเคราปัดเป่าฝุ่น “พินัยกรรมและความรักของพ่อเธอมันยิ่งใหญ่น่าทึ่งมากเรย์... ในเมื่อคีย์กู้ระบบพร้อมเพรียงรบรุก... พรุ่งนี้รุ่งอรุณศึกตัดสินสยบ Anomaly Kingdra ที่วิหารป่าไผ่สลักเหล็กชั้นในสุด จะเป็นเวทีชะล้างรอยแยกมณฑลเหนือนี้ให้ดับสูญไปถาวร!”

เรย์ลูบหัวสแนปโซการ์ดีทองคำที่เห่าเสียงใสอย่างผูกพันและไว้ใจเต็มร้อยขีดประสาท *‘ศึกพรุ่งนี้... ฉันจะสู้รบไม่ใช่ในฐานะบอท VGC ชิบูย่า แต่เพื่อสลัดมิติว่างเปล่ารักษาผืนแผ่นดินป่าไผ่ทองเหลืองนี้ไว้เคียงข้างเพื่อนแท้ทุกคน!’*
เรย์ลืมตาสีเทาอันคมกริบกู้คืนสติสัมปชัญญะหลังนิมิตความทรงจำคุณพ่อชิบูย่าเรื่อง Safety Memory Lock ปลดล็อกเสถียรประสาทสมาธิ Tier 3 เต็มพิกัด สแนปโซเห่าคำรามเรียกเปลวไฟทองอุ่นจัดพยาบาลความหนาวเย็น คีธและโนอาเร่งเตรียมอุปกรณ์ผนึกพร้อมรบรุกบุกห้องโถงวิหารใต้ดินลึกปราบคิงดราแปรปรวนในเช้าวันถัดไปอย่างไม่กลัวเกรงภัย
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
