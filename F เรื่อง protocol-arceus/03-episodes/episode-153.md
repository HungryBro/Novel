# ตอนที่ 153: "ศึกสู้รบความเร็วธรรมชาติของเกโควาชิ"

*ตูม... บึ้ม...!!!*

สะเก็ดเกล็ดหิมะเหล็กและไอแปรปรวนสีม่วงเข้มปะทุระเบิดกระจุยกระจายข้ามแนวกรงหน้าต่างวิหารลึกรอยแยกตะวันตก มานิวลา (Anomaly Weavile) ขยับร่างเพรียวลอยตัวต่ำบนดินน้ำแข็ง สลัดกรงเล็บคมยิงคลื่นสสารมืด *Void Claw* (กรงเล็บสอยความว่างเปล่า) ดาหน้ากระแทกสกัดขบวนรบเรย์รุนแรงสะเปรี้ยวปนเสียงตูมตามหนาแน่นขัดรหัสช่วยรบ

เรย์ยืนเบี่ยงกายหลบสะเก็ดน้ำแข็งคมจัด นัยน์ตาสีเทาสุขุมนิ่งสนิทภายใต้แว่นตาโมโนเคิลขวาประเมินผล VGC ในสมอง Tier 3 ที่แจ้งรหัส Error ลัดวงจรกะพริบถี่ยิบเกือบร้อยเปอร์เซ็นต์สะกดใจ

“เกโควาชิครับ! สบสายตากบประสานใจพร้อมกู้สปีดธรรมชาติประจัญรบเดี๋ยวนี้!” เรย์ตะโกนสั่งวาจาน้ำเสียงราบเรียบนิ่งสงบข้ามเสียงลมพายุกวัดแกว่ง

เมื่อม่านพิกัดหน้าจอรหัสจำลองสลายตัว เรย์ใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวสนามรบร่วมคู่หู:

*ละอองฝุ่นน้ำแข็งปลิวว่อนเฉียงใต้ขวาด้วยสปีดลมหนาว...*
*เล็บขวาของมันมีแสงคริสตัลแก้วสว่างไสวสีม่วงเข้มสะกดประจุไฟ Void...*
*ข้อต่อไหล่ขวาของมันสั้นเกร็งพราดยามหมุนปัดชาร์จตัว!*

นี่คือจุดบกพร่องสัจธรรมกายภาพจริงของมานิวลา Anomaly!

“เกโควาชิ! ชิงเปิดจังหวะรุกด้วยสปีด Tailwind หนุนความเร็ว! Aerial Ace (เหินเวหาจู่โจมฉับพลัน) ปาดดาบน้ำฟันสับฉับเข้าใส่ข้อไหล่ขวานั่นเดี๋ยวนี้เลย!” เรย์ตะโกนชี้นิ้วสั่งการขวางมลพิษเสียงกึกก้องลั่นวิหาร

กบนินจาเกโควาชิกระโดดชาร์จตัวจากแผงหินทราย ร่างกายกะพริบตาเร็วแสงทองสว่างวับพุ่งทะลักผ่านคลื่นไอ Void ปาดดาบน้ำกายภาพสับเปรี้ยงชนขวับเข้าข้อไหล่ขวามานิวลา Anomaly อย่างแม่นยำรวดเร็วระดับสปีดสูงสุดสะเด็ดขาด!

*เพล้ง... ตูม...!!!*

เสียงระเบิดกรงเล็บเหล็กมานิวลาสลายตัวดังกึกก้องกังวาน ปลดปล่อยละอองทองชะล้างไอเย็นจัดกวาดล้างรอยแยกมณฑลตะวันตกจนหมดสิ้น มานิวลา Anomaly ทรุดเอียงตัวล้มคว่ำราบไปบดดินทรายสลับสนิทไร้ประกายบ้าคลั่งหมดสภาพการรบราบคาบสิ้นเชิงในทันที!

“สยบด่านปะทะด้วยสปีดธรรมชาตินินจาสำเร็จแล้วเรย์! ทางลอดวิหารกู้ภัยเปิดออกแล้วเว้ย!” โโนอาตะโกนลั่นยิ้มกว้างปาดน้ำตาชื่นใจ

เรย์กระชับเสื้อคลุมพุ่งตัวนำแถวขบวนคู่ยากเข้าสู่ใจกลางวิหารรัง Tyranitar ทันทีอย่างเด็ดเดี่ยวเด็ดขาดปราบภัย!
กระแสลมหนาวจัดกรีดร้องพัดกวาดความเย็นยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกยะเยือกเกาะผิวแก้ว ผ่านซอกหลืบกำแพงน้ำแข็งยักษ์ของหุบเหวหิมะขาวสูญสิ้นดึงเศษละอองเกล็ดหิมะสีม่วงทมิฬ Void ให้ลอยว่อนร่วงหล่นปะทะแก้มขาว ทุกตารางนิ้วบนผนังหินศิลาร้าวแยกแผ่ริ้วรังสีความแปรปรวน Anomaly บิดมิติสลับขั้วกลืนมวลวัตถุธรรมชาติ เรย์เพ่งนัยน์ตาสีเทาสุขุมดับสนิทหน้าจอประมวลผลประสาทช่วย Tier 3 เพื่อใช้สายตาจริงมองหาแนวทิศรอยพับขอบบิดเบี้ยวเส้นตารางกริดมิติ Void Grid Lines จุดอ่อนบอสใหญ่บันกิรัส Tyranitar Anomaly อย่างสงบนิ่งดั่งเหล็กกล้า สแนปโซแยกเขี้ยวแผ่ประกายอัคคีเพลิงทองคำเหลว Golden Flame Wheel ชะล้างลบไอเย็นแช่แข็งเพื่อปกป้องอบอุ่นและพยาบาลกำลังรบของโนอาและจูลส์เกโควาชิอย่างเคียงข้างมิตรภาพชีวิตจริง
