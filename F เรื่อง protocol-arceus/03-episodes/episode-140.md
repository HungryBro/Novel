# ตอนที่ 140: "ม่านพลังงาน Void บุกถล่มลานประลอง"

*วิ้ง... ติ๊ด... เปรี๊ยะ...!!!*

เสียงระเบิดกระแสมิติลำดับห้าดังลั่นสนั่นลานประลองหลักกลางแจ้งระหวางด่านศึกรอบก่อนรองชนะเลิศ ดึงความอบอุ่นยามเช้าสลายตัวลงไปในเสี้ยววิพิกัดสแกน หน้าจอภาพโฮโลแกรมยักษ์เหนือเพดานเวทีช็อตดับวูบลงชั่วพริบตาเดียวนำพาไฟมืดสิบด้าน ก่อนที่ม่านรังสีสีม่วงทมิฬขนาดมหึมา ‘ม่านกักมิติความว่างเปล่า’ (Void Barrier Dome) จะกางเปิดตัวขึ้นกักขังลานและฝูงชนชาวเมืองแกรนด์ซัมมิทนับพันไว้เบื้องในหนาแน่นหมดทางเดินสัญจร

เบื้องหลังแนวม่านสีทมิฬ สมาชิกกลุ่มเสียงประสานเสื้อคลุมเทาลายดวงตาร้องไห้เกือบสามสิบคนปรากฏตัวชี้นิ้วสั่งขวางหน้าปล่อยออร่า Anomaly

“เรย์! ม่านกักมิติ Void กำลังสกัดถอนสิทธิ์ไฟฟ้าลัดวงจรลบล้างระบบ Firewall ทั่วทวีปเหนือแล้วค่ะ!” โนอาตะโกนลั่นชี้นิ้วรายงานอย่างหวาดกลัวขีดจำกัดสแกนสถิติ “ค่ามลพิษสะสมกระเพื่อมทะยานแตะหกสิบแปดเปอร์เซ็นต์สะสมแล้ว! นี่คือภัยวิบัติรบระดับสี่คุกคามลบข้อมูลโลกธรรมชาติ!”

*“โฮก...!!!”*

โปเกมอนรูปร่างประมวลคลั่ง Anomaly ‘โดเบล’ และ ‘ดอนแฟน Anomaly’ ยักษ์สองตัวกระโดดพุ่งร่างเหินเลื้อยข้ามเวทีประลองขว้างสะเก็ดไอสีม่วงเข้มสลายโครงสร้างหินศิลาจนแตกละเอียดร้าวหลุดหายวับกลายเป็นแก้ว

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทภายใต้โมโนเคิลวิเคราะห์ขวาช่วยรบ Tier 3 แจ้งเตือน:

[ตรวจพบรหัสวิบัติภัยลานประลอง: ม่านกักมิติความว่างเปล่ากางระบบ (Void Barrier Spawning Event)]
[ระดับการกักรังสีประสาทสมอง: ผู้เล่นสแกนข้อมูลบอสไม่ได้ร้อยเปอร์เซ็นต์ (Interference Mode)]
[ระดับสายสัมพันธ์คู่หูรบประสาน: สแนปโซ 68 | เกโควาชิ 62 | อีวุย 55]
[รหัสจูลส์ประสาทร่วมรบ: พร้อมกู้ชีพพยุงความเสถียร]

*‘ม่านกักมิติขวางทิศพิกัด...’* เรย์จดจ้องมองจังหวะขยับตัวประมวลสมการในหัวดั่งมีดโกน *‘บิชอปวาลด์จงใจกางม่านกักลานเพื่อบีบให้พวกเราถอยระบบกู้ภัยไปไหนไม่ได้ และใช้ฝูง Anomaly กวาดลบข้อมูลชาวเมืองสลายรหัสหลัก หากเราไม่มีทักษะสปีด Gale Wings หรือดาบน้ำกายภาพฉีกม่านประสานงาน จะโดน Rollout บดกระแทกเป็นผงทรายแน่!’*

“จูลส์ครับ! สั่งโฮโดรอนมังกรเหล็กเปิด Flash Cannon (ปืนลำแสงเหล็กสว่าง) ยิงถล่มตรงกลางม่านกักเพื่อเปิดรอยเชื่อมช่องทางระบายความร้อน! เกโควาชิ! ตั้งท่า Aerial Ace ขวับบิดตัวชาร์จดาบวารีฉีกร่างดอนแฟนฝั่งซ้ายขวา!” เรย์กู่สั่งเสียงกึกก้องลั่นสะท้านวิหารลึกปักธงรุก

“เข้าใจแล้วเว้ยแชมป์ซีโร่! โฮโดรอน! ปืนแสงเหล็กถล่มพิกัดยิงเปรี้ยงพังด่านลานพวกมันไปเลยเว้ย!” จูลส์ตะโกนกู่สั่งรบร่าเริงขยับข้อมือกราด

การต่อสู้ทลายม่านกัก Void พิทักษ์ดวงจิตประชากรโลกประลองสัจธรรมเดือดพล่านขึ้นทัวร์นาเมนต์ยอดเขาในทันทีอย่างหาญกล้ามั่นคงรบรุก!
รอบบริเวณห้องโถงวิจัยและลานประลองแกรนด์ซัมมิทอารีน่ายังคงอบอวลไปด้วยไอหมอกเย็นชื้นปนกลิ่นกระแสไฟฟ้าจากท่อไอน้ำใต้ดินที่พ่นระเหยควันขาวฟุ้งกระจาย แสงไฟนีออนสีส้มเหลืองและเขียวสะท้อนผ่านผลึกเกล็ดหิมะบางเบาที่ร่วงหล่นลงมาทับถมแนวกำแพงศิลาหนาเตอะดึงทัศนียภาพหนาวเหน็บ โนอากระชับผ้าพันคอหนาสีกรมท่าคอยคีย์ปรับความเสถียรข้อมูลระบบวิจัยอย่างใส่ใจดูแลในขณะที่สแนปโซและเกโควาชิเฝ้าระแวดระวังแนวขอบเขตสนาม เรย์ใช้สายตาสีเทาสุขุมนิ่งสนิทจับจ้องวิเคราะห์ริ้วสัญญาณรหัสสีทองของตารางกริดจำลองกู้ภัยผ่านโมโนเคิลขวา พยายามประสานพลังเจตจำนงรบ Willpower ขั้นเต็มร้อยเปอร์เซ็นต์เพื่อชะล้างลบรอยร้าวมลพิษ Anomaly ข้ามขีดปวดล้าสะสม การเดินทางร่วมทาง VGC ในชีวิตจริงที่เกิดจากความผูกพันและสายสัมพันธ์ร่วมรบพยุงระดับความมั่นคงสมาธิประสาทช่วยรบ Tier 3 ให้คงที่เสถียรรหัสเต็มพิกัดโดยไม่มีการยอมจำนนต่อกฎบิดเบี้ยวมิติ
