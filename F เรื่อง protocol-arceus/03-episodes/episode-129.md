# ตอนที่ 129: "ความพ่ายแพ้ของเมทากรอสและการปิดแกน"

*ฟู่... ฟู่... เกร๊ง...!!!*

สะเก็ดเปลวไฟทองคำและละอองเหล็กเหลวชะล้างไอความแปรปรวนสีม่วงดำจนดับมลายหายวับสลายตัวสิ้นเชิง เมทากรอส (Anomaly Metagross) ของบิชอปวาลด์ล้มคว่ำราบสลับสนิทไร้รอยบิดเบี้ยว รอยแยกใหญ่ปากบ่อถลุงสลับประสานขอบทางลาดคืนสัจธรรมฟิสิกส์ธรรมชาติแรกรบกู้โลกกวาดล้างรอยร้าว

ทว่าบิชอปวาลด์ที่ได้รับความเสียหายสะสาง สัญญาณกายสั่นระริกเกร็งข้อมือคว้าฉุดดึงคีย์รหัสชีวิตลูคัสที่ล่ามตรึงในโซ่ไอ Void ดีดสปริงเปิดพอร์ทัลเคลื่อนย้ายมิติทลวงอีกช่องหนึ่งลึกสุดตอนเหนือสุดอย่างรวดเร็วและบ้าคลั่ง!

“ฮ่า ๆ ๆ ! แกนปิดกู้สำเร็จเสถียรแล้วแชมเปี้ยนซีโร่...” บิชอปวาลด์แผดเสียงหัวเราะไอความแค้นเดือดพล่านสลัว “แต่ตัวตนของลูคัสจะโดนพวกเรานำดิ่งลึกเข้าไปประกอบพิธีกรรมแท่นแทรกคอร์ที่ ‘ยอดเขาหิมะขาวสูญสิ้น’ (White-Peak Abyss) ทวีปเหนือสุดขอบแร้นแค้น! ถ้ากล้าแกร่งนักจงก้าวปีนเขาหิมะท้าความตายตามมาซะ!”

*วิ้ง... ตูม...!!!*

ร่างของบิชอปวาลด์ลูคัส และกรงขังไฟแอโร่วิวัฒน์แล้ว กวาดปลิวร่วงลับหายเข้าพอร์ทัลเหนี่ยวนำขวาหลบหนีไปทันใด ทิ้งไว้เพียงประกายความหนาวเหน็บเกล็ดหิมะป่าหนาวสลัดเกาะผนังอู่ซ่อมรถร้างสว่างวาบประสาทสะท้าน

เรย์ก้าวเท้าพึ่งมั่นคงขึ้นแท่นบูชากลาง ยื่นกานิ้วมือขวากุมตรากุญแจทองเหลืองปิดตราประทับตรงแกนกลาง ‘คริสตัลแกนลำดับห้า’ (Fifth Core Crystal) ของโรงถลุงเหล็กช้า ๆ ทรงพลังดึงแสงรังสีสีทองประสานแกน

รังสีแสงสีทองสว่างจ้าเรืองรองทับซ้อนระบบจำลองจำลองในหัวเรย์แจ้งรายงานผล Progress ลุล่วงมั่นคง:

[ยืนยันการปิดผนึกแกนรอยแยกมิติระดับห้า: โรงถลุงเหล็กแกนกลาง (Origin Core Sealed)]
[ความก้าวหน้าการกู้คืนสมดุลโลกสะสม: ปลดล็อกแตะขีด 65% (Progress 65% Reached!)]
[ความเสถียรของ Firewall ภูมิภาคเตาหลอม: สมบูรณ์ 100% (System Security Level: High)]
[ระดับการปลดล็อกความทรงจำชิบูย่า: สะสม 90% (คงที่เสถียร)]

*“ Progress 65%... ค่าสมดุลกึ่งหนึ่งผ่านพ้นสะสมใกล้สุดปลาย...”* เรย์เอ่ยพึมพำกระซิบเสียงเรียบนิ่งเด็ดเดี่ยว นัยน์ตาสีเทาสุขุมทอดมองเกล็ดหิมะป่าหนาวแสนแห้งแล้งที่เกาะซากประตูด่านเหนือลานรบ *‘พวกมันล่าลูคัสก้าวขึ้นเหนือสุด... ศึกสุดท้ายเพื่อชะล้างมิติกวาดล้างลัทธิจะปะทุขึ้นกลางพายุหิมะขาว... ฉันจะไม่ทรยศต่อสัญญาและความไว้วางใจของเพื่อนแท้คนนี้เด็ดขาด!’*

เรย์กุมซากบอลฉีกขาดของลูคัสในข้อนิ้วแน่น ดึงสแนปโซการ์ดีและเกโควาชิร่วมแถว มุ่งก้าวเท้ากลับไปรับโนอาเพื่อเตรียมเปิดแผนรุกคืบขึ้นสู่ภูมิภาคทวีปเหนือถัดไปทันทีอย่างเด็ดเดี่ยว!
บิชอปวาลด์สอยฉุดรหัสลูคัสและกรงไฟแอโร่หลบหนีเข้าพอร์ทัลมิติดิ่งลึกตอนเหนือสู่ยอดเขาหิมะขาวสูญสิ้น White-Peak Abyss ทิ้งเกล็ดหิมะบางเบา เรย์ก้าวปิดตรากุญแจผนึกคริสตัลแกนลำดับห้า กอบกู้ Progress โลกเสร็จสมบูรณ์แตะค่า 65% Firewall เสถียรดี
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
