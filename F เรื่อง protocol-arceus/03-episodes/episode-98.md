# ตอนที่ 98: "ข้อมูลที่รั่วไหลและต้นกำเนิดสแนปโซ"

*บึ้ม...!!!*

แรงระเบิดจากระเบิดควันแปรปรวน Anomaly สีทมิฬปะทุขึ้นกลางห้องแล็บวิจัยชั้นในสกัดพิกัดขวางหน้า เผยให้เห็นเงานักพรตสูงวัยของกลุ่มเสียงประสานที่อาศัยช่องว่างความสลัวควบเศษข้อมูลสำรอง (Backup Data Crystal) ส่วนหนึ่งขึ้นยานมิติพรางตัวหนีหายลึกลับ ทิ้งคราบไอความแปรปรวนสีม่วงฟุ้งกระจายขจัดรหัสรอยต่อระบบ

“แย่แล้ว! พวกมันชิงเศษข้อมูลรหัสสถาปนิกเรื่องประจุ Firewall ส่วนที่เหลือหนีไปได้ค่ะเรย์!” โนอาตะโกนลั่นชี้นิ้วพยายามคีย์ประคองแผงควบคุมระบบ แต่ควันดำทำสลายสัญญาณลัดวงจรช็อตเสียงดังวิ้ง ๆ น่าเกลียด

คีธย่ำเท้าพาไคริวมังกรยักษ์ก้าวพ้นประตูดินทรายอุโมงค์เข้ามา สมทบในห้องวิจัย ใบหน้าคมเข้มปัดฝุ่นดำออกช้า ๆ “สกัดพิกัดป้องกันกองหลังข้างนอกได้เกือบหมด... แต่ดูเหมือนพวกมันจะได้ข้อมูลประจุพลังทองคำไปบางส่วนแล้วเพื่อชาร์จระบบคอร์ของพวกมันในการดัดแปลงกฎโลก”

คีธปรายตาสีเทาฟ้ามองสแนปโซการ์ดีทองคำที่ยืนคุมเชิงด้วยดวงตาสีทองสดใสสว่างวับ นิ้วกอดลูบขนแร่อุ่นของสแนปโซ “แต่สิ่งสำคัญที่สุดตอนนี้ไม่ใช่แผ่นดิสก์ข้อมูลนั่นหรอกเรย์... ความจริงเรื่อง ‘ต้นกำเนิดแท้จริงของสแนปโซ’ คือคีย์สำคัญที่คุณพ่อของเธอพยายามปกปิดป้องกันไว้ตลอดสิบห้าปีเต็ม”

“ต้นกำเนิดของสแนปโซเหรอครับ?” เรย์ขมวดคิ้วถามสบดวงตาสีเทาของแชมเปี้ยนรุ่นพี่อย่างนิ่งนิ่งสุขุม

คีธพยักหน้าช้า ๆ นั่งลงข้างกองกระดาษวิจัยเก่า “สแนปโซไม่ได้เกิดตามธรรมชาติในภูมิภาคแอชโกรฟหรือมีบันทึกใน Pokédex มาก่อน... ฌอนและข้าพบบันทึกรหัสลับว่า... สแนปโซถูกสร้างขึ้น ณ ใจกลางแกนรอยแยกมิติระดับห้าตอนเหนือ โดยคุณพ่อของเธอใช้ ‘ส่วนผสมระหว่างอัคคีเทพอาร์เซอุส’ (Divine Flame) และ ‘สสารความว่างเปล่าบริสุทธิ์’ (Pure Void Matter) หลอมรวมร่างรหัสพันธุกรรมเข้าด้วยกันดั่งสิ่งมีชีวิตสองสัจธรรมผสมผสาน!”

“นี่คือเหตุผลที่สแนปโซแผ่เปลวไฟสีดำอมส้ม (Dark Flame) ในเทิร์นปะทะแรก ๆ และสามารถเปลี่ยนสภาพเป็นเปลวเพลิงทองคำแปรปรวน (Golden-Void Fire) ได้ในเวลาถัดมา... แกเป็นสิ่งเดียวในโลกจริงใบนี้ที่มีรหัสสะเทินสลายขอบเขตความว่างเปล่าได้ร้อยเปอร์เซ็นต์โดยไม่มีผลเสียหายต่อแกนคอร์ระบบโลกธรรมชาติ!”

โนอาอ้าปากกว้างเบิกดวงตาสลัวตกตะลึงล้นพ้น “แสดงว่า... สแนปโซคือ firewall ร่างกายสิ่งมีชีวิตที่ถูกสร้างมาเพื่อชะล้างลบรอยแยกมิติโดยตรงเฉกเช่นกุญแจทองเหลืองของเรย์งั้นเหรอคะอาจารย์คีธ!”

“ใช่แล้วแม่หนู...” คีธตอบเสียงทุ้มนุ่มทรงพลัง “และคุณพ่อของเรย์ยอมล็อกความทรงจำยี่สิบเปอร์เซ็นต์ของลูกชายไว้ก็เพื่อไม่ให้สมองเรย์สะดุดกับการมีสัญญารหัสสสารความว่างเปล่าในตัวสแนปโซจนลัดวงจรลบล้างตัวตนกันเอง!”

หน้าจอทองคำระบบประสาทในสมองของเรย์สั่นระรัวแจ้งยืนยันค่ารหัสพิเศษ:

[ปลดล็อกสถิติสัมพัทธ์สัมบูรณ์: สแนปโซ — สิ่งมีชีวิตสองสัจธรรม (Dual-Aspect Code Unlocked)]
[ความทรงจำแชมเปี้ยนปลดผนึกก้าวหน้าสะสม: 90% (เสถียรรหัสเต็มขั้น)]

เรย์ทรุดตัวลงลูบขนหน้าอกอุ่นจัดของสแนปโซการ์ดีทองคำ สบดวงตาสีทองของมันด้วยประกายดวงตาอันแสนอบอุ่นและไว้ใจเป็นที่สุด “ไม่ว่าเธอจะเกิดจากอะไรสแนปโซ... เธอคือมิตรแท้และคู่หูร่วมเดินทางที่ฉันจะปกป้องและร่วมสู้ปิดฉากภัยพิบัตินี้ไปด้วยกันตลอดกาล”

สแนปโซการ์ดีทองคำกู่คำรามเสียงกังวานดั่งยืนยันสายสัมพันธ์ร่วมศึกที่ไม่มีวันทำลายล้างได้!
แม้เศษข้อมูลสำรองวิจัยบางส่วนโดนคัลต์ช่วงชิงหนีไปได้ทางมิติบิดขั้ว ทว่าคีธเปิดเผยความลับกำเนิดสแนปโซการ์ดีทองคำที่เป็นสิ่งมีชีวิตสองสัจธรรมหลอมรวมระหว่างอัคคีเทพและสสารว่างเปล่า Pure Void Matter ทำให้แกเป็นกุญแจทองเหลือง Firewall ที่ชะล้างไอ Void ได้โดยไม่มีผลกระทบเสียหายสะท้อนลบข้อมูลแกนคอร์ระบบโลกจริง

เปลวไฟริบหรี่ขยับไหวโอนเอนตามสายลมดั่งความหวังที่ยังไม่ยอมดับสูญลงในค่ำคืนนี้
