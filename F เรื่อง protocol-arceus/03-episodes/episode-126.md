# ตอนที่ 126: "คลื่นพลังจิตไร้หน้าต่างสแกนระบบ"

*ตูม... ตูม... บึ้ม...!!!*

กระแสคลื่นพลังจิตแร้นแค้นแผ่ความชื้นจัดกระแทกลานเหล็กของโรงถลุงเหล็กแกนกลางพังทลายสล้างละเอียดเศษเสี้ยวลอยละลิ่ว เมทากรอส (Anomaly Metagross) ขยับร่างยักษ์ลอยตัวเหนือบ่อเหล็กหลอมเดือด ยกแขนเหล็กคริสตัลขวาหวดหวดฟาดคลื่นสสารดำ *Meteor Mash* (พลาสม่าดาวตก Void) ดาหน้ากระแทกสกัดขบวนรบเรย์รุนแรงสะเปรี้ยวปนเสียงตูมตามหนาแน่น

เรย์ยืนเบี่ยงตัวหลบสะเก็ดไฟร้อนจัด นัยน์ตาสีเทาสุขุมนิ่งสนิทจ้องมองผ่านโมโนเคิลขวาที่กำลังกะพริบรหัส Error สีแดงสะดุดตาถี่ยิบ หน้าต่างแสดงผลพารามิเตอร์จุดอ่อนสูญสิ้นสลายตัว ดั่งกระดานเกม VGC ที่โดนลบปุ่มควบคุมการเดินหมากออกไปสิ้นเชิงประมวลผลเอเร่อล้มพินาศ

“เกโควาชิ! หลบไปทางซ้ายระเบิดควันพรางพิกัด! โโบกุเระใช้รากไม้พันเสาดักสปีดมันดึงเทิร์นรุก!” เรย์ตะโกนสั่งวาจาน้ำเสียงราบเรียบนิ่งสงบข้ามเสียงคลื่นพลังจิตสะเทือนขวัญ

ทว่าคลื่นพลังจิต *Psychic Void Wave* ของเมทากรอสแผ่กระจายตัวทะลุสแกนมิติหลบหลีก ทรงพลังข้ามหน้าต่างระบบสแกนควบคุมประสาท พุ่งชนปะทะกลางอกโบกุเระตอไม้น้อยจนตอไม้ปลิวกระเด็นลื่นไถลไปบนลานเหล็กร้าวละเอียดส่งเสียงร้องเจ็บปวด ค่าน้ำหนักแถบเลือดสีทองดิ่งร่วงทรุดเหลือเพียงยี่สิบเปอร์เซ็นต์ปนสลบหมดสภาพรบในชั่ววิครากสูญสิ้นความมั่นคง!

“โบกุเระหมดสภาพการรบกู้ภัยแล้ว!” บิชอปวาลด์หัวเราะแผดเสียงก้องกังวานดุจปีศาจเหล็กดุดันสะใจ “ในสภาพไร้หน้าจอจำลองวิเคราะห์คอร์... เธอมันก็แค่เด็กหนุ่มตาบอดที่กิ๊กก๊อกไร้หนทางต่อกรกับข้า! ยอมโดนบดขยี้สลายร่างไปซะ!”

เมทากรอสกู่ก้องร้องคำรามขู่กวาด เรียกเกลียวพายุหมุดดาวตกพลาสม่าดำล้อมกรอบตัวเรย์และสแนปโซการ์ดีทองคำขังไว้เหนือน้ำเหล็กเดือดพวยพุ่ง

เรย์ทรุดเข่าข้างขวาเหยียบดินแข็งพยุงโบกุเระกลับเข้าบอลอย่างถนอมใจ รวบรวมจิตเจตจำนงรบสถิติ VGC สะสม *‘การมองไม่เห็นหน้าจอระบบวิเคราะห์... ไม่ใช่จุดสิ้นสุดเควส... พ่อบอกผมเสมอว่า... รหัสกู้ภัยที่แท้จริงสลักลึกอยู่ในดวงตาสองข้างของผม ไม่ใช่ริ้วภาพจำลองจำลองในสมอง! ฉันจะปิดระบบช่วยรบ Tier 3 นี้ทิ้งและก้าวรบกู้ชีพด้วยสัมผัสธรรมชาติจริงด้วยสายตาของตนเอง!’*

เรย์ยื่นมือซ้ายกดปุ่มปิดการสลับแสดงผลหน้าจอโมโนเคิลพารามิเตอร์ขวาจนหน้าจอด้านประสาทดับสนิทวับ คืนสู่ความเงียบงันสัจธรรมเด็ดเดี่ยวในดวงใจ!
เมทากรอส Anomaly ใช้พลาสม่าดาวตก Meteor Mash และคลื่นพลังจิตไร้หน้าต่างสแกน Psychic Void Wave พุ่งปะทะฟาดอกโบกุเระสลบหมดสภาพการรบกู้ภัย บิชอปวาลด์เยาะเย้ยท้าทายเด็กหนุ่มแชมป์ตาบอด เรย์ตัดสินใจสั่งปิดจอช่วยรบ Tier 3 พึ่งประสาทตาเปล่าธรรมชาติจริง
ไอควันเขม่าถ่านหินดำหนาทึบและเศษขี้เถ้าโลหะยังคงปลิวว่อนร่วงหล่นลงมาปกคลุมทุกตารางนิ้วในโรงงานถลุงแร่เหล็กกล้าแห่งนี้ กลิ่นน้ำมันเตาร้อนอบอวลผสมผสานกับกระแสลมเย็นยะเยือกของมวลสสารความว่างเปล่าสีม่วงทมิฬที่แผ่ออกมาจากรอยร้าวมิติ ทุกโครงสร้างคานเหล็กและท่อไอน้ำยักษ์ส่งเสียงครางครืดคราดสลับการสั่นสะเทือนสั่นสะท้านจากแรงบีบอัดของพอร์ทัล Void เรย์เพ่งสายตาจริงสังเกตทิศทางของประกายไฟและเศษแบริ่งเครื่องจักรเพื่อคำนวณเวลาย่อยในการกู้ระบบ Firewall ธรรมชาติจริงโดยไม่ยอมจำนน สแนปโซแยกเขี้ยวแผ่เปลวอัคคีสีทองเหลวชะล้างคราบเขม่าสนิมเหล็กบนผิวหน้าดิน เกโควาชิแฝงกายในเงาฝุ่นควันคอยสกัดพิกัดป้องกันซ้ายขวาอย่างสุขุมและมั่นคงข้ามขีดความล้าสะสม
