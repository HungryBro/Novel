# ตอนที่ 101: "เขาวงกตไผ่และจุดแกนกลางรอยแยก"

เสียงลมพายุแปรปรวนสีม่วงดำดังเลื่อนลั่นพัดสะท้านพัดผ่านแนวกอไผ่ทองเหลืองยักษ์เบียดเสียดกันเป็นประกายเสียงดังกริ๊งกริ๊งเคร่งเครียดสะกดขวัญ เรย์ โนอา ลูคัส และคีธก้าวเท้าย่ำฝุ่นข้ามสะพานดินลาดลงมาสู่ใจกลาง ‘เขาวงกตไผ่สลักเหล็ก’ (Steel-Etched Bamboo Maze) พื้นที่ส่วนวิหารชั้นในสุดที่เป็นปมศูนย์กลางความบิดเบี้ยวของภูมิภาคที่สี่ส่งคลื่นเสียงสะท้อน

ทัศนียภาพรอบด้านหมุนบิดสลับทิศทางอย่างผิดธรรมชาติน้ำไหลย้อนฟ้า กอไผ่เหล็กสีทองเหลืองงอกคดเคี้ยวพันกันเป็นแนวกรงขังจำลองกั้นสัญจรหลายสิบจุด ทุกครั้งที่ระบบช่วยรบ Tier 3 สแกนวิเคราะห์ กราฟสีทองและสีแดงกะพริบแจ้งเตือนระดับรอยแยกอย่างสับสนรวดเร็ว:

[เข้าสู่ใจกลางแกนรอยแยกมิติ: ห้องโถงวิหารแกนป่าไผ่ (Core Labyrinth Zone)]
[ความหนาแน่นสสารความว่างเปล่าสะสม: 48% (ระดับเหนี่ยวนำขีดวิบัติ)]
[การสั่นสั่นลัดวงจรของระบบคอร์ธรรมชาติ: เพิ่มขึ้นเป็น 15%]
[สิทธิ์ผ่านระบบ Firewall: ได้รับอนุมัติรหัสกุญแจทองเหลืองกู้ภัย 90%]

“เรย์! เขาวงกตไผ่เหล็กพวกนี้ขยับปรับเปลี่ยนทิศทางสลับตำแหน่งทุก ๆ สามสิบวินาที!” โนอาตะโกนลั่นชี้นิ้วฝ่าหมอกทมิฬ “รังสีความว่างเปล่าบิดเวลาและแรงโน้มถ่วงในทางคณิตศาสตร์สลับขั้ว ถ้าเราเดินตามพิกัดแผนที่ปกติ จะหมุนวนกบดานกลับจุดเดิมถาวรแน่! เราต้องการข้อมูลชี้เป้ากริดจำลองในการหาทิศสัญจร!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทจับจ้องมองแนวทิศรอยต่อกริดสีทองที่สั่นกะพริบใต้รากไม้ *‘การปรับสลับเขาวงกตขวางทิศ... คล้ายคลึงกับกลไกยุทธวิธีกดดันพื้นที่ใน VGC ชิบูย่า! หากเราใช้กำลังกายรุดเดินดุ่ม ๆ จะเสียเทมโปเวลาสปีดรบรุกจนความล้าสะสมขึ้นพีก... เราต้องผสานคลื่นรหัสสะสมของสแนปโซและเครื่องสแกนทองคำตรวจหาเส้นรอยเชื่อมกริดหลักที่เป็นจุดคงที่ (Fixed Node Point) ในการเจาะเขาวงกตเพื่อความมั่นคง!’*

“คุณคีธครับ... ใช้ไคริวเปิดพายุ Tailwind พัดสลายหมอกดำกั้นพิกัดชั่วคราว โโบกุเระใช้รากไม้เกาะพยุงรากไผ่กอซ้ายดึงกระแสพลังงานทองเหลืองเพื่อระบุจุดสมดุลชี้ทางลาด!” เรย์ชี้นิ้วสั่งการน้ำเสียงเรียบนิ่งเด็ดเดี่ยวมีพลัง

ไคริวมังกรเขียวยักษ์กู่ก้องสลัดปีกลม Tailwind พัดผ่านแนวป่าไผ่ทองเหลืองชะล้างหมอกดำ สลายประกาย Anomaly ชั่วประเดี๋ยว โบกุเระปล่อยกิ่งรากไม้ฝังเข้าโคนต้นไผ่ตรึงระนาบ ส่งผลให้เส้นตารางกริดสีทองคำเหลวโผล่สว่างวาบขึ้นมาเป็นเส้นทางลาดตรงแหวกแนวกรงเขาวงกตเปิดช่องทางเดินตรงเข้าสู่ลานพิธีการปะทะกอบกู้หลักรวดเร็วปานประกายแสงสว่าง!

“ทลายเขาวงกตสถิติได้เฉียบขาดมากซีโร่! ไปมุ่งตรงสู่วิหารปิดรอยแยกกันด่วนเถอะ!” คีธเอ่ยตบคอกล่าวพยุงนำขวา

เรย์สั่งการสแนปโซการ์ดีทองคำและเกโควาชิพุ่งตัวนำแนวรบรุกข้ามแนวกริดสีทอง ก้าวฝ่าเศษฝุ่นดินทองเหลืองเผชิญหน้ากับบอสใหญ่คิงดราในเทิร์นประลองสุดท้ายทันทีอย่างกล้าหาญ!
ทางทิศเขาวงกตไผ่เหล็กขยับสับเปลี่ยนตารางทิศทางดักพิกัดตลอดเวลาเพื่อกดสปีดขวางกู้ภัย โโบกุเระปล่อยกิ่งไม้ตรึงเสาตารางกริดสีทองสว่างวาบแหวกม่านเขาวงกตเปิดทิศทางลาดให้สแนปโซและเกโควาชิพุ่งตัวนำแนวรบรุกข้ามบ่อน้ำทมิฬแกนรอยแยกวิบัติภัยลำดับสี่ภูมิภาคเหนือ
บรรยากาศโดยรอบทิวเขาไผ่ทองเหลืองยังคงเต็มไปด้วยเสียงสะท้อนกังวานของโลหะที่สั่นไหวเมื่อปะทะลมหนาว ลำต้นไผ่สีทองตระหง่านเรียงรายดูมั่นคงดั่งเสาเหล็กกล้าที่ถูกออกแบบมาเพื่อตรึงระนาบของมิติในจุดเชื่อมต่อระบบนี้ ทุกรายละเอียดบนพื้นหินทรายสลักแสดงริ้วแสงสีทองของตารางกริดจำลองที่ส่องประกายสอดประสาน โนอากระชับแท็บเล็ตวิจัยเฝ้าดูความผันผวนของพลังงานแปรปรวนในขณะที่สแนปโซขยับกายไปตามแนวหญ้าเพื่อขจัดเศษสนิมเหล็กที่สะสมอยู่ตามอุ้งเท้า เรย์ตรวจดูสเตตัสความเสถียรข้อมูลผ่านจอโมโนเคิลขวา พยายามเชื่อมโยงกฎเกณฑ์ฟิสิกส์ธรรมชาติเข้ากับขบวนรบเพื่อหาจุดสมดุลที่ดีที่สุดในการทะลวงม่านหมอกมลพิษของความว่างเปล่า ความรักและสายสัมพันธ์ระหว่างทีมคือสิ่งที่ช่วยพยุงใจร้าวให้เด็ดเดี่ยวในศึกครั้งนี้
