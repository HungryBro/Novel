# ตอนที่ 172: "ประจันหน้ากาเบรียส Anomaly บอสพสุธาว่างเปล่า"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดกระแสมิติลำดับสี่จากใจกลางลานศิลายักษ์วิหารก้องกังวานเลื่อนลั่นสะเทือนสลับเสียงเสาศิลาปฐมกาลถล่ม บิชอปวาลด์แม่ทัพใหญ่คัลต์ยกข้อมือขวาสลัดปล่อยกูบโปเกบอลสีดำขลิบทองแดงปะทุรังสีความแปรปรวนสีม่วงเข้มระเบิดลงลานหินทรายกวาดคลุมแนวสายตาจนมืดสลัว

ท่ามกลางมวลพายุทรายทมิฬที่หมุนวนเป็นเกลียวสูง เงาร่างมังกรยักษ์รูปร่างเปรียวดั่งฉลามบกตระหง่านสีน้ำเงินอมเทา ‘กาเบรียส’ (Garchomp Anomaly) ปรากฏตัวคำรามเสียงต่ำแหลมหูแยกเขี้ยวพุ่งทะยานตัว แผ่นหลังปีกฉลามหนาปะติดด้วย **เกล็ดแก้วผลึกดำสะท้อนประกายมิติมืด** และปลายเล็บก้ามกรงเล็บคมสองข้างยาวแผ่ประกายกระแสสสารสีม่วงเข้มเย็นยะเยือก — ‘กาเบรียส Anomaly (ประเภทดิน/มังกร/Void)’!

*“โฮก...!!!”*

มันกู่เสียงเรียกพายุทรายทมิฬพัดซัดกระแทกขวางพิกัดสแกน ขุดค้อนเหล็กแดงปลิวว่อนเฉียงใต้ขวาสล้างพัดฝ่าลานกว้างจนแตกร้าวละเอียด

โนอาตาคลอน้ำใสยกมือขวาทาบอกเสียงสั่นสั่นระริก “เรย์! กรงเล็บ Void Slash และพายุทรายของกาเบรียสตัวนี้ขยับปัดสอยสปีดรอยแยกขึ้นสูงพีกร้อยละเก้าสิบแล้วค่ะ! หน้าต่างประเมิน VGC ในประสาท Tier 3 เอเร่อดับหมดก๊อกเลยค่ะ! ถ้าเราสั่งสู้ล้มเหลว ดวงจิตของพวกเราจะโดนบดสลายตัวกลายเป็นเศษแก้วทราย!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทภายใต้แว่นตาโมโนเคิลขวาประเมินพารามิเตอร์ VGC ในประสาทช่วยรบ Tier 3:

[ตรวจพบรหัสอสุรกายแกนเจ็ด: กาเบรียสแปรปรวน (Anomaly Garchomp) | เลเวล 76]
[ค่าสถานภาพพิเศษ: การควบคุมสนามพายุทรายมิติมืด (Void Sandstorm field)]
[กฎบิดเบี้ยว: ดาเมจพิเศษประเภทธาตุไฟและน้ำจะโดนพายุทรายหักทอนกำลัง 75% ชดเชยน้ำหนัก!]
[ขีดวิเคราะห์ความจำแชมเปี้ยนชิบูย่า: เสถียร 90%]

*‘พายุทรายมิติมืดหักลดดาเมจไฟและน้ำ 75%...’* เรย์หลับตาลงรวบรวมใจเจตจำนงรบสบจังหวะปิดเกมด้วยสายตาจริงเด็ดเดี่ยว *‘นี่คือวิกฤต VGC สลับอากาศที่บีบคั้นชีวิตจริงยิ่งนัก! ภายใต้พายุทรายทมิฬนี้ เพลิงทองคำของสแนปโซและดาบน้ำเกโควาชิจะสูญเสียพลังจู่โจมแทบสิ้นซากจนสร้างดาเมจได้ล้นพ้นลำบาก หากเราไม่มีวิธีสลายสภาพอากาศหรือดึงสปีด Tailwind หนุนนำ จะโดนพายุทรายขยี้ราบเป็นผงทราย!’*

“ลูคัสครับ! สั่งไฟแอโร่บินใช้ Brave Bird สกัดปีกซ้าย! เกโควาชิ! โโบกุเระ! ตั้งแนวคุมขวาดักทางศัตรู!” เรย์สั่งคำรามเสียงเรียบนิ่งเด็ดขาดหนักแน่น

กบนินจาเกโควาชิและ Growlithe สแนปโซกระโจนสปริงตัวฝ่าฝุ่นปูนมลพิษเข้าปะทะดวลเดือดศึกสุดท้ายแกนเจ็ดทันทีอย่างหาญกล้า!
ท่ามกลางความเงียบงันสลัวของห้องโถงปฐมกาลสัมบูรณ์ ไอความว่างเปล่าสีม่วงทมิฬยังคงสาดซัดถล่มรอยร้าวของแผ่นศิลาปิดกั้นทางลาดประจบศึก โนอาตาคลอน้ำใสจ้องมองดูตรากุญแจทองเหลืองกู้ระบบในมือของเรย์ด้วยความเศร้าสร้อยปนห่วงใยในชะตากรรมดวงจิต ลูคัสขยับยืนป้องหลังอย่างเหนียวแน่นมั่นคงเตรียมพร้อมรับสถานะละลายสลายร่างหากเกิดข้อผิดพลาด เรย์ดับแว่นตาโมโนเคิลวิเคราะห์ขวาเพ่งดวงตาคมประคองเจตจำนงรบปิดผนึกคอร์ 100% ร่วมกับสแนปโซการ์ดีทองคำสองสัจธรรมด้วยสายสัมพันธ์ประสานจิตใจสูงสุดยอด ปลดล็อกวิถีเลือกทางเลือกที่สามกู้ภัยระบบ Firewall โลกจริงโดยไม่มีการละทิ้งมิตรแท้ชั่วนิรันดร์
