# ตอนที่ 163: "นิมิตประวัติศาสตร์ของระบบรอยแยกมิติ"

*วิ้ง... วิ้ง... เปรี๊ยะ...!!!*

เสียงกระแสคลื่นเหนี่ยวนำ Anomaly สั่นเตือนสะท้านสะเทือนห้วงคิดวิเคราะห์ของเรย์ ก่อนที่แสงรังสีสีทองจากตรากุญแจทองเหลืองกู้ระบบกะพริบวาบสว่างเจิดจ้าดุจตะวันสว่าง สาดสาดส่องฉายภาพนิมิตโฮโลแกรมยักษ์ประวัติศาสตร์ระบบ ‘นิมิตประวัติศาสตร์รอยแยกมิติ’ (Void Historical Chronology Vision) ลอยพวยพุ่งขึ้นมาประสานระบบประสาทสมองของเรย์กวัดแกว่ง

ในภาพนิมิตสลัวเผยเหตุการณ์เมื่อสามสิบปีก่อน... โลกเสมือนจริงใบนี้ไม่ได้พังทลายเพราะการลักดึงแชมเปี้ยนเรย์ข้ามมิติมา...

แต่ **รอยแยกมิติสีม่วงดำความว่างเปล่าเกิดมีขึ้นอยู่ก่อนหน้าแล้ว** จากความบกพร่องสะสมลัดวงจรของระบบคอร์ธรรมชาติเสื่อมสภาพกติกา มหาพิทักษ์อาร์เซอุสพยายามแก้ไขระบบคอร์แต่ติดกฎของตนเองไม่ให้แก้ไขตรงสิทธิ์แอดมิน จึงจำเป็นต้องกวาดสร้าง ‘ระบบบันกิรัสกู้ภัย Memory Lock’ ดึง Candidates เทรนเนอร์ VGC นอกโลกเดิมมาทำหน้าที่เป็นผู้บัญชาการกู้รอยแยกชะล้างมณฑล

ภาพตัดฉาย Candidates รุ่นพี่สิบสองคนแผดร้องคำรามล้มทรุดสลายตัวจมลงเหวกลายเป็นสสาร Void เพราะพ่ายแพ้ศึกปิดผนึกล้าสะสมประสาท

“เรย์! นิมิตประวัติศาสตร์แสดงความจริงรอยแยกมิติทั้งหมดแล้วเว้ย!” ลูคัสตะโกนกู่ร้องเอามือปาดเหงื่อพยุงสแกนขวา “ที่แท้อาร์เซอุสดึงเธอมาไม่ใช่ตัวร้ายบิดเบี้ยว แต่ดึงเธอมาเพื่อพิทักษ์ช่วยเหลือล้างรอยร้าวที่กำลังจะถล่มธรรมชาติจริงโลกของพวกเราทิ้งไป!”

โนอาตาคลอน้ำใสตกตะลึงล้นพ้น “ใช่ค่ะ... และคุณพ่อของเรย์พยายามแฝงตัวสแนปโซการ์ดีสองสัจธรรมไว้ก็เพื่อช่วยไม่ให้เรย์โดนระบบดีดสะท้อนร่างละลายสลายตัวดั่ง Candidates รุ่นก่อน!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทประเมินประสาทช่วยรบ Tier 3:

[ปลดล็อกข้อมูลประวัติศาสตร์ Firewall สัจธรรม: สมบูรณ์ 100% (Truth Unveiled)]
[ระดับการกู้คืนความทรงจำคงเหลือ: 90% (คงที่เสถียร)]
[ขีดวิเคราะห์ความมั่นคง Willpower: เสถียรเต็มร้อยพิกัด]

*‘ฉันไม่ได้เป็นเหยื่อรับเคราะห์ VGC...’* เรย์แตะลูบขนหน้าอกอุ่นจัดของสแนปโซการ์ดีทองคำคู่ยากลึกซึ้งห้วงจิต *‘พ่อรักฉันและพยายามตั้งขบวนหนทางเลือกที่สามเพื่อช่วยพวกเรากู้ชีพกลับบ้าน! ศึกตัดสินสุดท้ายสยบ Tyranitar บิชอปวาลด์ จะเป็นคำตอบที่ฉันเลือกเองเพื่อปกป้องโลกธรรมชาติจริงนี้เคียงข้างเพื่อนร่วมทาง!’*

ขบวนกู้ภัยรุกขยับก้าวเดินฝ่าฝุ่นปูนมุ่งตรงสู่วิหารแกนกลางปฐมกาลเพื่อประเดิมเปิดประจัญรบศึกตัดสินทันทีอย่างองอาจหาญกล้า!
รอบบริเวณระเบียงศิลายอดเจดีย์ปฐมกาลทองเหลืองยังคงอบอวลไปด้วยกระแสพลังงานทองคำหนาแน่นที่สั่นเตือนสะท้อนกังวานสลับกับไอเย็นจัดจากพายุหิมะภายนอก เกล็ดหิมะบางเบาสล้างปลิวว่อนผ่านลวดลายสลักโบราณประดับยอดเสา โโนอากุมกล่องวิจัยสแกนค่าความเสถียรมิติด้วยใบหน้าเคร่งเครียดปนตื่นตัว ลูคัสคุมขบวนรบร่วมคู่หูไฟแอโร่อย่างตั้งใจพึ่งพา เรย์ใช้สายตาสีเทาสุขุมเพ่งมองริ้วรอยพับกริดสีทองใต้พื้นด้วยระบบช่วยรบ Tier 3 ปรับใจเจตจำนงรบ Willpower คงที่ร้อยเปอร์เซ็นต์ ความเป็นผู้สมัคร Candidates คนนอกระบบชี้ทางให้เขาเข้าใจความผูกพันดั้งเดิมที่ล้ำค่ากว่าสูญสิ้นสถิติ VGC ทวีคูณสะสะสม
