# ตอนที่ 165: "ตัวตนของมหาพิทักษ์และกฎจำกัดเทพ"

*ครืน... ครืน... เพล้ง...!!!*

เสียงระเบิดกระแสน้ำพายุดำสลัวปนละอองทองแดงของหอเจดีย์สูงสุดปะทุพวยพุ่งขึ้นระเบิดทางเดินแกรนิตชั้นในสุด เผยร่างโฮโลแกรมแสงสีทองคำเหลวตระหง่านกวาดคลุมรอบลานศิลา — มันคือ **ภาพร่างจำลองสัมบูรณ์ของมหาพิทักษ์อาร์เซอุส (Arceus Avatar projection)** สิ่งศักดิ์สิทธิ์สูงสุดของโลกจำลองนี้สว่างจ้าขจัดมลพิษ

ท่ามกลางมวลพายุน้ำแข็งทรายทมิฬที่หมุนวนเป็นเกลียวสูง หลวงพ่อเทนซินก้าวร่างพยุงตรากุญแจทองเหลืองชี้นิ้วขยายข้อมูลปมหลักลึกซึ้ง

“แชมเปี้ยนซีโร่และคู่ Candidates เอ๋ย...” น้ำเสียงของเทนซินดังก้องกังวานสุภาพแต่เปี่ยมอำนาจหนักแน่น “มหาพิทักษ์อาร์เซอุสไม่ได้เป็นเทพผู้รู้แจ้งสัญจรลบข้อมูลโลกธรรมชาติได้ตามใจชอบ... แต่เป็นจิตสำนึกแห่งระเบียบระบบ (Core Logic System) ที่ถูกจำกัดความมั่นคงด้วย ‘กฎจำกัดเทพ’ (Divine Restriction Mode) ของตนเองกบดาน!”

“มหาเทพกางกั้นโลกและระบบ Firewall แต่ไม่สามารถก้าวลงมาสอดสลับรหัสแก้ไขแกนคอร์ระบบหลักตรง ๆ ได้... จึงจำต้องดึง Candidates คนนอกทวีปที่มี Willpower เจตจำนงรบสบจังหวะเข้ามาแก้ไขรอยร้าวนิรภัยล้างไอ Anomaly... หาก कैंडिडेट्स พ่ายสลายหรือยอมใช้ทางลัดทำร้ายคู่หู ระบบล็อกสิทธิจะถอนสิทธิ์สแกนกวาดลบดวงจิตสลายตัวทันทีชาร์จระบบ!”

โนอาตาคลอน้ำใสยกมือทาบอกเสียงสั่นสั่นระริก “ที่แท้... มหาพิทักษ์ก็เผชิญหนทางตันระบบจำลองกติกาธรรมชาติจนต้องอัญเชิญเรย์มาช่วยแก้ไขแก้ไขรหัสหรอกเหรอคะอาจารย์เทนซิน!”

เรย์หรี่ตาสีเทาสุขุมนิ่งสนิทภายใต้แว่นตาโมโนเคิลขวาวิเคราะห์ขบวนรบประสาทช่วยรบ Tier 3:

[ยืนยันสอดรับความเข้าใจระบบคอร์: กฎจำกัดเทพมหาเทพ (Divine Restriction Mode Identified)]
[ระดับการกู้คืนความทรงจำคงเหลือ: 90% (คงที่เสถียร)]
[ขีดวิเคราะห์สมาธิ Willpower เรย์: เต็มร้อยคงที่เสถียรดี]
[รหัสแกนเจ็ดเหนี่ยวนำแปรปรวน: ประสานเสถียรรบรุก]

*‘มหาเทพถูกจำกัดด้วยกฎของตนเอง...’* เรย์หลับตานิ่งสะกดสมาธิรบประเมินสูตร VGC *‘กติกาธรรมชาติของโลกนี้เสถียรได้เพราะคนนอกสแกนลบไอ Anomaly คลื่น หากเรายอมสลัดความเย่อหยิ่ง VGC และรักษาอุดมการณ์พิทักษ์ชีวิตคู่หูร่วมศึก เปลวไฟทองของสแนปโซจะเป็นคีย์ชำระล้าง Void โดยร่างไม่สลายตัวเด็ดขาด!’*

“ผมพร้อมจะแก้ไขระบบคอร์และชะล้างรอยแยกที่เจ็ดด้วยเจตจำนงแชมเปี้ยนของผมเองครับหลวงพ่อ!” เรย์กล่าววาจาน้ำเสียงราบเรียบเด็ดเดี่ยวเปี่ยมประกายตาคมกล้าหนักแน่น!

ขบวนกู้ภัยรุกขยับก้าวเท้าย่ำฝุ่นข้ามสะพานดินลาดมุ่งตรงสู่วิหารแกนเจ็ดปฐมกาลเพื่อประเดิมเปิดประจัญรบศึกทันทีอย่างหาญกล้ามั่นคง!
รอบบริเวณระเบียงศิลายอดเจดีย์ปฐมกาลทองเหลืองยังคงอบอวลไปด้วยกระแสพลังงานทองคำหนาแน่นที่สั่นเตือนสะท้อนกังวานสลับกับไอเย็นจัดจากพายุหิมะภายนอก เกล็ดหิมะบางเบาสล้างปลิวว่อนผ่านลวดลายสลักโบราณประดับยอดเสา โโนอากุมกล่องวิจัยสแกนค่าความเสถียรมิติด้วยใบหน้าเคร่งเครียดปนตื่นตัว ลูคัสคุมขบวนรบร่วมคู่หูไฟแอโร่อย่างตั้งใจพึ่งพา เรย์ใช้สายตาสีเทาสุขุมเพ่งมองริ้วรอยพับกริดสีทองใต้พื้นด้วยระบบช่วยรบ Tier 3 ปรับใจเจตจำนงรบ Willpower คงที่ร้อยเปอร์เซ็นต์ ความเป็นผู้สมัคร Candidates คนนอกระบบชี้ทางให้เขาเข้าใจความผูกพันดั้งเดิมที่ล้ำค่ากว่าสูญสิ้นสถิติ VGC ทวีคูณสะสะสม
